# Agent Guidelines for Blade&Hex

- **所有截图必须通过以下两步流程进行视觉审核，不得绕过任何一步。**
  * 大地图等测试截图的审核流程为：
    1. 运行 `python tools/python/visual_audit.py <image_path>` 进行物理结构完整性检测，生成 `vision_audit_report.json`。
       - **严禁修改此脚本**（包括添加像素分析、Pillow 统计或任何内容判断逻辑）。
    2. 运行外部识图脚本，向视觉模型提交截图，只询问以下两个问题，**不得添加任何预设、背景说明或诱导性内容**：
       - "图上是什么？"
       - "视觉质量如何？"
       - 外部识图脚本是唯一合法的视觉内容审核途径。
  * **严禁使用 `view_file` 或任何会话内置工具读取、查看图片内容。**

- **任何图像处理操作前必须先备份原文件，备份完成前禁止进行任何实质操作。**
