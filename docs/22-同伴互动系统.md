# 同伴互动系统

## 概述

同伴不是沉默的战斗单位——他们有性格、有偏见、有过去、有私人恩怨。当你在酒馆招募了一个半兽人狂战士和一个精灵游侠，把他们放在同一支队伍里，火花是必然的。本系统定义同伴之间、同伴与玩家之间的互动规则。

---

## 一、同伴性格系统

### 1.1 性格标签

每个英雄在生成时获得 **1-2个性格标签**（由 `TraitData` 和属性分布决定）：

| 性格 | 倾向属性 | 对话风格 | 行为偏好 |
|------|----------|----------|----------|
| 勇敢 | 高STR/CON | 直接、自信、不退缩 | 支持战斗选项，反对逃跑 |
| 谨慎 | 高WIS/INT | 小心、犹豫、分析 | 支持侦察选项，反对冒险 |
| 贪婪 | — | 务实、计较、逐利 | 关注金钱选项，不满低报酬委托 |
| 忠诚 | 高CHA | 坚定、护卫、仗义 | 支持保护弱者，反对背叛行为 |
| 愤世 | — | 冷嘲、挖苦、刻薄 | 负面评论一切，偶尔给出犀利见解 |
| 好奇 | 高INT | 多问、兴奋、好学 | 支持探索选项，想调查一切遗迹 |
| 高傲 | — | 居高临下、不耐烦 | 不服从弱者命令（低CHA玩家） |
| 温和 | 高WIS | 关心、体贴、耐心 | 支持治疗/外交，反对残忍行为 |

### 1.2 性格对忠诚度的影响

| 玩家行为 | 勇敢 | 谨慎 | 贪婪 | 忠诚 | 愤世 |
|----------|------|------|------|------|------|
| 接受危险委托 | +2 | -2 | +1 | +1 | 0 |
| 拒绝帮助被袭村庄 | -3 | +1 | 0 | -5 | +1 |
| 抢劫商队 | -2 | -1 | +5 | -5 | +2 |
| 花大钱装备队伍 | 0 | +1 | -2 | +3 | 0 |
| 队友死亡不救援 | -5 | 0 | 0 | -5 | -2 |
| 长期不给饷 | -2 | -1 | -5 | -3 | -3 |

## 二、种族冲突系统

### 2.1 冲突触发

当队伍中有种族关系为负的队友时，会触发冲突事件：

| 冲突组合 | 初始紧张度 | 触发频率 |
|----------|-----------|----------|
| 精灵 + 半兽人 | 高（-20种族好感） | 每3-5天 |
| 矮人 + 半兽人 | 高（-20） | 每3-5天 |
| 精灵 + 矮人 | 低（+0） | 极少 |
| 人类 + 半兽人 | 中（-5） | 每7-10天 |
| 同族队友 | — | 不触发（正面事件） |

### 2.2 冲突事件

| 冲突等级 | 场景 | 后果 |
|----------|------|------|
| 轻微 | 休息时争吵 | 双方士气-2，持续1天 |
| 中度 | 营地中拒绝配合 | 双方忠诚度-3，任务完成效率降低 |
| 严重 | 战斗中不互助 | 拒绝治疗/支援对方，忠诚度-10 |
| 激化 | 要求玩家选边 | 玩家必须选择一方，另一方忠诚度-20 |
| 离队 | 忠诚度归零 | 冲突方离开队伍，不可重新招募 |

### 2.3 化解冲突

| 方式 | 效果 | 条件 |
|------|------|------|
| 并肩作战（同一场战斗） | 紧张度-5 | 两人都存活 |
| 玩家调解（CHA检定） | 紧张度-10 | CHA检定DC 15 |
| 共同经历生死（互相救助） | 紧张度归零 | 在战斗中互相治疗/救援 |
| 赠送装备/金币 | 紧张度-3 | 无 |
| 完成个人任务 | 紧张度-15 | 需特定触发 |

## 三、同伴关系发展

### 3.1 关系等级

| 等级 | 关系值 | 效果 |
|------|--------|------|
| 敌对 | -100~-50 | 拒绝配合，可能离队 |
| 不和 | -49~-20 | 士气-2，偶尔冲突 |
| 一般 | -19~+19 | 正常 |
| 友好 | +20~+49 | 并肩时双方+1命中，解锁联合对话 |
| 挚友 | +50~+79 | 并肩时双方+2命中+1AC，互救优先 |
| 生死之交 | +80+ | 并肩时双方+3命中+2AC，死亡时对方狂暴（伤害+50%，1回合） |

### 3.2 关系增长来源

| 事件 | 关系变化 |
|------|----------|
| 并肩战斗（每场） | +2 |
| 互相治疗/救援 | +5 |
| 同族队友 | +1/天 |
| 种族冲突 | -1~-5/次 |
| 队友被击倒时你选择撤退 | -10（对被抛弃者） |
| 完成队友个人任务 | +20 |
| 长期不给饷 | -3/天 |

## 四、同伴个人任务

### 4.1 任务触发

每个英雄在生成时随机分配一个**个人背景模板**（不显示在角色卡上，直到触发）：

| 背景模板 | 触发条件 | 任务内容 |
|----------|----------|----------|
| 寻找失散家人 | 声望≥100，特定城镇 | 寻找NPC，可能需要战斗或交涉 |
| 复仇誓言 | 遇到特定敌人类型（如杀死其家人的哥布林） | 追踪并击败特定目标 |
| 古代誓言 | 进入特定遗迹 | 探索遗迹，解开秘密 |
| 背叛者 | 遇到特定势力的NPC | 与旧友/旧敌重逢，做出选择 |
| 秘密身份 | 忠诚度≥50时自动揭示 | 揭示真实身份（可能是贵族/法师会成员等） |

### 4.2 任务奖励

| 奖励类型 | 说明 |
|----------|------|
| 关系大幅提升 | +20~+50 |
| 独特特质 | 完成任务后获得个人专属特质 |
| 声望 | +10~+30 |
| 独特装备 | 与背景相关的装备（如家族遗物） |
| 对话解锁 | 新的同伴对话内容 |

## 五、同伴对话事件

### 5.1 触发场景

| 触发 | 频率 | 条件 |
|------|------|------|
| 营地休息 | 每次长休息 | 队伍有≥2人 |
| 大地图行军 | 每5-10游戏日 | 随机 |
| 进入特定区域 | 一次性 | 关联角色背景 |
| 战后 | 每次战斗后 | 有条件触发（如队友濒死） |
| 阶段晋升 | 一次性 | 声望阈值 |

### 5.2 对话类型

| 类型 | 示例 |
|------|------|
| 闲聊 | "你知道为什么我离开银叶森林吗？" |
| 战术建议 | "前面像是山地地形，让我先行侦察？" |
| 关系评论 | "你今天的表现……还算不赖。" |
| 冲突 | "你真的要让那个半兽人碰我的东西？" |
| 背景推进 | "我听说了关于我家族的新消息……" |
| 士气评论 | "大家都累了，也许该休息一下。" |

## 六、原型阶段

### 必须实现
- 性格标签分配（2个/角色）
- 忠诚度系统（已有代码基础）
- 基础种族冲突（精灵+半兽人）
- 营地对话（2-3条模板对话）

### 原型之后逐步扩展
- 完整性格标签系统
- 关系发展系统
- 个人任务
- 对话事件丰富化
- 联合技能（挚友以上解锁）
