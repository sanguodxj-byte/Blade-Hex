# 战略层系统

## 概述

游戏分为**三个阶段**，玩家的身份和可支配的力量随阶段递进：

```
阶段1：雇佣兵团长    →  4-6名英雄的小队战斗（纯战棋RPG）
阶段2：受封骑士      →  英雄+少量士兵的混合战斗
阶段3：领主          →  英雄+完整军队的军团战斗（HOMM式）
```

## 大地图系统（骑砍式）

### 地图规格
- 实时行军（非回合制大地图），遭遇时切入战术战斗
- 所有势力在地图上可见移动

### 大地图实体
| 实体 | 说明 |
|------|------|
| 玩家队伍/军团 | 玩家控制，可招募、补给、战斗 |
| 冒险者队伍 | AI控制的独立小队，有自己的目标 |
| 领主军队 | AI控制的军团，隶属某诸侯/势力 |
| 外族聚落 | 哥布林营地/狗头人矿坑/牛头人石堡 |
| 外族掠夺队 | 从聚落出发的掠夺小队，骚扰村庄 |
| 史诗怪物 | 龙/远古魔像等，在特定区域游荡 |
| 商队 | 城镇间移动的商队，可被袭击/护送 |
| 城镇/城堡 | 可进入、交易、招募、攻占 |
| 村庄 | 委托来源，可被袭击，可被玩家保护/攻占 |

### 行军与时间
- 大地图为实时，1秒现实时间 = 约1小时游戏时间
- 按空格键暂停 → 可以查看信息、下达指令
- 行军速度受地形、队伍规模、补给影响
- 疲劳系统：连续行军不休息 → 战斗中有劣势

### 地图地形
| 地形 | 移动速度 | 视野 | 遭遇概率 |
|------|----------|------|----------|
| 道路 | ×1.5 | 正常 | 低 |
| 平原 | ×1.0 | 正常 | 中 |
| 森林 | ×0.7 | 减少 | 高 |
| 山地 | ×0.5 | 减少 | 中 |
| 沼泽 | ×0.5 | 减少 | 高 |
| 水域 | ×0.3（需船） | 正常 | 低 |

## 阶段1：雇佣兵时期

### 玩家身份
- 雇佣兵团长，率领4-6名英雄角色
- 无军队，只有个人装备和技能

### 玩法循环
```
村庄接委托 → 大地图行军 → 遭遇战斗（小队战棋）→ 回村领赏
```

### 遭遇类型
| 遭遇 | 规模 | 说明 |
|------|------|------|
| 野外遭遇 | 4-8人 vs 4-8人 | 哥布林/狼群/强盗 |
| 委托战斗 | 4-6人 vs 6-12人 | 带地形和目标的战斗 |
| 精英遭遇 | 4-6人 vs 1-3强敌 | 牛头人/魔像/亡灵领主 |
| 冒险者遭遇 | 4-6人 vs 4-6人 | 与其他冒险者冲突 |

### 招募
- 酒馆招募新英雄（有随机属性和职业）
- 英雄有个人背景、性格、成长路线
- 每个英雄都是独立个体（不是兵种堆叠）

---

## 阶段2：骑士时期

### 晋升条件
- 声望达到一定值 + 完成特定主线委托
- 被诸侯册封为骑士 → 获得封地（一个小村庄）

### 变化
- 保留英雄小队（4-6人），同时可以招募士兵
- 士兵不再是独立英雄，而是**兵种单位**
- 战斗规模扩大

### 混合战斗规则
- 战场上的单位分为：**英雄单位** 和 **兵种单位**
- 英雄：完整属性、装备、技能树、可升级
- 兵种：简化属性、无装备选择、固定行为模式
- 英雄被击倒 → 可以被救回；兵种被击倒 → 永久损失

---

## 阶段3：领主时期

### 晋升条件
- 占领并持有至少1座城堡 + 声望达到更高值
- 自称领主 或 被王室正式册封

### 军团系统（HOMM式）

#### 兵种堆叠
- 士兵以**兵种堆叠**形式存在
- 每个堆叠 = 同类型士兵的集合（如"长枪兵 × 24"）
- 战斗中每个堆叠是**一个单位**，占据一个六边形格子
- 堆叠的HP = 单兵HP × 数量
- 攻击伤害 = 单兵伤害 × sqrt(数量)（避免线性膨胀）

```
战场示例（领主时期）：
┌─────────────────────────────┐
│  我方：                       │
│  - 英雄战士 ×1（独立单位）      │
│  - 英雄法师 ×1（独立单位）      │
│  - 长枪兵 ×24（1个堆叠单位）    │
│  - 弓箭手 ×18（1个堆叠单位）    │
│  - 骑兵 ×12（1个堆叠单位）      │
│                               │
│  敌方：                       │
│  - 敌方领主 ×1                 │
│  - 哥布林战士 ×40              │
│  - 哥布林弓手 ×20              │
│  - 哥布林头目 ×8               │
└─────────────────────────────┘
```

#### 兵种分类
| 类型 | 特点 | 克制关系 |
|------|------|----------|
| 长枪兵 | 高防御，移动慢 | 克制骑兵 |
| 剑盾兵 | 攻防均衡 | 通用 |
| 弓箭手 | 远程，近战弱 | 克制无盾单位 |
| 骑兵 | 高移动，冲锋加成 | 克制弓手 |
| 法师团 | 范围法术，脆弱 | 被骑兵克制 |
| 重甲兵 | 极高防御，极慢 | 克制弓手 |

#### 战场规模
| 时期 | 战场格子 | 每方单位上限 | 每方士兵上限 |
|------|----------|-------------|-------------|
| 雇佣兵 | 12×10 | 6个英雄 | — |
| 骑士 | 15×12 | 4英雄+4兵种 | 约80人 |
| 领主 | 20×16 | 4英雄+7兵种 | 约200人 |

#### 堆叠伤害结算
```
单次攻击伤害 = 基础伤害 × sqrt(堆叠数量)
被攻击后损失人数 = (受到伤害 - 单兵AC减伤) / 单兵HP
剩余数量 = max(0, 原数量 - 损失人数)

示例：
弓箭手堆叠(18人) 攻击 哥布林堆叠(30人)
→ 弓箭手伤害 = 5 × sqrt(18) ≈ 21
→ 哥布林单兵HP=4, 减伤后承受 15点
→ 损失人数 = 15/4 ≈ 3-4人
→ 哥布林剩余 26-27人
```

### 攻城系统

#### 城堡等级
| 等级 | 防御设施 | 守军上限 |
|------|----------|----------|
| 木栅村庄 | 木墙、拒马 | 50人 |
| 石堡 | 石墙、箭塔 | 150人 |
| 要塞 | 高墙、双塔、护城河 | 300人 |

#### 攻城战规则
- 攻方需突破城门或攀墙
- 城墙提供全掩体和高度优势
- 箭塔每回合自动射击攻方
- 攻方可使用攻城器械（攻城锤/云梯/投石车）

## 领地管理

### 收入来源
| 来源 | 说明 |
|------|------|
| 村庄税收 | 按繁荣度和人口征收金币 |
| 商路关税 | 控制商路要冲可收过路费 |
| 矿场产出 | 铁矿/金矿提供资源和收入 |
| 雇佣委托 | 即使成为领主仍可接委托 |

### 支出
| 项目 | 说明 |
|------|------|
| 军队薪饷 | 每个士兵每日消耗金币 |
| 补给消耗 | 食物/水，按军队规模 |
| 城堡维护 | 城墙修复、箭塔维护 |
| 英雄薪资 | 高等级英雄要求更高报酬 |

### 领地建设
- 升级城堡防御
- 修建商路（增加收入）
- 建设训练场（提升新兵质量）
- 建设法塔（法师招募和法术研究）

## 势力互动

### 大地图AI行动系统

大地图上的每个非玩家实体都按**固定逻辑循环**行动，它们之间会互相交互——追击、战斗、逃跑、结盟。世界在你不做任何事的时候也在运转。

### AI实体行为逻辑

#### 领主军队
```
每日循环：
1. 检查领地是否受威胁 → 有 → 回防
2. 检查是否有宣战目标 → 有 → 行军至目标城/村
3. 检查是否遇到敌对军队 → 有 → 判断战力比
   - 优势 → 追击
   - 劣势 → 撤退向己方城堡
   - 均势 → 视领主性格决定（谨慎型撤退，激进型迎战）
4. 无紧急事务 → 巡逻领地 / 在城堡招募
```

| 领主性格 | 巡逻范围 | 遭遇强敌时 | 遭遇弱敌时 |
|----------|----------|-----------|-----------|
| 谨慎型 | 领地内 | 立即撤退 | 谨慎追击 |
| 均衡型 | 领地+边境 | 评估后决定 | 追击 |
| 激进型 | 扩张型巡逻 | 较少撤退 | 激进追击 |

#### 冒险者队伍
```
每日循环：
1. 检查当前金币 → 不足 → 前往最近城镇接委托
2. 有委托 → 前往目标区域
3. 途中遭遇敌对势力 → 判断战力
   - 可打 → 战斗
   - 不可打 → 逃跑
4. 无委托 → 在城镇间巡游 / 探索传闻点
5. 冒险者也会互相遭遇 → 可能冲突（竞争同个委托）
```

| 冒险者类型 | 队伍规模 | 常见区域 | 对玩家态度 |
|-----------|----------|----------|-----------|
| 新手冒险者 | 2-3人 | 安全区域 | 友好 |
| 老练佣兵 | 4-6人 | 全境 | 视声望 |
| 精英猎团 | 5-8人 | 危险区域 | 视竞争 |
| 通缉犯 | 3-5人 | 偏远地带 | 敌对 |

#### 外族掠夺队
```
每日循环：
1. 从聚落出发（每N天生成一队）
2. 沿路寻找最近的人类聚落
3. 途中发现村庄 → 评估防御
   - 弱防御 → 袭击
   - 有守军 → 绕行或等援军
4. 袭击成功 → 掠夺资源 → 携带战利品返回聚落
5. 袭击失败 → 逃跑回聚落
6. 被领主军队追上 → 战斗
7. 被玩家追上 → 战斗
```

| 外族类型 | 队伍规模 | 移动范围 | 目标偏好 |
|----------|----------|----------|----------|
| 哥布林掠夺队 | 8-15人 | 聚落周边3-5格 | 粮仓/农场 |
| 狗头人采矿队 | 6-10人 | 聚落周边2-3格 | 矿场 |
| 牛头人战团 | 2-4人 | 聚落周边5-8格 | 任何 |
| 暗影教团巡逻 | 4-8人 | 较远范围 | 遗迹/特定目标 |

#### 史诗怪物
```
行为逻辑：
- 在领地范围内巡逻（龙：霜冠山脉周边5格）
- 偶尔外出觅食（低概率扩大巡逻范围）
- 被任何势力接近 → 警告 → 攻击
- AI领主遇到史诗怪物 → 撤退（除非兵力极其充足）
- 史诗怪物不会主动攻城，但会摧毁路上的村庄
```

#### 商队
```
行为逻辑：
- 在固定路线上的城镇间往返
- 每N天从城镇出发
- 被外族/强盗发现 → 被追击 → 被拦截则损失货物
- 玩家可护送商队（委托）/ 也可抢劫商队（降低声望）
- 商队安全到达 → 该城镇繁荣度微增
```

### AI间战斗与追逐

#### 追逐规则
```
每tick（大地图更新）：
1. 所有AI实体检测视野范围内的目标
2. 对每个目标评估：
   - 关系（敌对/中立/友好）
   - 战力比（己方总战力 / 对方总战力）
3. 决策：
   - 敌对 + 战力比 > 1.5 → 追击
   - 敌对 + 战力比 < 0.7 → 逃跑
   - 敌对 + 中间 → 视性格
   - 中立/友好 → 忽略

追击速度 = 追击方速度 × 1.1（追击方略微加速）
逃跑方被追上 → 进入战斗（自动结算或玩家可观看）
```

#### AI间战斗结算
- 非玩家势力之间的战斗由系统自动结算
- 结算公式：`战力比 × 随机因素(0.8-1.2) → 胜方`
- 败方损失 40-70% 兵力，胜方损失 10-30%
- 玩家附近发生的战斗 → 玩家可以看到结果（战场残迹）

#### 大地图事件通知
| 事件 | 通知方式 |
|------|----------|
| 村庄被外族袭击 | 消息弹窗 + 地图标记 |
| 领主攻占城堡 | 消息 + 势力地图变色 |
| 史诗怪物出现 | 特殊警告 |
| 商队被劫 | 如果接了护送委托则通知 |
| 冒险者完成委托 | 如果你也在竞争该委托则通知 |

### 开局出身系统

#### 出身选择
开局时选择出身，决定初始队伍组成、资源、起始位置和种族。

| 出身 | 初始队伍 | 初始资源 | 起始位置 | 特殊 |
|------|----------|----------|----------|------|
| 落魄贵族 | 1战士+1游侠 | 200金, 好装备 | 城镇 | 初始声望略高 |
| 佣兵老兵 | 2战士+1牧师 | 150金, 普通装备 | 村庄 | 初始队友等级+1 |
| 学院辍学生 | 1法师+1游荡者 | 100金, 法术卷轴 | 城镇 | 额外法术位 |
| 猎人出身 | 1游侠+1战士 | 80金, 弓箭 | 森林边缘 | 野外生存加成 |
| 传教士 | 1牧师+1圣骑士 | 120金, 治疗药水 | 城镇 | 与教会势力初始友好 |

#### 种族选择
每个出身可选不同种族（见 12-种族与招募.md），种族决定：
- 初始队友的种族
- 初始位置附近的友好聚落
- 与各势力的初始关系

## 势力关系系统

### 关系等级
| 关系值 | 态度 | 行为 |
|--------|------|------|
| 同盟(80+) | 友好 | 联合作战、共享情报、互借城池 |
| 友好(50-79) | 开放 | 贸易优惠、允许通行、可招募特殊兵种 |
| 中立(20-49) | 一般 | 正常交易、互不侵犯 |
| 冷淡(-19~19) | 戒备 | 限制通行、加价、拒绝高级招募 |
| 敌对(-50~-20) | 攻击 | 主动追击、袭击村庄 |
| 交战(-50以下) | 战争 | 攻城、全面敌对、煽动外族攻击你 |

### 关系变动事件
| 事件 | 关系变化 |
|------|----------|
| 完成该势力委托 | +5~+15 |
| 袭击该势力商队 | -20 |
| 攻占该势力村庄 | -30 |
| 击败该势力敌对的外族 | +5~+10 |
| 在该势力领地犯罪 | -10~-30 |
| 赠送礼物/贿赂 | +5~+20（花钱买关系） |

## 地图探索系统（战争迷雾）

> 详细设定见 [14-大地图设定.md](14-大地图设定.md) 第六章

### 概述
参考文明系列地图揭示机制。玩家开局只知道出身种族的故乡区域，其余地图被神话生物插画覆盖。只有亲自探索才能揭示真实地形。

### 三层迷雾状态

| 状态 | 渲染 | 信息 |
|------|------|------|
| 未探索 | 神话生物贴图（海蛇/巨龙/奇美拉等） | 完全不可见，无法移动到此 |
| 已揭示 | 正常地形（静态快照） | 地形+POI可见，AI实体不可见 |
| 视野内 | 正常地形（实时） | 一切可见，包括移动中的AI实体 |

### 初始已知区域（按出身种族）

| 种族 | 已知区域 |
|------|---------|
| 人类 | 中央平原 + 西海岸 |
| 精灵 | 银叶森林 + 平原西部边缘 |
| 矮人 | 霜冠山脉 + 北关隘附近 |
| 半兽人 | 丘陵草原 + 平原东部边缘 |
| 半精灵 | 平原 + 森林边缘 |

### 揭示途径
- **亲自探索**：移动到区域自动揭示（VISION_RANGE半径）
- **购买地图**：城镇商店出售区域地图
- **NPC情报**：完成委托后获取情报
- **侦察技能**：游侠/斥候扩大揭示范围
