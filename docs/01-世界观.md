# 世界观

## 风格定位
日式西幻（JRPG/CRPG交叉风格）：严肃但不阴郁，有轻松感，怪物设计偏经典奇幻。参考：火焰纹章的政治结构 + 上古卷轴的种族野外生态 + 正义之怒的冒险基调。

## 大陆：阿瓦隆尼亚

面积约西欧大小，被一条横贯大陆的山脉分为南北两域。

### 地理分区

```
            ┌─────────────────────────────┐
            │        北域·霜冠山脉          │
            │   龙族领地 / 矮人遗迹         │
            ├─────────────────────────────┤
  西海 ←── │         中央平原              │ ──→ 东海
            │   王国腹地 / 村庄密集          │
            │                              │
            ├─────────────────────────────┤
            │        南域·焦土荒原          │
            │   外族领地 / 废墟 / 魔物滋生   │
            └─────────────────────────────┘
```

| 区域 | 地形 | 特点 |
|------|------|------|
| 中央平原 | 丘陵/农田/森林 | 人类王国核心，村庄城镇密集 |
| 霜冠山脉 | 高山/雪地/矿洞 | 龙族栖息地，矮人古代遗迹 |
| 银叶森林 | 密林/溪谷 | 精灵隐居地，古老魔法残留 |
| 焦土荒原 | 荒漠/火山岩 | 外族聚集，最危险的区域 |
| 蛮荒沼泽 | 湿地/毒雾 | 蜥蜴人与魔物出没 |

## 人类王国

### 艾尔特兰王国
- **政体**：封建王国，国王+诸侯
- **现状**：表面和平，实则诸侯割据，王权衰落
- **军事**：骑士团+雇佣兵团，镇守边疆
- **玩家定位**：玩家率领的雇佣兵团受雇于各村/诸侯

### 多种族共存
虽然人类是大陆主体民族，但精灵、矮人、半兽人、半精灵等种族早已融入社会：
- **精灵**：多数隐居银叶森林，少数在城镇做学者或法师
- **矮人**：聚居霜冠山脉地下，少数在城镇做铁匠或商人
- **半兽人**：边缘化群体，多在佣兵团或做苦力，少数被接纳
- **半精灵**：两个世界的桥梁，常做外交官、商人或吟游诗人

种族间存在历史恩怨，但并非全面敌对——更多是偏见和不信任，通过冒险和并肩作战可以化解。

### 村庄生态
- 村庄散布于王国全境，是玩家接委托、补给、招募的核心
- 偏远村庄常年受外族骚扰，需要雇佣兵保护
- 村庄有繁荣度，受玩家行动影响

## 外族生态（上古卷轴式分布）

外族不是"入侵大军"，而是**早已存在于荒野中的原住种族**。它们在恶劣地带建立了自己的聚落，偶尔因为食物短缺、领地争夺或被邪教驱使而骚扰人类村庄。

### 哥布林（Goblin）
- **聚居地**：洞穴、废弃矿坑、森林边缘
- **社会组织**：部落制，由最强的"哥布林王"统领
- **行为模式**： opportunistic — 趁夜偷袭村庄粮仓，很少正面对抗
- **战斗风格**：数量多、弱小、会用陷阱和毒
- **威胁等级**：低（单只），中（部落规模）

### 狗头人（Kobold）
- **聚居地**：地下洞穴、矿脉深处
- **社会组织**：氏族制，崇拜龙或伪龙
- **行为模式**：领地意识极强，在矿道中设大量机关
- **战斗风格**：陷阱+伏击+数量，偶尔有龙脉术士
- **威胁等级**：中（有陷阱时极高）

### 牛头人（Minotaur）
- **聚居地**：荒原上的石堡、迷宫般的峡谷
- **社会组织**：小规模战团，最强者为酋长
- **行为模式**：不主动入侵，但领地被侵犯时会暴怒报复
- **战斗风格**：极高HP和伤害，冲锋碾压，少数会战术配合
- **威胁等级**：高（单只即BOSS级）

### 龙族（Dragon）
- **聚居地**：霜冠山脉高峰、古代遗迹
- **社会组织**：独居或母龙带幼崽，极度领地化
- **行为模式**：极少离开领地，但被惊扰时毁灭性报复
- **战斗风格**：喷吐武器+飞行+恐惧光环+法术，多阶段BOSS
- **威胁等级**：极高（整个小队的终极挑战）

### 魔像（Golem）
- **来源**：古代文明遗留的守护者，或由法师制造
- **分布**：遗迹、古墓、法师塔
- **行为模式**：遵循古老指令守卫特定区域，无自主意识
- **战斗风格**：高AC、免疫多数状态、元素抗性、物理碾压
- **威胁等级**：中-高（取决于类型）
- **类型**：石魔像/铁魔像/火魔像/冰魔像

### 其他魔物
| 种类 | 说明 | 威胁 |
|------|------|------|
| 亡灵 | 古战场/墓穴中滋生，被邪术唤醒 | 中 |
| 史莱姆 | 森林/洞穴中常见，吞噬有机物 | 低 |
| 狼人 | 诅咒者，满月时暴走 | 高 |
| 元素 | 被召唤或自然凝聚的元素体 | 中 |
| 巨蛛 | 地下巢穴，产卵型 | 中 |

## 委托系统（村庄驱动的任务结构）

### 委托来源
```
村庄布告栏 → 玩家接取 → 前往目标区域 → 完成/失败 → 回村领取报酬
```

### 委托类型
| 类型 | 内容 | 触发 |
|------|------|------|
| 讨伐 | 清除骚扰村庄的哥布林/狼群 | 村庄繁荣度下降时 |
| 护卫 | 护送商队/旅人通过危险区域 | 商路被威胁时 |
| 探索 | 调查遗迹/废墟，寻找古代遗物 | 传言/地图发现 |
| 防卫 | 保护村庄抵御外族袭击 | 外族大举进犯时 |
| 紧急 | 某只牛头人/巨龙正在逼近！ | 特殊事件触发 |

### 委托与外族的关系
- 哥布林偷粮 → 讨伐任务（低难度，入门级）
- 狗头人占据矿道 → 探索+讨伐（中难度，陷阱多）
- 牛头人袭击村庄 → 紧急防卫（高难度，BOSS级）
- 龙族被惊扰 → 终极讨伐（最高难度，多阶段战斗）

## 势力与阵营

| 势力 | 特色 | 与玩家关系 |
|------|------|-----------|
| 艾尔特兰王室 | 正规军、骑士团 | 提供高报酬委托 |
| 地方诸侯 | 各怀心思，互相倾轧 | 提供区域委托，声望影响态度 |
| 村庄自治会 | 普通村民，无力自保 | 基础委托来源，最需要帮助 |
| 雇佣兵公会 | 同行，竞争与合作 | 招募队友、交换情报 |
| 银塔法师会 | 施法者组织，研究古代魔法 | 法术学习、法术卷轴交易 |
| 暗影教团 | 崇拜古代邪神，操纵外族 | 主要敌对势力 |

## 魔法世界观
- 魔法来源于世界残留的**以太**，集中在古代遗迹和地脉交汇处
- 施法者是少数能感知和引导以太的人
- 古代文明曾滥用魔法导致毁灭，现在魔法正在缓慢衰退
- 法术卷轴和古代遗物是珍贵的魔法资源

## 对玩法的影响
- **大地图**：中央平原安全，越偏远越危险
- **遭遇表**：不同区域外族分布不同
- **任务循环**：村庄委托 → 战斗 → 回村领赏 → 升级装备 → 下一个委托
- **法术来源**：银塔法师会 / 古代遗迹中的卷轴
- **永久死亡**：外族是真实的威胁，战斗不总是能赢

## 玩家叙事线：从佣兵到领主

### 第一幕：流浪佣兵
- 玩家初到阿瓦隆尼亚，身无分文
- 在村庄酒馆招募初始队友，接低级委托
- 讨伐哥布林、护卫商队、探索小遗迹
- **玩法**：4-6人小队战棋，纯RPG

### 第二幕：受封骑士
- 声望渐高，引起诸侯注意
- 被册封为骑士，获得一个小村庄作为封地
- 开始招募士兵，面对更大规模的威胁
- 与其他冒险者队伍竞争/合作
- 可能遭遇牛头人、魔像等强敌
- **玩法**：英雄+小规模军队的混合战斗

### 第三幕：割据领主
- 攻占第一座城堡，建立自己的势力
- 与AI领主在大地图上博弈：结盟、背叛、攻伐
- 可能面对龙的威胁、暗影教团的阴谋
- 最终目标：统一阿瓦隆尼亚（或选择其他结局）
- **玩法**：英雄+完整军团的HOMM式大战

### 叙事与系统的融合
- 每一幕不是"解锁"，而是**世界对玩家的回应**
- 声望高了 → 诸侯主动拉拢 → 选择效忠或独立
- 攻下城堡 → 其他领主重新评估与你的关系
- 史诗怪物（龙/远古魔像）是移动的灾害，任何阶段都可能遭遇
- 暗影教团在幕后操纵，不同阶段有不同的阴谋

## 原型阶段
需要至少：
- 1个村庄场景（接委托/恢复/装备）
- 1个"哥布林骚扰"委托（2-3场战斗）
- 哥布林×3 + 哥布林弓手×1 + 哥布林头目×1
- 村庄布告栏UI
- 仅实现第一幕（雇佣兵时期）的玩法
