# 世界观

> **关于本文档**
> 本作的大地图在每次开局时**程序化生成**，因此本文档不再钉死任何具体地名、领土疆界或城镇坐标。它只描述**会被每次新世界共享的"恒定层"**：种族文化、势力类型、神话设定、外族生态。具体地理实例（哪座山叫什么、哪条河流向何方、某个王国占了几块地）由 `GeographicNameGenerator` 与 `WorldGenerator` 在开局时基于种子产生。
>
> 写作原则：
> - 世界设定要能**容纳任何噪声地图**（雪域可能在北也可能在西，平原可能围海也可能腹地）
> - 不写"某地一定是某族领土"，而是写"某族倾向于在某种气候/地形扎根"
> - 所有举例的地名都标注【示例】，提醒它只是某次开局的产物

## 风格定位
日式西幻（JRPG/CRPG交叉风格）：严肃但不阴郁，有轻松感，怪物设计偏经典奇幻。参考：火焰纹章的政治结构 + 上古卷轴的种族野外生态 + 正义之怒的冒险基调。

## 大陆：阿瓦隆尼亚

阿瓦隆尼亚是本作的世界，但每个存档对应大陆中**一块被海洋包围的腹地**。这块腹地可能是大陆的中段、半岛、被山脉切开的内陆盆地——由地图种子决定。玩家在大地图上看到的，永远是这块腹地的轮廓。

### 共有特征（无论种子如何）
- **被海洋环绕**：高程图通过 falloff 强制使外圈沉入海，因此玩家始终在一块"陆地中央"开局。海洋是绝对边界，无法跨越。
- **气候带由噪声 + 纬度 + 高程决定**：纬度越靠近上下边缘越冷，高程越高越冷；湿度独立噪声决定植被茂密度。最终通过惠特克矩阵映射为生物群系。
- **不存在"南北域"这种二元划分**：每次开局的气候带分布都不同。某次世界可能"东冰西沙"，下次可能"北雨林南雪原"。
- **几个永恒主题区域必然存在**（位置随机，但一定有）：
  - 至少一片**寒冷区**（雪山/冻原/针叶林）—— 矮人/龙族出没
  - 至少一片**茂密森林**—— 精灵传统聚居地
  - 至少一片**温带平原**—— 人类王国的农业核心
  - 至少一片**贫瘠高危区**（荒漠/荒原/沼泽）—— 外族与暗影教团滋生

### 区域类型清单
程序化生成时，世界会在 `RegionRegistry` 中分配若干语义区域（每次开局名字不同，命名由 `GeographicNameGenerator` 给出）。常见类型：

| 区域类型 | 偏好地形 | 危险度 | 居民倾向 |
|---|---|---|---|
| 高山带 | 雪山/裸岩/冰原 | 高 | 矮人地下城邦 + 龙族独居 |
| 茂密森林 | 森林/密林 | 中 | 精灵 + 哥布林边缘营地 |
| 中央农业区 | 平原/草地 | 低 | 人类王国核心，城镇密集 |
| 干旱带 | 荒漠/稀树草原/荒原 | 高 | 牛头人 / 暗影教团 |
| 沼泽湿地 | 沼泽/冻土沼泽 | 中高 | 蜥蜴人 / 亡灵 / 哥布林 |
| 丘陵草原 | 丘陵/平原 | 中 | 半兽人游牧部落 |

> 这些类型的**数量、大小、位置**因种子而异。玩家可能在某次开局看到两片森林被山脉隔开，下一次只有一大片连绵森林。

## 人类王国

人类是大陆主体民族，但**没有"唯一的王国"** —— 每次开局可能生成 1~3 个独立的人类政体（取决于平原区域的数量与道路网络结构）。

### 共通设定
- **政体**：封建王国，国王或诸侯领有城镇与城堡
- **现状**：表面和平，实则诸侯割据，王权衰落（哪怕只生成一个王国，内部也是分裂的）
- **军事**：骑士团 + 雇佣兵团，正规军镇守边疆，但兵力不足以保护所有边境村庄
- **玩家定位**：玩家率领的雇佣兵团受雇于各村/诸侯
- **文化基调**：英法中世纪，命名风格见 `naming_logic_design.md` 的人类章节

### 多种族共存
虽然人类是大陆主体，但精灵、矮人、半兽人、半精灵已融入人类社会：
- **精灵**：多隐居森林，少数在城镇做学者或法师
- **矮人**：聚居高山地下，少数在城镇做铁匠或商人
- **半兽人**：边缘化群体，多在佣兵团或做苦力，少数被接纳
- **半精灵**：两个世界的桥梁，常做外交官、商人或吟游诗人

种族间存在历史恩怨，但并非全面敌对——更多是偏见与不信任，通过冒险和并肩作战可以化解。

### 村庄与城镇生态
程序化生成会在中央农业区放置 Town/Village/Castle，在海岸放置 Port，在丘陵放置 Outpost。其特征：
- **村庄**密度最高，是玩家接委托、补给、招募的核心
- **偏远村庄**（远离 Town/Castle 的）常年受外族骚扰，需要雇佣兵保护
- 每个 POI 有**繁荣度**（Prosperity），受玩家行动与外族 Raid 影响
- 城堡（Castle）扼守要冲，由诸侯/王室控制，可在第三幕被玩家攻占

## 非人种族（程序化生成的种族阵营）

每次开局会按区域类型生成各种族的核心据点。

### 精灵（Elven Court）
- **聚居地**：茂密森林深处
- **政体**：长老议会制（具体王庭名由命名生成器产出，例如【月影议会】【银叶王庭】等）
- **军事**：游侠弓手 + 法师团，无常规军但全民皆兵
- **经济**：自给自足，出口稀有药材、魔法卷轴、精灵工艺品
- **特点**：封闭但不排外——外人可入森林边缘交易，深入需要许可
- **对外关系**：
  - 与人类：冷淡通商
  - 与矮人：历史友好（共同对抗暗影教团的远古传统）
  - 对暗影教团：仇恨

### 矮人（Dwarven Holds）
- **聚居地**：高山地下城邦
- **政体**：城邦联盟制，每座地下城独立自治，遇外敌联合
- **军事**：重甲步兵 + 攻城器械，山地防守难破
- **经济**：矿业 + 锻造业，是人类王国最大武器供应商
- **典型城邦**：1~3 座，命名风格如【铁炉堡】【深根堡】【霜塔堡】（每局随机）
- **对外关系**：
  - 与人类：友好通商
  - 与精灵：互相尊重
  - 与龙族：紧张共存（挖矿偶尔惊扰龙族）

### 半兽人（Copper Tribes）
- **聚居地**：丘陵草原 / 干旱带边缘
- **政体**：部落联盟，最强酋长担任"赤铜大汗"，武勇试炼上位
- **军事**：轻骑兵 + 狂战士，速度极快但装备粗劣
- **经济**：游牧畜牧 + 抢掠，偶尔与边境黑市交易
- **内部**：5~7 个部落，大汗权威靠实力维系，部落内斗频繁
- **对外关系**：
  - 与人类：敌对（抢掠边境村庄）
  - 与牛头人：尊重（视为"最强战士"）
  - 与暗影教团：警惕

### 暗影教团（Shadow Cult）
- **种族**：人类叛教者为主 + 被腐化的各族成员
- **政体**：严密等级制，最高领袖"暗影先知"据称能听到古神低语
- **领土**：散布于干旱带与沼泽深处的据点 + 各地秘密祭坛（程序化生成时作为高威胁 Settlement 出现）
- **军事**：邪教徒 + 被操纵的亡灵 + 雇佣的外族炮灰
- **目的**：唤醒沉睡于地底的"深渊之主"，将大陆变为暗影笼罩的死亡之地（这是教团自己的目标，与终局机制中的"灰之造物者"无关——见下）
- **手段**：
  - 在人类王国内部渗透，收买诸侯和官员
  - 操纵哥布林和狗头人骚扰边境，制造混乱
  - 在遗迹中搜刮古代魔法，用于黑暗仪式
  - 在暗处培养暗影刺客，暗杀反对者
- **对外关系**：与所有势力敌对（多数势力不知道教团真实规模）

## 外族生态（POI Settlement / Lair）

外族不是"入侵大军"，而是**早已存在于荒野中的原住种族**。他们在恶劣地带建立聚落，偶尔因为食物短缺、领地争夺或被邪教驱使而骚扰人类村庄。

程序化生成会按地形偏好把这些外族聚落散布在地图上：

### 哥布林（Goblin · Settlement）
- **生成偏好**：森林边缘 / 沼泽
- **社会组织**：部落制，由最强的"哥布林王"统领
- **行为模式**：opportunistic — 趁夜偷袭村庄粮仓，很少正面对抗
- **战斗风格**：数量多、弱小、会用陷阱和毒
- **威胁等级**：低（单只），中（部落规模）

### 狗头人（Kobold · Settlement）
- **生成偏好**：丘陵地下 / 山脉东段支脉
- **社会组织**：氏族制，崇拜龙或伪龙
- **行为模式**：领地意识极强，在矿道中设大量机关
- **战斗风格**：陷阱+伏击+数量，偶尔有龙脉术士
- **威胁等级**：中（有陷阱时极高）

### 牛头人（Minotaur · Settlement）
- **生成偏好**：荒漠 / 荒原 / 稀树草原
- **社会组织**：小规模战团，最强者为酋长
- **行为模式**：不主动入侵，但领地被侵犯时会暴怒报复
- **战斗风格**：极高HP和伤害，冲锋碾压，少数会战术配合
- **威胁等级**：高（单只即BOSS级）

### 龙族（Dragon · Lair）
- **生成偏好**：高海拔雪山（elevation > 0.80）
- **社会组织**：独居或母龙带幼崽，极度领地化
- **行为模式**：极少离开领地，但被惊扰时毁灭性报复
- **战斗风格**：喷吐武器+飞行+恐惧光环+法术，多阶段BOSS
- **威胁等级**：极高（整个小队的终极挑战）

### 魔像（Golem · Lair / Ruins / GolemForge）
- **来源**：古代文明遗留的守护者，或由法师制造
- **生成偏好**：远古遗迹 / 魔像工坊
- **行为模式**：遵循古老指令守卫特定区域，无自主意识
- **战斗风格**：高AC、免疫多数状态、元素抗性、物理碾压
- **威胁等级**：中-高（取决于类型）
- **类型**：石魔像/铁魔像/火魔像/冰魔像

### 亡灵（Undead · AncientTomb）
- **生成偏好**：远离城镇的荒漠 / 沼泽 / 古战场
- **来源**：古战场或墓穴中滋生，被邪术唤醒
- **典型组合**：骷髅战士 + 僵尸 + 幽魂
- **威胁等级**：中（数量取决于墓穴等级）

### 其他魔物
| 种类 | 说明 | 威胁 |
|------|------|------|
| 史莱姆 | 森林/洞穴中常见，吞噬有机物 | 低 |
| 狼人 | 诅咒者，满月时暴走 | 高 |
| 元素 | 被召唤或自然凝聚的元素体 | 中 |
| 巨蛛 | 地下巢穴，产卵型 | 中 |

## 委托系统（村庄驱动的任务结构）

### 委托来源
```
POI 布告栏 → 玩家接取 → 前往目标区域 → 完成/失败 → 回 POI 领取报酬
```
任何 Town/Village/Castle 都可发布委托。

### 委托类型
| 类型 | 内容 | 触发 |
|------|------|------|
| 讨伐 | 清除附近 Settlement / Lair | 该 Settlement 的 ThreatLevel 上升 / 周围 POI 繁荣度下降 |
| 护卫 | 护送商队/旅人通过危险区域 | 商路（道路）被威胁时 |
| 探索 | 调查遗迹/废墟，寻找古代遗物 | 传言/地图发现 |
| 防卫 | 保护村庄抵御外族袭击 | Settlement 即将派出 RaidingParty |
| 紧急 | 某只牛头人/巨龙正在逼近！ | 特殊事件触发 |

### 委托与外族的关系
- 哥布林偷粮 → 讨伐任务（低难度，入门级）
- 狗头人占据矿道 → 探索+讨伐（中难度，陷阱多）
- 牛头人袭击村庄 → 紧急防卫（高难度，BOSS级）
- 龙族被惊扰 → 终极讨伐（最高难度，多阶段战斗）

## 势力类别（程序化生成时的角色）

每次开局会基于程序化生成的政治格局实例化以下类别。

| 类别 | 性质 | 与玩家关系 |
|------|------|-----------|
| 人类王室 / 诸侯 | 城堡（Castle）的所有者 | 提供高报酬委托；声望影响态度 |
| 村庄自治会 | Village 的居民 | 基础委托来源，最需要帮助 |
| 雇佣兵公会 | 商业组织，中立 | 招募队友、交换情报；总部会被生成器钉在某座 Town |
| 银塔法师会 | 学术组织，中立偏善 | 法术学习、法术卷轴交易；散布于 Tavern / Town |
| 精灵长老议会 | 森林深处 | 受声望与种族关系影响 |
| 矮人城邦 | 高山地下 | 锻造与商贸 |
| 半兽人部落 | 丘陵草原 | 多数敌对，部分可结盟 |
| 暗影教团 | 主要敌对势力 | 不可结盟（除非黑暗结局线） |
| 外族部落（哥布林等） | 散布的 Settlement | 永远敌对 |
| 商队 | 中立经济实体 | 在道路上随机生成 |

## 魔法世界观
- 魔法来源于世界残留的**以太**，集中在古代遗迹和地脉交汇处
- 施法者是少数能感知和引导以太的人
- 古代文明曾滥用魔法导致毁灭，现在魔法正在缓慢衰退
- 法术卷轴和古代遗物是珍贵的魔法资源
- 程序化生成的 Ruins / Shrine / GolemForge 是法术与卷轴的主要来源

## 灰之造物者（终局神话锚点）

> 终局推杆机制的叙事中心。详见 `docs/34-灰潮纪元终局.md`。

灰之造物者是一个**远早于阿瓦隆尼亚已知历史**的存在——早于精灵长老议会建立、早于矮人挖出第一块秘银。他不是邪神、不是恶魔、不是任何"已知种族的对立面"。

- **本质**：他造的不是生命，是**统一的虚无**。他相信"丰富多样"是噪声，世界应该回归一种简洁的灰色调和
- **动机**：他不憎恨这片大陆，他只是在重塑它。这种冷漠比任何敌意都可怕
- **创造物**：从地图边缘的虚空中走出的**深渊生物**——不是龙、不是亡灵、不是任何阿瓦隆尼亚原住物种
- **苏醒迹象**：地图边缘的羊皮纸开始焦化，未知生物在荒野出没。最终在某座城镇陷落处，**深渊入口**会撕开通向他的殿堂

### 与暗影教团的关系
**没有关系**。这是关键的世界观分界：
- 暗影教团是阿瓦隆尼亚现存的反派，其目标是用召唤"深渊之主"统治大陆
- 灰之造物者远在他们之上、之外，可能也之先。教团甚至**在阻止他**——教团想"统治世界"，造物者想"擦除世界"，目标互斥
- 玩家的终局对手是灰之造物者，不是暗影教团

### 传奇生物的角色
大地图上的中立独立 Lair 单位（龙、远古魔像、远古亡灵、独居牛头人长老等）也察觉到了灰潮的威胁。他们**生而为本能地拒绝被擦除**：
- **未被玩家击杀的**：终局事件开启后可被挑战说服，加入玩家小队对抗灰之造物者
- **已被玩家在前期击杀的**：永久不可重生，永远失去结盟机会——前期的每一次屠龙都是一次终局选择

## 对玩法的影响
- **大地图**：低危区（中央农业区）安全，越偏远越危险（越靠近高山/荒漠/沼泽）
- **遭遇表**：不同区域类型外族分布不同；具体见 `Strategic/Encounter` 模块
- **任务循环**：村庄委托 → 战斗 → 回村领赏 → 升级装备 → 下一个委托
- **法术来源**：银塔法师会 / 程序化生成的 Ruins 中的卷轴
- **永久死亡**：外族是真实的威胁，战斗不总是能赢

## 玩家叙事线：从佣兵到领主

### 第一幕：流浪佣兵
- 玩家初到本块腹地，身无分文（出身决定起点城镇，见 `origin_questions.json`）
- 在 Tavern / Village 招募初始队友，接低级委托
- 讨伐哥布林、护卫商队、探索小遗迹
- **玩法**：4-6人小队战棋，纯RPG

### 第二幕：受封骑士
- 声望渐高，引起诸侯注意
- 被册封为骑士，获得一座 Village 或 Outpost 作为封地
- 开始招募士兵，面对更大规模的威胁
- 与其他冒险者队伍竞争/合作
- 可能遭遇牛头人、魔像等强敌
- **玩法**：英雄+小规模军队的混合战斗

### 第三幕：割据领主
- 攻占第一座 Castle，建立自己的势力
- 与 AI 领主在大地图上博弈：结盟、背叛、攻伐
- 可能面对龙的威胁、暗影教团的阴谋
- 最终目标：统一这块腹地（或选择其他结局）
- **玩法**：英雄+完整军团的HOMM式大战

### 叙事与系统的融合
- 每一幕不是"解锁"，而是**世界对玩家的回应**
- 声望高了 → 诸侯主动拉拢 → 选择效忠或独立
- 攻下城堡 → 其他领主重新评估与你的关系
- 史诗怪物（龙/远古魔像）是移动的灾害，任何阶段都可能遭遇
- 暗影教团在幕后操纵，不同阶段有不同的阴谋

## 原型阶段
原型仅需：
- 程序化生成的 1 块腹地（约 80×60 hex 即可）
- 至少 2~3 座 Village + 1 座 Town
- 1~2 个哥布林 Settlement 作为试金石
- 1 个布告栏 → 完成"讨伐哥布林"循环
- 仅实现第一幕（雇佣兵时期）的玩法

## 与程序化生成的对接

| 设定 | 实现位置 |
|---|---|
| 区域类型与地形偏好 | `BladeHexCore/src/Map/RegionRegistry.cs` |
| 区域 POI 密度与危险度 | 同上，`PoiDensity` / `DangerLevel` |
| POI 类型 / 外族子类 / 巢穴子类 | `BladeHexCore/src/Strategic/OverworldPOI.cs` |
| 区域 / 海洋 / 城镇命名 | `BladeHexCore/src/Character/GeographicNameGenerator.cs` |
| 角色 / 领主 / 家族命名 | `naming_logic_design.md` + `NameGenerator` |
| 遭遇生成（Settlement → Raid） | `Strategic/Encounter/EncounterUnitFactory.cs` |
| 玩家出身（起点城镇/初始物品） | `BladeHexCore/src/Data/origin/origin_questions.json` |

> **维护规则**：当 `RegionRegistry` / `OverworldPOI.POIType` 增减条目时，本文档"区域类型清单 / 外族生态 / 势力类别"三节必须同步更新。
