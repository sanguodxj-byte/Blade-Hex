# 声望系统

## 概述

声望是玩家的**全局影响力指标**。与势力关系（04-战略层系统）不同，声望反映的是玩家在整个阿瓦隆尼亚大陆上的**知名度和公信力**——你做了什么好事，打败了什么强敌，帮了哪些人。声望直接影响：阶段晋升、招募价格、商店折扣、委托等级、NPC态度、世界事件触发。

---

## 一、声望值范围

| 等级 | 声望范围 | 称号 | 效果 |
|------|----------|------|------|
| 无名之辈 | 0 ~ 49 | 无名之辈 | 无特殊效果 |
| 小有名气 | 50 ~ 149 | 小有名气 | 商店价格-5%，酒馆刷新+1人 |
| 颇有声望 | 150 ~ 299 | 颇有声望 | 商店价格-10%，解锁中级委托，可被册封骑士 |
| 名扬四方 | 300 ~ 499 | 名扬四方 | 商店价格-15%，解锁高级委托，NPC主动找你 |
| 传奇英雄 | 500 ~ 799 | 传奇英雄 | 商店价格-20%，解锁史诗委托，部队招募+10% |
| 活着的传说 | 800+ | 活着的传说 | 商店价格-25%，所有委托可接，铁人模式独有声望 |

## 二、声望来源

### 2.1 战斗声望

| 事件 | 声望变化 | 说明 |
|------|----------|------|
| 击败杂兵队（CR≤3） | +1~+3 | 小规模战斗 |
| 击败精英队（CR 4-7） | +5~+10 | 认真战斗 |
| 击败首领级敌人（CR 8-12） | +15~+25 | 区域大事 |
| 击败传奇生物（CR 13+） | +50~+100 | 全大陆轰动 |
| 击败龙族 | +100~+200 | "屠龙者"称号 |
| 战斗全灭敌方 | ×1.0 | 正常 |
| 战斗中有队友死亡 | ×0.7 | 惨胜，声望打折 |
| 战斗全灭（团灭） | -20 | 失败扣声望 |
| 屠杀平民/投降者 | -30~-100 | 恶名增加 |

### 2.2 委托声望

| 委托类型 | 基础声望 | 加成条件 |
|----------|----------|----------|
| 讨伐（低级） | +3~+5 | — |
| 讨伐（中级） | +8~+12 | — |
| 讨伐（高级） | +15~+25 | — |
| 护卫 | +5~+10 | 无损失+50% |
| 探索 | +5~+8 | 发现稀有遗物+5 |
| 防卫 | +10~+20 | 保护村庄免于毁灭 |
| 紧急 | +20~+40 | — |
| 未能在时限内完成 | -5~-15 | 声望下降 |

### 2.3 其他声望来源

| 事件 | 声望变化 | 说明 |
|------|----------|------|
| 完成同伴个人任务 | +5~+15 | — |

## 三、声望与阶段晋升

### 3.1 晋升条件

| 晋升 | 声望门槛 | 其他条件 | 效果 |
|------|----------|----------|------|
| 无名 → 小有名气 | 50 | — | 解锁基础功能 |
| → 颇有声望（骑士） | 150 | 完成至少3个委托 | 可被册封为骑士，获得封地 |
| → 名扬四方（高级骑士） | 300 | 占有至少1座城堡 | 解锁领主相关功能 |
| → 传奇英雄 | 500 | 击败至少1个CR 13+敌人 | 全大陆NPC态度变化 |
| → 活着的传说 | 800 | 完成主线剧情关键节点 | 最终阶段 |

### 3.2 晋升事件

声望达到门槛时，**不会自动晋升**——需要触发一个剧情事件：

| 晋升 | 触发方式 |
|------|----------|
| 雇佣兵 → 骑士 | 在城镇遇到诸侯使者，邀请你参加册封仪式 |
| 骑士 → 领主 | 攻占第一座城堡后，自动触发"自立门户"事件 |
| 领主 → 传奇 | 击败龙族/巫妖等传奇生物后，吟游诗人传唱你的事迹 |

## 四、声望与各系统的交互

### 4.1 与招募系统的交互

| 声望等级 | 招募影响 |
|----------|----------|
| 无名之辈 | 酒馆只出1-3级英雄，价格×1.0 |
| 小有名气 | 酒馆出1-4级，偶尔有4-5级，价格×0.95 |
| 颇有声望 | 酒馆出2-6级，常驻4-5级，价格×0.9 |
| 名扬四方 | 酒馆出3-8级，常驻6-7级，价格×0.85 |
| 传奇英雄 | 酒馆出5-10级，传奇英雄主动来投，价格×0.8 |
| 活着的传说 | 任何等级英雄都可能来投，价格×0.75 |

> 与 12-种族与招募 的价格公式叠加：最终价格 = 基础价格 × 种族系数 × 关系系数 × 声望系数

### 4.2 与委托系统的交互

| 声望等级 | 可接委托 |
|----------|----------|
| 无名之辈 | 仅低级（CR 0-2） |
| 小有名气 | 低级+少量中级（CR 0-4） |
| 颇有声望 | 中级（CR 2-7） |
| 名扬四方 | 中级+高级（CR 2-12） |
| 传奇英雄 | 全部（CR 0-20） |

### 4.3 与NPC态度的交互

声望影响NPC的初始态度（与对话系统联动）：

| 声望 | NPC态度 | 对话影响 |
|------|---------|----------|
| 无名 | 冷淡 | 选项少，无特殊对话 |
| 小有名气 | 中立 | 正常对话 |
| 颇有声望 | 友好 | 解锁额外选项，情报更详细 |
| 名扬四方 | 尊敬 | 商人打折，领主愿意交谈 |
| 传奇英雄 | 敬畏 | NPC主动提供帮助 |
| 活着的传说 | 传说 | 几乎所有NPC都认识你 |

### 4.4 与世界事件的交互

| 声望 | 世界事件 |
|------|----------|
| ≥ 150 | 暗影教团开始注意到玩家 |
| ≥ 300 | 暗影教团派遣刺客 |
| ≥ 500 | 龙族可能主动接触玩家 |
| ≥ 800 | 暗影教团全面开战 |

## 五、恶名系统（负面声望）

声望可以降到负数。负声望代表**恶名**——你不是英雄，而是恶棍。

| 恶名等级 | 声望范围 | 称号 | 效果 |
|----------|----------|------|------|
| 无 | ≥ 0 | — | 正常 |
| 小恶 | -1 ~ -49 | 小恶棍 | 商店加价10%，卫兵盘问 |
| 恶名昭彰 | -50 ~ -99 | 恶棍 | 商店加价20%，城镇禁入，赏金猎人追杀 |
| 万人唾弃 | -100以下 | 恶棍头目 | 全面敌对，仅黑市交易 |

### 恶名来源

| 行为 | 恶名变化 |
|------|----------|
| 袭击商队 | -20~-40 |
| 屠杀投降者 | -30 |
| 抢劫村庄 | -40~-60 |
| 背叛盟友 | -50 |
| 与暗影教团合作 | -30~-100 |

### 恶名消除

| 方式 | 效果 | 限制 |
|------|------|------|
| 完成大量正面委托 | +声望冲抵 | 缓慢 |
| 捐赠大笔金币 | 每100金+1声望 | 上限恢复到0 |
| 击败恶势力（外族/教团） | +10~+30 | — |
| 时间流逝 | 每30天+1 | 缓慢 |

## 七、与存档系统的关系

- 声望值持久化到存档
- 恶名同样持久化
- 已触发的晋升事件标记持久化
- 声望相关世界事件状态持久化

## 八、原型阶段

### 必须实现
- 声望数值范围（0-500）
- 战斗和委托声望获取
- 声望对招募价格的影响
- 声望面板UI
- 声望变化通知

### 原型之后逐步扩展
- 完整声望等级（含恶名）
- 声望与NPC态度联动
- 声望与委托等级联动
- 声望与阶段晋升事件
- 世界事件触发
