# 法术系统

## 核心设计原则

**法术不是"学技能"，是"开盲盒 + 收集"。**

1. 点出技能盘上的法术主节点 → **随机获得1个**该体系对应等阶的法术（不可自选）
2. 想要更多法术 → 在世界中找到**法术书**来学习
3. 战斗中施法受**冷却**和**魔力**双重限制

这意味着两个点了完全相同技能节点的法师，可能拥有完全不同的法术。法术书是宝贵的战略资源。

---

## 一、法术获取方式

### 方式1：技能盘节点（随机获取）
当角色点亮技能盘上的法术主节点时，从对应体系的法术池中**随机抽取1个**法术。

| 技能盘节点 | 获得的法术等阶 | 对应体系池 |
|-----------|--------------|-----------|
| 以太感知 | 1个戏法 + 1个1环法术 | 随机1个体系 |
| 法术精通 | 1个2环法术 | 同一体系 |
| 高阶法术位(2环) | 1个2环法术 | 随机新体系 |
| 高阶法术位(3环) | 1个3环法术 | 同一体系 |
| 高阶法术位(4环) | 1个4环法术 | 同一体系 |
| 高阶法术位(5环) | 1个5环法术 | 同一体系 |

随机规则：
- 第一次点法术节点 → 从8个体系中随机抽1个体系，从该体系获得1个法术
- 后续同一体系的节点 → 从该体系中抽取更高阶法术
- **不可重抽** — 抽到什么就是什么，除非用法术书替换
- 随机种子由角色生成时确定，存档保存

### 方式2：法术书学习（指定获取）
法术书是世界中的可收集物品，使用后**指定学习**某个具体法术。

| 法术书来源 | 说明 |
|-----------|------|
| 遗迹宝箱 | 探索古代遗迹时发现 |
| 法塔研究 | 领地建设法塔后可定期产出 |
| 商店购买 | 银塔法师会/特殊商人出售（昂贵） |
| 敌方掉落 | 击杀法师型敌人概率掉落 |
| 任务奖励 | 特定委托的固定奖励 |

法术书使用规则：
- 使用后永久学会该法术
- 不受体系限制（塑能系的法师也能用书学防护系法术）
- 每本法术书只能用1次
- 稀有法术书（3环以上）极为珍贵

### 方式3：法术卷轴（一次性）
卷轴不属于角色的法术列表，使用一次后消耗。
- 不需要会施法，任何人都能用
- 装备在副武器槽时可在战斗中使用
- 效果等同于施放该法术1次

## 三、魔力系统

### 魔力池
- 每个角色有**魔力池**，上限由属性和技能盘决定
- 基础魔力 = 10 + INT修正 × 2
- 技能盘节点可增加魔力上限（"魔力扩展"小节点 +5）
- 施放法术消耗魔力

### 魔力消耗
| 环阶 | 魔力消耗 |
|------|----------|
| 戏法 | 0（免费） |
| 1环 | 3 |
| 2环 | 5 |
| 3环 | 8 |
| 4环 | 12 |
| 5环 | 18 |
| 6环 | 25 |
| 7环 | 35 |

### 魔力恢复
- **战斗外**：短休息恢复50%，长休息恢复100%
- **战斗中**：每回合自然恢复1点魔力
- **药剂**：魔力药剂恢复指定数值
- **技能盘节点**：某些节点提供"战斗中施法恢复魔力"效果

## 四、冷却系统

每个法术有独立的**冷却回合数**，施放后需等待冷却结束才能再次施放。

| 环阶 | 基础冷却 |
|------|----------|
| 戏法 | 0（无冷却） |
| 1环 | 1回合 |
| 2环 | 2回合 |
| 3环 | 3回合 |
| 4环 | 4回合 |
| 5环 | 5回合 |

- 冷却在回合结束时减1
- 战斗结束后所有冷却重置
- 某些法术有特殊冷却（如终极法术冷却8回合）
- 技能盘节点"快速施法"可使所有法术冷却-1（最低0）

### 冷却 UI 表现
```
法术图标右上角显示冷却数字（灰色覆盖层）
可用法术 → 正常亮度
冷却中 → 灰暗 + 数字
```

## 五、施放规则

### 施放条件
1. 手持**法杖**或**魔导书**（必须装备对应武器）
2. 法术**不在冷却中**
3. 当前**魔力 ≥ 法术消耗**
4. 目标在**射程内**且有**视线**（除非注明不需要）
5. 不在**沉默**状态

### 施放消耗
- 消耗1个**主行动**（除非注明为反应/次要行动）
- 消耗对应的**魔力值**
- 法术进入**冷却**

### 法术攻击与豁免
- **攻击型法术**：法术攻击检定 = 随机数(1-20) + 施法属性修正 + 专精加值 vs 目标AC
- **豁免型法术**：目标做豁免检定 vs 法术DC = 8 + 施法属性修正 + 专精加值
- **自动命中型**：无检定，直接造成效果

### 施法属性
- 默认使用INT（智力）
- 技能盘节点"术士血脉"可改为CHA
- 某些种族特质可能改变施法属性

## 六、范围形状定义

法术的范围由具体法术决定，统一使用以下形状：

### 形状类型
| 形状 | 描述 | 示意图 |
|------|------|--------|
| 单体 | 1个目标格子 | ● |
| 射线 | 从施法者出发的直线，N格长 | ●─●─● |
| 锥形 | 施法者面前120°扇形，N格长 | △ |
| 球形 | 以目标格为中心，N格半径 | ○○○ |
| 线形 | 两点之间的直线，N格长 | ●───● |
| 十字 | 以目标格为中心的十字形，N格长 | ＋ |
| 自身 | 以施法者为中心 | ★ |
| 触碰 | 近战范围内的1个目标 | ● |

### 范格对应表（六边形网格）
| 半径 | 影响格数 |
|------|----------|
| 1格 | 7格（中心+6邻居） |
| 2格 | 19格 |
| 3格 | 37格 |

### 地形交互
- **全掩体**阻挡射线和线形法术（球形和锥形不受阻挡）
- 高处施法者射击低处时，射线可越过低矮障碍
- 墙壁后的目标不受射线/线形影响，但球形法术如果中心在墙内则仍有效
- 友军在范围内也会受伤（无友军保护）

| 稀有度 | 环阶范围 | 获取难度 | 商店价格 |
|--------|----------|----------|----------|
| 普通法术书 | 1环 | 常见 | 100金 |
| 精良法术书 | 2环 | 较少 | 300金 |
| 稀有法术书 | 3环 | 稀有 | 800金 |
| 史诗法术书 | 4环 | 极稀有 | 2000金 |
| 传说法术书 | 5环 | 世界级稀有 | 5000金+ |

### 法术书使用
- 使用法术书 → 选择"学习"→ 永久获得该法术
- 若角色不会施法（没有法术相关技能盘节点）→ 法术书保存在背包，等学会施法后可用
- 每个角色学习的法术总数有上限 = INT修正 + 3（最低3个）
- 超过上限时需"遗忘"一个旧法术才能学新的（遗忘不可恢复）

### 关键设计：技能盘不限定体系

- 但玩家可以用法术书学其他体系的法术
- 跳跃机会可以跳到其他法术分支的节点 → 改变/扩展体系方向

## 十、原型阶段

### 实现范围
- 2种戏法 + 4种1环法术 + 2种2环法术 = 8个法术
- 魔力系统完整实现
- 冷却系统完整实现
- 技能盘"以太感知"节点随机分配法术
- 1种法术书（1环，商店可买）
- 法术范围形状：单体 + 球形 + 锥形 + 射线

### 不在原型中
- 3环以上法术
- 法术书稀有度/掉落
- 独特法术书
- 高阶法术位节点
