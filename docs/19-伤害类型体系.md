# 伤害类型体系

## 概述

本文档统一游戏中所有伤害类型的定义、分类、克制关系和交互规则。伤害类型影响：武器伤害、法术伤害、状态效果、敌人抗性/免疫、装备词缀。

当前代码基础：`WeaponData.DamageType` 定义了 `{SLASH, PIERCE, CRUSH, MAGIC}`，需扩展为完整的元素/物理体系以匹配法术系统（07-法术系统）和敌人抗性表（08-敌方与AI）。

## 六、伤害计算流程（代码实际实现）

```
基础伤害 = 武器骰(XdY) + 等级Nd20 + STR修正 + 技能盘加成 + 词缀加成
    │
    ▼
暴击判定（翻倍）→ 擦伤判定（减半）→ 偷袭加成 → 被动近战加成
    │
    ▼
包夹加成（侧翼×1.25, 背后×1.5）→ 冲锋加成 → 骑乘+2
    │
    ▼
被动减伤（SkillEffectExecutor）
    │
    ▼
装甲穿透（DR系统）:
    d20自然骰 >= DR阈值 → 穿透
      砍伤穿透: 70%HP + 30%DR
      刺伤穿透: 100%HP（无视装甲）
      钝伤穿透: 30%HP + 70%DR
    d20 < DR阈值 → 未穿透
      砍伤未穿: 0%HP + 40%DR
      刺伤未穿: 0%HP + 10%DR
      钝伤未穿: 10%HP + 90%DR（始终漏伤）
    │
    ▼
目标抗性（策划目标，代码待扩展）:
      免疫 → 伤害归零
      抗性（减半）→ 伤害÷2
      弱点 → 伤害×1.5或×2
    │
    ▼
最终HP伤害 = max(1, 结果)
```
