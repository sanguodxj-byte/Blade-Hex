# 34 - 灰潮纪元终局（Grey Tide Endgame）

> **本文档定位**
> 本作的终局推杆机制设计稿。在玩家进入领主中后期后，启动一场**不可逆的、由地图边缘向内蔓延**的"灰潮入侵"。被占领的 hex 永久焚毁，羊皮纸地图视觉转为"被烧穿的旧地图"。终局 BOSS 是**灰之造物者（Grey Creator）**。
>
> 本文档**不是 spec**，只是设计意图与系统蓝图。Phase 5 后段开始实现时再拆 spec。
>
> 与本作其他系统的对接：
> - 终局 BOSS **灰之造物者** 与暗影教团**完全无关**——是一个独立的、远古级别的存在
> - 入侵生物作为新的"深渊生物"种族类别，与现有外族（哥布林/狗头人/牛头人/暗影教团）严格区分
> - 推进逻辑挂在 `WorldEventEngine` 的每日 Tick 上，复用现有 `ThreatLevel` / `IsUnderSiege` / `Prosperity` 字段
> - 视觉层新增 `InvasionOverlay3D`，与 `FogOverlay3D` 平行（不修改被锁定的 `HexOverworldRenderer3D`）
> - 与 §28 地图纹理与拼接规格、§01 世界观、§14 大地图设定 紧密耦合

---

## 一、设计目标

| 目标 | 说明 |
|------|------|
| 解决后期目标缺失 | 当上领主之后，必须有一个明确的终极威胁让玩家继续投入 |
| 让大地图本身在变化 | 程序化生成的世界不再是一张铺好的桌布，而是一张正在燃烧的地图 |
| 制造不可逆失去感 | 默认规则：被占领的 hex 永久焚毁。失败有重量 |
| 强迫玩家做"外交"主动叙事 | 共抗外敌的联盟必须由玩家本人发起、推动、执行——这是玩家在领主阶段最重要的一次行动 |
| 双路线终局 | 联军远征（标准路线）/ 玩家小队独闯（地狱路线），两者都是合法终局 |
| 保留逃生通道 | 高代价的"净化"机制让硬核玩家有慢速反推路线，避免无路可走的弃坑 |
| 对接现有系统 | 不发明新数值轴，复用 ThreatLevel / Prosperity / Reputation / Faction 系统 |

## 二、世界观锚点

### 2.1 灰之造物者（Grey Creator）
- **身份**：阿瓦隆尼亚世界开始之前就存在的远古存在，远早于精灵长老议会建立、远早于矮人挖出第一块秘银
- **本质**：他不是邪神，不是恶魔，不是任何"已知种族的对立面"。他是**"造物"**——他造的不是生命，是**统一的虚无**
- **动机**：他相信所有"丰富多样"都是噪声，世界应该回归一种简洁的灰色调和。他不憎恨阿瓦隆尼亚，他只是**在重塑它**
- **与暗影教团的关系**：**没有关系**。暗影教团是现存反派，灰之造物者远在他们之上、之外，可能也之先。教团甚至在阻止他（教团想"统治"，造物者想"擦除"，目标互斥）
- **创造物**：从地图边缘的虚空中走出的**深渊生物**（详见 §3）。它们是造物者的工具，不是它的同族
- **地图焚毁的本质**：羊皮纸是"创世神记录世界的契约"，灰之造物者焚烧的不是土地，是**契约**——所以视觉上是羊皮纸被烧穿、卷曲、灰化

### 2.2 与"开局程序化生成（板块式）"的兼容
本作主图采用**板块式生成**（`WorldTemplate`），世界被切成 `TemplateGridW × TemplateGridH` 个板块，每板块从 10 种模板（大陆 / 群岛 / 地中海 / 环火山 / 大漩涡 / 泛大陆 / 双生大陆 / 内陆之海 / 高原堡垒 / 破碎诸岛）随机抽取并拼接。世界大小根据玩家选择有 3~4 档（参考 §30 中"小型 65k / 中型 221k / 大型 459k hex"）。

灰潮机制**完全跟着板块式生成走**：
- **入侵起点**：在地图最外圈的板块边界格中（不分海陆，但优先紧邻陆地的水域格）按距离均匀+种子随机选出 `K` 个候选
  - `K = clamp(round(地图周长 hex 数 / 60), 4, 12)`
  - 即小型地图约 4~6 个候选，大型地图约 8~12 个
- **裂隙激活节奏**：跟"地图大小档位"成比例，避免大图节奏太快
- **裂隙位置由种子确定性产生**：同一存档里灰潮重新加载时位置可重现（便于调试与攻略）
- **"地图边缘"的定义**：不是矩形边界——板块网格的外圈板块。如果板块是"群岛"或"破碎诸岛"，则取最外圈的陆地 hex

> ⚠️ 不写"裂隙总在西海"或类似绝对方位。某次开局可能裂隙都在北面，下次都在东南——板块抽到什么模板就跟着走。

## 三、深渊生物（新种族类别）

深渊生物**必须与现有外族严格区分**。

### 3.1 命名约定
- 新增 `OverworldPOI.SettlementRace.AbyssalSpawn`（深渊造物）作为**入侵期专属** Settlement 子类型
- 新增 `OverworldPOI.LairType.AbyssGate`（深渊入口）作为**入侵期专属** Lair 子类型——但**不是裂隙**：裂隙是早期视觉提示无 POI；深渊入口只在第一座 Town 被摧毁后产生（详见 §4）
- 不复用现有 ShadowCult，因为暗影教团**不是终局推手**

### 3.2 单位族谱（与现有 CR 系统对齐）

| 阶段 | 单位 | CR | 设计定位 | 设计要点 |
|---|---|---|---|---|
| 隐蔽期 | 灰刃残影 (Grey Wraith) | 3~5 | 杂兵，群体 | 边缘地区偶尔刷出，玩家以为是新型亡灵 |
| 隐蔽期 | 啮灰兽 (Ash Gnawer) | 4~6 | 中型怪兽 | 形如被剥皮的狼，破坏力强 |
| 中期 | 灰烬燃魂 (Ashen Wraith) | 5~7 | 远程/魔法 | 飞行，喷射灰烬云，使周围 hex 变为"灰烬"地形 |
| 中期 | 噬影者 (Shadow Devourer) | 8~12 | 精英 | 击杀后有几率把战场上死亡的**玩家友军单位**复活为腐化版 |
| 中期 | 虚空魔像 (Void Golem) | 10~14 | 高 HP 推进 | 复用现有 Golem 模型 + 紫黑色调 |
| 后期 | 灰塑使者 (Grey Sculptor) | 15~17 | "造物者"的工匠 | 击杀附近敌人后会用尸体重塑出新的灰刃残影 |
| 后期 | 深渊使徒 (Abyssal Herald) | 18~20 | 准 BOSS | 守在深渊入口，多阶段，召唤深渊生物 |
| 终末 | 灰之造物者 (Grey Creator) | 30 | 最终 BOSS | 殿堂内独有，多阶段，全场地形战 |

### 3.3 生物美术资产策略
最大化复用：
- **灰刃残影 / 啮灰兽** = 现有亡灵/狼模型 + 全身灰白色 + 锐利剪影
- **灰烬燃魂** = 现有亡灵立绘换色 + 飘动尾巴
- **噬影者 / 灰塑使者** = 半人马/牛头人骨架 + 黑曜石质感
- **虚空魔像** = 现有 Golem 模型材质换色
- **深渊使徒** = 必须新做（值得为终局做）
- **灰之造物者** = 必须新做（最终 BOSS 必须有冲击力）

### 3.4 与现有伤害类型的对接
新建/扩展两条伤害类型（见 §19 伤害类型体系）：
- **虚空 (VOID)**：无视一半 DR；造成"虚空腐蚀"状态（每回合 -1 最大 HP，战斗结束清除）
- **灰烬 (ASH)**：减速 + 视野缩短

深渊生物的攻击主要走 VOID 与 ASH，给玩家**装备词缀新维度**：抗虚空/抗灰烬。这等于在 Phase 5 后段开放一批新装备词缀，让玩家有装备升级的目标。

## 四、推进节奏（4 阶段）

整个机制由 `WorldEventEngine` 每日 Tick 推进，复用现有 `ThreatLevel` 字段。

### 4.1 触发条件
触发由两个阈值组合判定：

```
hard_floor_day  = 200        # 硬地板：第 200 天前不会触发
hard_ceiling_day = 400       # 硬天花板：第 400 天必触发
trigger_score = (CurrentDay - hard_floor_day) / (hard_ceiling_day - hard_floor_day)
trigger_score += PlayerActions.AcceleratorWeight
trigger_score -= PlayerActions.DelayWeight
trigger_score += WorldEventEngine.ThreatLevel * 0.3
```

| 玩家行为 | 影响 |
|---|---|
| 清除暗影教团 Settlement | 推迟 -0.05 / 个（与教团无关，但教团活跃在地下消耗能量，间接延缓造物者苏醒） |
| 摧毁特定 Shrine | 推迟 -0.10 / 个 |
| 击杀传奇生物（龙/远古魔像）| **加速 +0.05 / 个**——这些远古存在的死亡正好释放出造物者所需的"创世余烬" |
| 失败的护卫委托 | 加速 +0.03 / 次 |
| 玩家声望 ≥ 领主级 | 触发概率门槛降低 |

> 当 `trigger_score >= 1.0` 且当天满足"非战时" → 进入隐蔽期

**地图大小调整**：以上参数针对中型地图。小型地图 floor/ceiling = 150 / 300，大型地图 = 250 / 500。

**难度选项**：
- 标准 / 经典 / 史诗 三档（节奏从 200/400 缩到 100/250）
- 关闭终局：玩家可在新游戏时勾选"无尽沙盒"，灰潮永远不触发

### 4.2 阶段 1：隐蔽期（trigger_score ≥ 1.0 起，持续约 30~60 天）

**核心特征**：玩家几乎察觉不到机制层面的变化，只能从蛛丝马迹中察觉。

**世界变化**：
- 地图边缘的板块（最外圈板块）开始**低概率**刷出深渊生物（灰刃残影 / 啮灰兽）
  - 这些怪在常规遭遇表里是"未知种族"
  - NPC 描述："这不是我认识的任何生物" / "它的血是灰色的"
- 各 Town/Village 布告栏出现**预兆委托**（QuestType: `OmenInvestigate`）：
  - "夜空出现紫红色极光"
  - "海岸渔船带回不属于任何已知生物的遗骸"
  - "某座 Shrine 的圣火变成灰色"
- NPC 对话中出现"老人们说……"开头的不安台词
- BGM 中加入低频持续音

**视觉**：
- **没有**任何裂隙地标
- **没有**地图焚烧效果
- 大地图最外圈一圈板块的羊皮纸**偶尔出现极淡的焦痕**（极低 alpha，几乎察觉不到）

**玩家可做**：
- 完成预兆委托 → 获得"先知线索"（永久消耗品），用于推迟正式触发 -0.03~-0.10
- 调查关键地点（生成器随机选定的 2~3 个传说级 NPC 学者所在地）→ 解锁"灰之造物者"知识，未来可在外交对话中说服盟友

**进入下一阶段的条件**：
- 隐蔽期开始后第 `30~60` 天（按地图大小缩放）
- **或** 玩家无视所有预兆，trigger_score 累积到 1.5

### 4.3 阶段 2：灰烬期（中期，持续直到第一座 Town 被摧毁）

**触发瞬间**（一次性）：
- 全屏闪白 + 屏幕震动
- BGM 切换到"灰潮"主题
- 弹出全屏新闻信件（动画手绘风）：
  - "**灰潮已至**——大陆边缘的羊皮纸正在焦化。从未被记录的生物大量涌出。"
- 大地图最外圈板块开始**真正的灰烬化**：
  - 全图启动**轻量级灰烬粒子 shader**（屏幕级，不是 hex 级）
  - 灰烬从地图边缘飘向中央，密度随焚毁面积递增
- 第一批边缘 hex 进入 `Threatened` 状态

**持续期间**：
- 每日：边缘 hex 沿"灰之锋面"以可控速度向内蔓延
  - 蔓延速度 = `base_rate × (1 + days_since_phase2_start / 60)`——**越来越快**
  - 即使玩家结盟成功，也只能减缓不能阻止
- 每 7 天：从已焚毁区域高频生成 RaidingParty（深渊生物组队，CR 6~12），向最近的人类 POI 进军
- POI 攻防：
  - Burning hex 上的 POI 自动 IsUnderSiege
  - 持续被 Burning 包围 14 天（无援军） → POI 摧毁，hex 转为 Burned
  - **Town 级别 POI** 摧毁有特殊处理（见下一阶段）
- 高频深渊生物刷新：在地图上 Threatened/Burning 区域内，深渊生物的常规遭遇率提升至 ×3

**视觉**：
- 地图边缘的羊皮纸开始烧黑卷曲（hex 级 shader）
- 全屏灰烬飘扬粒子（屏幕级，强度与焚毁比例联动）
- 屏幕边缘永久淡红 vignette
- 已焚毁的 POI 在地图上保留"残骸"标记

**玩家可做**：
- **救援抉择**：每周生成 2~3 个"紧急救援"委托
- **建造灯塔**：在自有 Castle/Outpost 投入资源（金币 + 矮人符文炸药）建造"启明灯塔"，灯塔半径 5 hex 内的入侵扩张速度减半
- **【中期可选】启动外交链**：访问其他势力首都，开始说服共抗外敌（详见 §6）
- **持续战斗**：在 Threatened/Burning 区域猎杀深渊生物，获得新词缀装备

**进入下一阶段的条件**：
- 第一座 Town 级别 POI 被摧毁（必发生，因为入侵越来越快）
- **或** trigger_score ≥ 2.0（极端无视情况兜底）

### 4.4 阶段 3：深渊期（后期，自第一座 Town 摧毁起）

**触发瞬间**：
- 第一座 Town 被摧毁（设其为 `T_first`）
- `T_first` 的位置发生质变：
  - hex 状态从 Burned 转为 `AbyssGate`（深渊入口，新 LairType）
  - 视觉：羊皮纸上烧出一个圆形大洞，洞底是流动的灰白色虚空（shader 实现）
  - 玩家可走过去并互动 → 触发"是否进入深渊？"对话（独立大地图场景，详见 §7）
- 全屏新闻信件：
  - "**第一座城镇陷落了。在它的废墟之上，一道通向虚无的入口睁开了眼睛。**"
- BGM 切换到"深渊"主题

**变化**：
- 全图剩余的人类 POI 进入"末日"模式：永久减员、布告栏出现"远征"专属委托、招募关闭部分种类
- 地图焚烧速度再次提升（×1.5）
- 每周还会有新 Town 陷落，但**只有第一座转化为 AbyssGate**——其余只是变 Burned
  - 已陷落的 Town 在地图上变为"陷落地标"（持续产怪，但玩家不能进入）
- 出现**深渊使徒**（CR 18~20）作为"深渊入口"的守卫，在入口附近巡逻
- **【后期强制】外交链强制进入下一步**：如果玩家此前没启动外交，所有已结识的盟友势力 NPC 主动派使节找玩家（强制对话，详见 §6.4）

**玩家路线**（终局两选一，或第三种"流亡"）：
1. **远征军路线（联军）**：详见 §7.1
2. **斩首路线（小队独闯）**：详见 §7.2
3. **流亡路线**：去港口登船离开（结局动画：玩家船队在血红色浪潮中向远海驶去）

### 4.5 阶段时序图
```
day 0 ── trigger_score ≥ 1.0 ──── 边缘有怪刷出 ──── 第一座 Town 摧毁 ──── 玩家选择终局 ── 终局
              ↓                          ↓                      ↓                       ↓
         [隐蔽期 阶段1]              [灰烬期 阶段2]          [深渊期 阶段3]           [深渊场景]
                                  T+0~第一 Town 倒下      ←可选进入深渊→
```

> **关键约束**：无论玩家是否结盟、是否在前期建造灯塔，**hex 都会持续被入侵焚毁，且越来越快**。结盟只能让远征军路线更轻松，**不能阻止地图被烧**。这是一条"末日时钟"，不是"可解除的诅咒"。

## 五、地图状态系统

### 5.1 hex 状态机
每个大地图 hex 在入侵开始后获得一个 `InvasionState` 字段：

```
enum InvasionState : byte
{
    Pristine = 0,      // 完好
    Threatened = 1,    // 受威胁（视觉上有焦黑斑点）
    Burning = 2,       // 燃烧中（橙红辉光 + 火焰粒子，不可通过）
    Burned = 3,        // 已焚毁（焦炭色羊皮纸 + 灰烬纹理，永久不可通过）
    AbyssGate = 4,     // 深渊入口（仅第一座陷落 Town，唯一）
    Purified = 5,      // 已净化（玩家用昂贵手段恢复，可通过但 Prosperity 上限 -50%）
}
```

### 5.2 状态转换规则
| 起始 → 终止 | 触发 |
|---|---|
| Pristine → Threatened | 邻接 1 格变为 Burning，且距活跃灰之锋面 ≤ 5 hex |
| Threatened → Burning | 邻接的 Burning 持续 ≥ `5 / acceleration_factor` 天 |
| Burning → Burned | Burning 状态持续 ≥ `7 / acceleration_factor` 天 |
| Burned → AbyssGate | 仅当该 hex 是第一座被摧毁的 Town，且阶段 3 触发 |
| Burned → Purified | 玩家执行净化仪式（仅相邻已 Pristine 或 Purified 的 hex 可净化）|
| 任意 → Pristine | **永不发生** |

`acceleration_factor` 自阶段 2 起从 1.0 每 30 天 +0.2（即上限速度大约是初始的 2 倍）。

### 5.3 状态对玩法的影响
| 状态 | 移动 | POI | 遭遇 | 视觉 |
|---|---|---|---|---|
| Pristine | 正常 | 正常 | 正常 | 羊皮纸正常 |
| Threatened | 正常 | 正常但 Prosperity-1/天 | 深渊生物概率 +50% | 羊皮纸边缘有焦痕 |
| Burning | 不可通过 | 自动 IsUnderSiege | 仅深渊 RaidingParty 进出 | 橙红辉光 + 火焰粒子 |
| Burned | 不可通过 | POI 残骸（不可入），持续产怪 | 深渊巡逻 | 焦炭色羊皮纸 + 灰烬 |
| AbyssGate | 玩家可进入触发深渊场景 | 残骸 + 深渊使徒守卫 | 深渊使徒 + 灰塑使者 | 圆形虚空洞，灰白流动 |
| Purified | 正常但 ×0.8 移速 | POI 半繁荣度，可重建 | 偶尔深渊残党 | 羊皮纸有补丁 |

### 5.4 路径系统的应对
- `HexOverworldAStar` 在每次 InvasionState 变化时（chunk 级粒度）刷新一次开放集合
- 已规划的 NPC 商队/领主军队路径，遇到 Burning/Burned 时自动重新规划
- 玩家小队若被困在孤立的"Pristine 飞地"中，自动获得"绝望"buff（每天 -1 morale）

### 5.5 净化机制（不可还原规则的逃生口）
**为什么需要这条**：完全不可还原会让"错过早期窗口"的玩家觉得无望弃坑。净化让规则保持张力但不绝望。

**净化代价**（每次净化 1 个相邻已焚毁 hex）：
- 1000 金币
- 1 个圣物（来自完成精灵长老议会的"森林祭礼"任务）
- 1 个矮人符文炸药
- 队伍中需有等级 ≥ 10 的法师或圣职者
- 仪式需现场进行 4 小时游戏内时间（期间不可移动，会被入侵单位攻击）

**结果**：
- 该 hex 状态变为 Purified（带着伤疤）
- 该 hex 上的 POI 不会自动恢复，但玩家可花钱"重建"

**注意**：净化不能恢复已转化为 AbyssGate 的 hex（叙事上：那个洞已经通到造物者那里，凡人手段无法封闭）。

## 六、共抗外敌：外交链（玩家驱动）

### 6.1 设计原则
- **必须由玩家本人发起**：没有任何剧情会"自动"让各国结盟
- **中期可选 / 后期强制开启**：阶段 2 时玩家可主动跑外交；阶段 3 时若还没开始，盟友使节会上门
- **每个对象都需要单独说服**：不存在"召开会议一次性结盟"
- **失败有真实代价**：被拒绝的势力短期内不能再说服

### 6.2 可结盟对象
默认大地图程序化生成时存在以下势力，灰潮开启后**它们之间互相内斗**（彼此交战 / 边境冲突 / 贸易壁垒），不会自发联合：

| 势力类型 | 数量 | 说服难度 | 加入后贡献 |
|---|---|---|---|
| 人类王国 | 1~3（按板块） | 中（声望 + 完成委托链） | 重装步兵 + 弓手 + 骑兵编队 |
| 精灵长老议会 | 1（森林深处） | 高（需提前完成森林祭礼） | 精灵游侠 + 大法师 |
| 矮人城邦联盟 | 1（高山地下） | 中（需提供"创世余烬"样本） | 矮人重甲 + 攻城器械 |
| 半兽人部落联盟 | 1~2 | 高（需武勇试炼） | 狂战士 + 轻骑兵 |
| 暗影教团 | 1 | **不可结盟** | — |

> 暗影教团此时可能也在挣扎（造物者的"擦除"对教团不利），但他们绝不会与玩家合作——剧情上他们走自己的路（背景叙事，不进入玩家路线）。

### 6.3 传奇生物作为可结盟个体
现有大地图上的**中立独立 Lair 单位**（龙、远古魔像、远古亡灵、独居牛头人长老等）此时**全部成为可结盟个体**。

**关键规则**：
- **生死决定可结盟性**：
  - **未被玩家击杀的传奇生物** → 阶段 3 开启后才可挑战说服。说服**等同于一场战斗**（依然要打），只是战斗结束时不杀死它，而是触发一段对话→它加入玩家**作为玩家小队成员**（占用一个 6 人队伍位）
  - **已被玩家在前期击杀的传奇生物** → 永久不可重生，永远失去结盟机会
- **每个被说服的传奇生物加入后**：
  - 成为玩家小队中的一员（占 1 个队伍位）
  - 不参加联军行军遭遇（它跟玩家走，不进入军团编队）
  - 在最终的殿堂决战中战力极强（CR 等同于它原 Lair 数据）
  - **数量上限 = 玩家小队当前空位**（最多让玩家全队都是传奇生物，但玩家本人始终在场=最多 5 个）
- **隐含的前期决策权重**：玩家在 Phase 3~4 击杀龙获得武器和经验，但代价是终局少一张牌。这是设计意图。

### 6.4 外交流程
**中期（阶段 2）开启时**：
- 玩家访问某势力首都 POI → 出现新对话选项："关于灰潮……"
- 选项展开后是该势力的**专属说服委托链**（3~5 步）
- 完成全部步骤 → 该势力同意"等灰潮深入时与你共同行动"
- 此时**势力的领主军队不会立即出兵**——他们承诺在阶段 3 玩家发起远征军时集结

**后期（阶段 3）触发时**：
- 已结盟的势力 → 自动派使节告知"我们已准备好"
- 未结盟的势力 → 派使节上门"局势已经无法忍受，我们愿听你号令"——但**说服难度大幅降低**（剩余说服步骤减半），代价是玩家被迫处理一堆延迟的外交
- 传奇生物 → 玩家可主动前往挑战说服

### 6.5 外交链委托示例（每势力一套）
```
人类王国说服链：
  1. 协助镇压贵族叛乱（防止王室被颠覆）
  2. 击退一支大型深渊 RaidingParty（证明你的实力）
  3. 在王宫朝会上演讲（CHA 检定 DC 18，鼓动诸侯放下成见）

精灵长老议会说服链（前置：完成"森林祭礼"）：
  1. 净化一片被灰烬污染的圣林 hex（消耗 2 次净化资源）
  2. 击败一只灰塑使者并带回它的"造物碎片"
  3. 在长老树下接受灵视试炼（一场单人精神战，无装备无队友）

矮人城邦说服链：
  1. 收集 5 块"创世余烬"（从灰之锋面 hex 上采集）
  2. 在矮人锻造坊协助打造"封印之锤"（消耗资源 + 等待 7 天）
  3. 帮矮人击退一次深渊魔像突袭（必赢战斗，但是高难度）

半兽人部落说服链：
  1. 单挑大汗（不能带队友）—— 失败=声望 -50，成功=继续
  2. 带半兽人战团袭击一处深渊营地
  3. 在祖灵岩前歃血盟誓（消耗 50 金币 + 10 声望）

传奇生物说服（每只独立）：
  1. 单独前往该 Lair（不能带 NPC 盟友）
  2. 触发战斗 → 把它打到 HP < 30%
  3. 此时弹出对话选项 "提议同盟" → 它要求你回答 1~3 个关于"为何而战"的问题（CHA + WIS 检定）
  4. 通过 → 加入玩家小队
```

### 6.6 已结盟势力随时间消耗
**关键设计**：结盟≠安全。已结盟的盟友也会受灰潮影响：
- 每个已结盟势力的 POI 一样会被深渊生物围攻（玩家可以选择优先去保护谁）
- 若某盟友的首都被摧毁 → 该盟友退出联盟（兵力归零）
- 玩家必须在外交奔走 + 战场救火之间平衡

## 七、终局路线（玩家选）

### 7.0 路线分歧的本质

两条路线的**核心差异不在深渊场景内，而在"从玩家所在地走到深渊入口"这段大地图行军过程**：

- 阶段 3 时，玩家所在地与深渊入口之间的区域大部分已经是 **Burned / Threatened** 状态
- 这片区域没有干净的城镇/铁匠铺/酒馆——意味着**无法在路上修理装备、补充弹药、补足食物**
- 同时这片区域**高频刷新深渊生物**（RaidingParty + 巡逻队 + 灰塑使者）
- 所以玩家必须靠"出发前在自家领地装满"的资源硬撑这段路

**这就是"无补给"的真实含义**——不是剥夺补给系统，而是**沿途没有补给点**。补给（食物 / 装备耐久 / 弹药 / Tools / Medicine）按现有 `EconomyManager` 体系正常消耗。

### 7.1 远征军路线（标准路线）

**前置条件**：至少结盟 1 个势力

**触发**：玩家访问深渊入口对话 → 选择"号召联军"
> 注意：玩家可以在还没去深渊入口前就发起联军号召，提前让联军开始集结。

**流程**：
1. **集结期（3 天游戏内）**：
   - 所有已结盟势力的领主军队从各自首都出发，前往**玩家指定的集结地**（默认：玩家当前所在 POI 或路线上最近的安全 POI）
   - 玩家可视化见到大军队往同一点汇集
   - 玩家可在这 3 天内做最后准备（购买装备、修理护甲、买箭）
2. **联军行军（核心差异点）**：
   - 联军作为一个**大地图实体**（StackEntity）从集结地向深渊入口移动
   - 联军在大地图上**主动拦截**通往深渊入口路径上的深渊 RaidingParty
   - 联军遭遇深渊军时**等额刷新对位战斗**（HOMM 式军团会战，联军 vs 等 CR 深渊军）
   - **玩家小队选择**：
     - 跟随联军行军 → 玩家可选择参与每场会战（作为联军前锋）或脱离去做别的事
     - 提前奔赴深渊入口 → 联军在玩家身后清扫，玩家小队几乎不被沿途杂兵骚扰
3. **抵达深渊入口**：
   - 联军主力在入口外建立营地
   - 玩家小队（最多 6 人，含说服的传奇生物）+ 联军中的精锐分队（玩家选最多 6 个堆叠）下到深渊
   - 联军主力**留在入口外**抵挡持续涌出的深渊 RaidingParty（叙事上：他们撑住后路让玩家无后顾之忧）
4. **进入深渊场景**（详见 §8）

**联军路线特点**：
- **大地图行军负担大幅降低**：联军吸火，玩家小队可以选择跳过沿途绝大部分战斗
- **装备/补给消耗最小化**：因为玩家不必频繁亲自参战，护甲耐久、远程弹药、食物的消耗远低于斩首路线
- **会战参与是可选**：玩家想要 XP/战利品可以打，想保留资源可以让联军独立解决
- **入口外有联军营地**：相当于一个临时"半补给点"，玩家可以从联军后勤队**有限地**补充弹药和食物（不能修理装备——铁匠没法在前线操作；不能完全恢复 HP——只是"营地短休"）

### 7.2 斩首路线（地狱路线）

**触发**：玩家访问深渊入口对话 → 选择"独自前往"
> **可选发起时机**：玩家可以从一开始就走斩首路线（无需结盟），也可以在结盟失败/不愿结盟后选这条

**前置规则**：
- 玩家**不需要结盟**，独自行动也允许
- 即便已结盟，玩家也可以放弃联军独自前往（联军会按兵不动，但已结盟势力关系**不会因此恶化**——他们理解你的选择）

**流程**：
1. 玩家小队（最多 6 人）从所在地一路打到深渊入口
2. 沿途所有深渊生物攻击都得自己扛
3. 没有任何 NPC 拦截深渊 RaidingParty —— 玩家是这片燃烧大陆上唯一的反抗者
4. 抵达深渊入口 → 进入深渊场景

**斩首路线的真实困难（按现有补给系统量化）**：

| 维度 | 远征军路线 | 斩首路线 |
|---|---|---|
| 沿途战斗次数 | 玩家可选，0~3 场 | 5~10 场（深渊 RaidingParty + 巡逻 + 守卫） |
| 护甲耐久消耗 | 低（每件护甲约 -10~20%） | 高（每件 -50~80%，大概率有件被打碎） |
| 远程弹药消耗 | 低（约 30 发） | 高（80~150 发，可能在 BOSS 战前耗尽）|
| 食物消耗 | 5 天 × 6 人 × 0.5 = 15 单位 | 8~12 天 × 6 人 × 0.5 = 24~36 单位 |
| 沿途修理/补给机会 | 联军行军营地（仅食物+弹药，无修理） | **无任何修理/补给机会** |
| Tools / Medicine 消耗 | 中 | 高 |
| 战利品 | 低（联军吸火） | 高（玩家全吃）|
| 进入深渊时小队状态 | HP/Mana 充裕，装备完好 | HP/Mana 紧张，装备多件破损 |

**斩首路线的预期心智模型**：
- 玩家在出发前必须**囤积大量物资**：满弹药 / 备用箭 / 满食物 / 大量治疗药水 / Tools / Medicine
- 玩家必须计算："我能不能在装备彻底报废前打到 BOSS？"
- **必须自带远程消耗品**：箭矢、投掷武器、卷轴
- 一旦护甲在沿途被打碎，没有任何机会换上新的——玩家要么硬撑，要么承认失败
- **设计目标：相对会非常困难**。这条路线是给"我自己一个人就能解决"的硬核玩家
- 传奇生物的价值在斩首路线下被极致放大——它们不需要装备维修

### 7.3 流亡路线（妥协结局）

**触发**：玩家访问港口（Port） → 选择"登船离开"

**条件**：阶段 3 触发后随时可用

**流程**：
- 短动画：玩家船队驶离海岸，远处大陆在燃烧
- 评级取决于带走的队友数量与剩余领土

### 7.4 切换路线的灵活性
- 玩家选了远征军路线后**不能中途改成斩首**（联军已出兵，不能撤回）
- 玩家选了斩首路线后**可以中途改成远征军**——只要玩家还活着回到友方 POI，可以发起一次延迟的联军号召（但联军集结时间会增加到 7 天，因为更紧急的形势要求他们重新调整后勤）
- 流亡路线在任何阶段都可启动（前提是港口未被摧毁）

## 八、深渊场景（独立大地图）

### 8.1 总体规格
- **独立大地图场景**：不在主图上，玩家进入后切场景
- **小型大地图**：约 30×20 hex（远小于主图的板块网格）
- **不被板块系统使用**：这是手工设计的固定布局（也由种子产生小变化，但结构是钦定的）
- **三层结构**：
  - **外围荒原（Outer Wastes）**：深渊地表，10×20 hex，散布深渊生物巡逻
  - **造物者工事（Creator's Bulwark）**：要塞外墙 + 城门，进入殿堂的最后屏障
  - **创世殿堂（Genesis Hall）**：BOSS 所在，是一个室内场景（用战斗地图风格渲染）

### 8.2 视觉风格
- 全场景羊皮纸已**完全焚毁**：白纸黑字 + 灰烬粒子 + 远处永恒的灰白色光
- 没有植被，没有水（除了远处一道流动的灰白色"虚空河"）
- 地形以"灰之结晶"为主——尖锐的几何体从地面长出
- 主光源是**没有方向的散射灰白光**，没有阴影
- BGM：极简，单一持续低频 + 偶尔的金属敲击声

### 8.3 玩法流程
```
进入深渊
  ↓
[外围荒原]
  几场战斗（远征军路线：联军精锐分队参战；斩首路线：独自）
  地形战，深渊巡逻队规模 = CR 50~80
  现有食物/HP/morale 系统正常运作（场景内不重置）
  ↓
[造物者工事 攻城战]
  一场高难度攻城战
    远征军路线：玩家+联军精锐 vs 工事守军（深渊使徒×2 + 灰塑使者×3 + 大量杂兵）
    斩首路线：玩家小队 vs 同样的工事守军（极困难）
  胜利后进入殿堂
  ↓
[创世殿堂 终局决战]
  BOSS：灰之造物者（CR 30）
  多阶段战
  胜利 → 终局动画
```

> **关于补给经济在深渊场景中的规则**：
> - 深渊内**没有 Town / Tavern / 商店 / 铁匠铺**，玩家无法在场景内购买、修理、补充任何物资
> - 现有 `EconomyManager.FoodSys` / `WageSys` 在深渊内**正常运作**——食物会消耗，断粮一样会扣 morale 不能恢复 HP；护甲耐久会被扣减；远程武器仍消耗弹药
> - 玩家**带多少进来，打多少战斗**——这是"准备充分性"的考验，不是额外的剥夺机制
> - 玩家进入深渊**不需要预付任何特殊门槛物资**（区别于"净化 hex"那种付费操作），但**强烈建议**在出发前在友方领地把所有补给/修理/弹药拉满
> - 深渊生物 / 工事宝箱掉落少量战利品（食物 / 治疗药水 / 偶尔一根高级箭），不能依赖但能救急

### 8.4 BOSS 战：灰之造物者
**多阶段（4 阶段）**：

| 阶段 | HP | 形态 | 关键机制 |
|---|---|---|---|
| 1：观察 | 100% ~ 75% | 静坐于殿堂中央，灰白色巨人轮廓 | 不主动攻击。每回合用"重塑"指令，把场上某个友军单位暂时变成灰塑使者（持续 2 回合，期间该单位为敌方）|
| 2：扰动 | 75% ~ 50% | 站立，殿堂地板开始裂开 | 每回合释放灰烬波（全场所有非自身单位 -10% 当前 HP，不致死）+ 召唤灰刃残影 |
| 3：擦除 | 50% ~ 25% | 走出殿堂中心，开始挥动手臂 | 主动技能：将场上某个 hex 永久从战场地图上"擦除"（变为虚空，无法通行） |
| 4：终焉 | 25% ~ 0% | 化为巨大的灰色风暴 | 全场地形战，每回合所有 hex 持续受 ASH 伤害，需要打破 4 根"锚"才能让 BOSS 显形 |

**奖励**：
- 唯一武器/装备（每周目可不同）
- 永久成就："灰潮终结者"
- 解锁结局动画

### 8.5 进入深渊后不可返回
- 一旦从 AbyssGate 下去 → 主大地图被锁定，不能再回去
- 失败 = 整个存档结束，进入末日结局动画
- **这是设计意图**：让"进入深渊"成为一个有重量的承诺

## 十、平衡与可玩性注意

### 10.1 节奏曲线（按地图大小缩放）
| 地图档位 | 隐蔽期长度 | 阶段 2 第一座 Town 倒下 | 阶段 3 时长 |
|---|---|---|---|
| 小型 (~65k hex) | 30 天 | 隐蔽期开始后 30~50 天 | 30 天内必须进深渊 |
| 中型 (~221k hex) | 45 天 | 60~90 天 | 60 天 |
| 大型 (~459k hex) | 60 天 | 90~120 天 | 90 天 |

### 10.2 失败的"质感"
- 失去一座村庄时不应只是数字变化：村长（程序化命名）的最后一封信、残骸标记、拾荒掉落
- Town 失陷时全屏 cinematic（手绘风信件 + 城镇剪影燃烧）

### 10.3 阻止"无脑硬撑"
- 入侵单位 CR 缩放跟随玩家平均等级
- 焚毁的城镇不能被反复"挽救"——一旦陷落就是永久（除非昂贵净化）
- AbyssGate 一旦激活，剩余 Town 陷落速度不可逆地加快

### 10.4 鼓励战略选择的设计
- 斩首路线**故意比远征军路线难得多**——硬核玩家有他们想要的挑战
- 但远征军路线**不是无脑通关**——需要前期投入大量外交工作
- "击杀 vs 说服"传奇生物的决策让前期行为产生终局回响

### 10.5 补给经济的真实压力（Phase 5 后期需调试）

灰潮终局的核心紧张感来自现有补给系统在"沿途无补给点"约束下的自然涌现。需要在 Phase 5 后期通过 SimulationHarness 仿真验证：

| 维度 | 验证方法 |
|---|---|
| 玩家是否买得起出发前的全套修理+满弹+满食物 | 用平均经济曲线模拟"领主期玩家进入阶段 3 时的金币储备 vs 补给囤积总价" |
| 远程武器箭矢是否够用 | 模拟斩首路线 8 场战斗的弹药消耗中位数，与玩家可携带量对比 |
| 护甲耐久是否撑得到 BOSS 战 | 模拟玩家全程参战时主战力角色的护甲耐久曲线 |
| 食物是否够 12 天行军 | 验证背包格能否塞下 12 天 × 6 人 × 0.5 = 36 单位的食物 + 其他必需品 |

如果仿真结果显示 **斩首路线"基本不可能完成"**，需要调整：
- 增加深渊生物的食物/药品掉落概率
- 让玩家的工坊/铁匠等级越高，出发前修理速度越快
- 增加 BeaconBuild 灯塔附近的轻量补给（玩家自己造的灯塔可以是小补给点）

如果仿真结果显示 **斩首路线"和远征军一样轻松"**，需要：
- 降低沿途深渊军 RaidingParty 出现间隔
- 拉远玩家所在地到深渊入口的中位距离

### 10.6 结局判定
游戏结束时计算评级：
- **抗战胜利（远征军）**：完成造物者战；评级取决于剩余未焚毁 hex 比例
- **抗战胜利（斩首）**：同上 + "孤胆英雄"成就
- **流亡**：登船离开；评级取决于带走的队友数量与剩余领土
- **失败**：所有 Town 焚毁 OR 进入深渊后失败 → 末日结局动画
