# 职业专属被动（v1.0 常驻被动版本）

> **剑与六芒星（Sword & Hex）** 职业称号对应的"被动"系统
>
> 每个职业称号提供一个**常驻被动效果**，无需消耗、无需激活，装备对应称号即生效。
> 数据权威源：`BladeHexCore/src/SkillTree/CareerSkillRegistry.cs`

---

## 设计核心

1. **全部为常驻被动** — 没有消耗、没有冷却、没有"每场×次"限制（除非效果中特别注明）
2. **高阶称号是完全不同的效果** — 不是低阶称号的简单数值升级，而是全新的战斗机制
3. **不引入额外数据层** — 所有效果基于已有战斗系统（伤害公式、移动规则、护甲/DR体系、法力系统）
4. **不依赖属性修正值** — 修正值范围仅 1-5，不适合作为技能触发/数值核心；采用百分比、固定机制、特殊规则
5. **现实逻辑可解释** — 每个被动都有直观的实战解释
6. **称号叠加生效** — 同时拥有多个称号时，所有对应被动同时生效

### 世界观基调（低魔西幻 / 无神圣魔法）

- "圣光、神圣、神迹" → 改为"信念、誓言、热情、纯粹意志"等世俗描述
- "奥术 / 魔法 / 符文" → 保留，作为稀有但合理的技艺
- WIS = 感知 → 暴击 / 抗暴击 / 洞察弱点

### 设计说明

本版本从 v0.8 的"消耗全部 AP + 每场 1 次"主动大招系统重构为常驻被动系统。核心动因：
- 加深技能盘的养成深度，鼓励玩家探索不同点法
- 每个称号都提供独特的战斗风格，而非一张"决战底牌"
- 被动持续生效，让职业身份贯穿整场战斗而非瞬间爆发

## 通用规则

### 生效方式

- 职业称号对应被动在**技能盘解锁对应节点后永久生效**。
- 多个称号的被动会**同时生效**（例如同时拥有"战士"和"剑舞者"称号，两个被动都有效）。
- 被动不占行动、不消耗 AP、无触发次数限制（除非效果中特别说明"每场战斗 1 次"等）。

### 数值采用

- 所有百分比效果基于面板数值计算。
- 所有特殊机制基于已有战斗系统（详见《03-战术战斗系统.md》）。
- 与主动技能、法术、物品效果**加法叠加**。
- **不依赖属性修正值**：所有效果使用百分比、固定乘数或特殊规则，避免因修正值低（1-5）导致效果微弱。

### 触发型被动

部分被动有严格触发条件（如"背面攻击"、"移动≥3 格"、"命中后"等），系统自动检测条件，条件满足时自动生效。

## 一、单属性职业（6） · 基础被动

### 战士 / Warrior（STR）

**重击 / Heavy Strike**（`warrior_heavy_strike`）
- 常驻被动 · 近战武器伤害 ×1.2
- **解释**：重装战士的每一次挥砍都势大力沉，在基础伤害上直接放大。

### 游侠 / Ranger（DEX）

**精准 / Precision**（`ranger_precision`）
- 常驻被动 · 远程攻击无视一半障碍物命中减益
- **解释**：久经训练的射手总能找到弹道的缝隙，障碍物对 ta 的影响减半。

### 守卫 / Guardian（CON）

**防御 / Defense**（`guardian_defense`）
- 常驻被动 · 每回合首次受到的伤害 -30%
- **解释**：经验丰富的前排懂得如何用盾牌和甲胄最厚的部分迎接第一击。

### 法师 / Mage（INT）

**过载 / Overload**（`mage_overload`）
- 常驻被动 · 法术伤害 +15%
- **解释**：对魔法能量的操控更为激进，每一发法术都注入了更多魔力。

### 刺客 / Assassin（WIS）

**背刺 / Backstab**（`assassin_backstab`）
- 常驻被动 · 背面攻击伤害 ×2
- **解释**：刀刃从不正面相迎——精准命中要害，伤害翻倍。

### 诗人 / Bard（CHA）

**战歌 / War Song**（`bard_war_song`）
- 常驻被动 · 周围 2 格内友军伤害 +5%
- **解释**：无需吟唱，仅仅是 ta 在战场上的存在就能鼓舞周围人的斗志。

## 二、双属性复合职业（15） · 进阶被动

### 剑舞者 / Blade Dancer（STR + DEX）

**旋斩 / Whirlwind**（`bladedancer_whirlwind`）
- 常驻被动 · 近战命中后，强制对另一个相邻敌方发动一次额外攻击（每场战斗 1 次）
- **解释**：剑锋所至，顺势回旋——命中一个目标后借力打力，削向旁边另一个。单次攻击只能触发一次额外攻击，不会连斩。

### 重战士 / Juggernaut（STR + CON）

**冲锋 / Charge**（`juggernaut_charge`）
- 常驻被动 · 移动至少 3 格后发动的近战攻击，伤害 ×1.5
- **解释**：助跑距离越远，冲击力越强。冲锋起来的重战士是一辆人形战车。

### 魔剑士 / Spellsword（STR + INT）

**魔刃 / Mana Blade**（`spellsword_mana_blade`）

- **解释**：在剑刃上附着魔力，每次挥砍都带出魔法能量的余波。

### 处刑人 / Executioner（STR + WIS）

**处刑 / Execution**（`executioner_execution`）

### 征讨者 / Warlord（STR + CHA）

**号令 / Command**（`warlord_command`）

### 决斗者 / Duelist（DEX + CON）

**反击 / Riposte**（`duelist_riposte`）

### 秘射手 / Arcane Archer（DEX + INT）

**锁定 / Lock On**（`arcanearcher_lock_on`）
- 常驻被动 · 对同一目标连续远程攻击时，每次命中后下一次伤害 +10%（最多 +30%，切换目标重置）
- **解释**：奥术标记会附着在目标身上，后续箭矢自动微调弹道，越射越准。

### 狩猎者 / Falconer（DEX + WIS）

**追猎 / Hunt**（`falconer_hunt`）
- 常驻被动 · 攻击已受伤的敌人时，命中 +30%
- **解释**：狩猎者循着血迹追踪猎物——受伤的目标动作迟钝，更容易被命中。

### 游荡者 / Rogue（DEX + CHA）

**佯攻 / Feint**（`rogue_feint`）
- 常驻被动 · 近战攻击未命中时，返还 1 AP（不限次数）
- **解释**：游荡者的剑法虚实难辨，虚招试探不中也可以快速收势，不浪费体力。

### 战法师 / Battlemage（CON + INT）

**法盾 / Mana Shield**（`battlemage_mana_shield`）
- 常驻被动 · 受到伤害时，自动消耗 1 法力抵消 1 点伤害；法力为 0 时不触发
- **解释**：战法师将法力编织成护盾——每一分魔力都用来抵消等量的冲击。

### 苦修者 / Veteran（CON + WIS）

**看破 / Insight**（`veteran_insight`）
- 常驻被动 · HP 低于 80% 时，暴击率 +5%
- **解释**：久经沙场的老兵在受伤后反而能看穿敌人的破绽，精准反击。

### 守御者 / Iron Commander（CON + CHA）

**坚守阵线 / Hold the Line**（`ironcommander_hold_the_line`）
- 主动 · 自身（光环） · 消耗全部 AP
- 给周围 2 格内所有友军挂"阵线" buff 2 回合：AC +2、免疫恐惧/溃逃、士气下限 -20（不再低于 -20）
- 友军主动选择"防御模式"时，再额外伤害减免 3 点

### 大贤者 / Sage（INT + WIS）

**充盈 / Fullness**（`sage_fullness`）
- 常驻被动 · 法力值为 100% 时，法术伤害 +50%；法力值 >50% 且 <100% 时，法术伤害 +30%
- **解释**：大贤者精于法力的精妙调控——魔力越充沛，法术越磅礴。

### 指引者 / Sorcerer（INT + CHA）

**引导 / Guidance**（`sorcerer_guidance`）
- 常驻被动 · 法术伤害使目标 AC -25%（向下取整），持续 3 回合
- **解释**：指引者的法术会扭曲目标周围的魔力场，让队友的攻击更容易命中。

### 预言者 / Prophet（WIS + CHA）

**庇护 / Sanctuary**（`prophet_sanctuary`）
- 常驻被动 · 周围 2 格内友军受到暴击时，该次伤害 -50%
- **解释**：预言者能瞥见命运的轨迹——在暴击降临前拨转它的锋芒，化致命为皮外伤。

### 被动叠加说明

所有已解锁称号的被动同时生效。例如同时拥有"战士"（重击：近战×1.2）和"剑舞者"（旋斩：额外攻击），则近战伤害始终×1.2，且每次命中后有机会触发额外攻击。

### 与 AI 的对称

AI 单位同样享有称号被动——玩家面对的精英敌人也拥有其职业称号对应的被动效果，无需额外处理。
