# 大地图设定 — 阿瓦隆尼亚中部区域

> **画风定位**：战场兄弟式古朴画风。俯视，大地色系，粗粝水彩质感，拒绝光滑华丽。
>
> 📐 距离 / hex 换算见 [31-比例尺与距离体系.md](./31-比例尺与距离体系.md)（SSOT）。
> 1 hex ≈ 250 m，大地图典型尺寸 200×150 hex ≈ 50 km × 37 km。
> 
> 本文档描述游戏大地图所覆盖的区域。该区域是阿瓦隆尼亚大陆的中部腹地，面积约等于法国+德意志，并非整块大陆。
> 地图向西延伸至西海海岸，北抵霜冠山脉主脊，东南两面深入内陆荒野。
> 用途：①生图模型参考 ②程序化地图生成指导 ③世界观一致性锚点。

## 一、地图总览

### 视角与风格
- **视角**：斜45度等距俯视（isometric），类似《战场兄弟》(Battle Brothers) 的大地图视角
- **画风核心**：古朴、粗粝、低饱和，拒绝光滑和华丽
- **整体色调**：以大地色系为主——赭石、焦褐、枯黄、苔绿、铁灰。天空是阴沉的灰白，不见阳光灿烂。像一块被反复折叠的旧皮革地图
- **参考作品**：
  - **《战场兄弟》大地图**：最主要参考。阴暗的中世纪写实感，地形粗糙但有辨识度，城镇是小小的木屋/石堡剪影，森林是深绿到黑的树冠团块
  - **《暗黑地牢》地图美术**：粗犷墨线+水彩晕染的质感，但比暗黑地牢更写实、少卡通
  - **中世纪古地图（Mappa Mundi）**：地名的标注方式、边缘的装饰插画
- **具体画风要素**：
  - **地形渲染**：不用平滑渐变，用粗笔触水彩块面。山是尖削的墨线三角，不是圆润的3D山丘。森林用深绿到墨黑的密集短笔触，像一窝纠缠的荆棘
  - **水域**：灰蓝到铅色，带手绘波纹线条。不是晶莹剔透的湖水，是阴天下的浑浊水面
  - **城镇/建筑**：小而精致的像素风剪影，有烟囱冒烟、城墙旗帜等细节。不是Q版也不是写实，是微缩模型感
  - **道路**：泥土色的虚线或双线，路上偶有小石子纹理
  - **角色图标**：小圆底座+纸片人立绘剪影（呼应HD-2D主视觉）
  - **UI叠加层**：羊皮纸质感的面板、蜡封图标、铁锈色边框
  - **天气效果**：雨天的水渍纹理、雪天的泛白、雾天的模糊边缘——都是手绘纹理叠加，不是粒子特效
- **绝对避免**：日系明亮色彩、卡通渲染、高光反射、渐变天空、华丽粒子效果、任何看起来"干净"的东西
- **比例感**：地图覆盖约800×500公里区域，从西海岸到内陆荒原

## 二、政治格局

地图上存在 **5个独立政治实体** + 多个非国家势力。它们彼此接壤，有贸易、冲突和边界纠纷。

### 2.1 艾尔特兰王国（人类）

- **种族**：人类为主，少量半精灵
- **政体**：封建王国，王权衰落，诸侯实控
- **领土**：中央平原大部，从西海岸港口到东部丘陵边缘
- **军事**：骑士团 + 雇佣兵团，正规军精锐但数量不足
- **经济**：农业为主，西海贸易港口带来大量商税
- **内部矛盾**：
  - 国王 vs 三大诸侯（北境伯爵、南海伯爵、东关伯爵）争夺实权
  - 南境村庄常年受外族掠夺，王室无力保护
  - 雇佣兵公会势力膨胀，部分诸侯认为比王室更可靠
- **对外关系**：
  - 与精灵王国：冷淡和平（历史恩怨，但贸易不断）
  - 与矮人城邦：友好通商（人类用粮食换矮人锻造品）
  - 与半兽人部落：敌对（边境冲突不断）
  - 与牛头人：无外交（互不往来）

### 2.2 银叶王庭（精灵）

- **种族**：精灵为主，少量半精灵
- **政体**：长老议会制，由最年长的七位精灵共同决策
- **领土**：银叶森林全域，从北麓延伸到西南海岸
- **军事**：游侠弓手 + 法师团，不设常规军，但全民皆兵
- **经济**：自给自足，出口稀有药材、魔法卷轴、精灵工艺品
- **特点**：
  - 封闭但不排外——外人可入森林交易，但深入腹地需要许可
  - 森林边缘设有隐秘哨站，监视一切进出
  - 银叶王庭不扩张领土，但绝不允许任何人砍伐古树
- **对外关系**：
  - 与人类：冷淡但通商（精灵视人类为"短命但有活力"的种族）
  - 与矮人：历史友好（远古时代曾并肩对抗暗影教团）
  - 对暗影教团：仇恨（教团曾焚烧精灵圣林）

### 2.3 铁炉堡联盟（矮人）

- **种族**：矮人为主
- **政体**：三城邦联盟，每座矮人城市独立自治，遇外敌时联合
- **领土**：霜冠山脉地下，从地表矿口到深层大厅
- **军事**：重甲步兵 + 攻城器械，山地防守几乎不可攻破
- **经济**：矿业（铁矿、秘银、宝石）+ 锻造业，人类王国最大的武器供应商
- **三城邦**：
  - **铁炉堡**（最大）：联盟首邑，锻造中心
  - **深根堡**：最深的矿城，出产秘银
  - **霜塔堡**：最靠近地表，负责对外贸易和防务
- **对外关系**：
  - 与人类：友好通商（粮食换锻造品，互利关系）
  - 与精灵：尊重（"他们活得久但不傻"）
  - 与龙族：紧张共存（矮人挖矿有时惊扰龙族，偶尔引发冲突）

### 2.4 赤铜部落联盟（半兽人）

- **种族**：半兽人为主，少量流亡人类
- **政体**：部落联盟，最强的酋长担任"赤铜大汗"，通过武勇试炼上位
- **领土**：东部丘陵草原，从中央平原边缘到焦土荒原北缘
- **军事**：轻骑兵 + 狂战士，速度极快但装备粗劣
- **经济**：游牧畜牧 + 抢掠，偶尔与边境村庄黑市交易
- **内部**：
  - 5-7个部落各有领地，大汗的权威靠实力维系
  - 部落间经常内斗，大汗更替频繁
  - 年轻半兽人常到人类佣兵团谋生
- **对外关系**：
  - 与人类：敌对（抢掠边境村庄，但部分半兽人在人类城市谋生）
  - 与牛头人：尊重（半兽人视牛头人为"最强战士"）
  - 与暗影教团：警惕（教团曾试图操纵半兽人当炮灰）

### 2.5 暗影教团（邪教）

- **种族**：混合（人类叛教者为主 + 被腐化的各族成员）
- **政体**：严密等级制，最高领袖"暗影先知"据称能听到古神低语
- **领土**：焦土荒原深处的据点 + 散布于各地的秘密祭坛
- **军事**：邪教徒 + 被操纵的亡灵 + 雇佣的外族炮灰
- **目的**：唤醒沉睡于地底的"深渊之主"，将大陆变为暗影笼罩的死亡之地
- **手段**：
  - 在人类王国内部渗透，收买诸侯和官员
  - 操纵哥布林和狗头人骚扰边境，制造混乱
  - 在遗迹中搜刮古代魔法，用于黑暗仪式
  - 在暗处培养暗影刺客，暗杀反对者
- **对外关系**：与所有势力敌对（但多数势力不知道教团的真实规模）

### 非国家势力

| 势力 | 性质 | 活动范围 |
|------|------|---------|
| 雇佣兵公会 | 商业组织，中立 | 全境，总部在艾尔德镇 |
| 银塔法师会 | 学术组织，中立偏善 | 中央平原，法师塔散布各城镇 |
| 哥布林部落 | 野外部落，敌对 | 森林边缘/沼泽/废弃矿坑 |
| 狗头人氏族 | 野外部落，敌对 | 山脉矿洞/地下通道 |
| 牛头人战团 | 领地意识，中立偏敌对 | 焦土荒原/峡谷 |
| 龙族 | 顶级掠食者，独居 | 霜冠山脉高峰 |
| 商队 | 中立经济实体 | 各城镇间的商路上 |

## 三、水系与地形

### 3.1 河流

地图上有 **2条主要河流** 和 **1条支流**，它们是文明的生命线，也是天然边界。

#### 白石河（主干河流）
- **源头**：霜冠山脉南麓冰川融水
- **流向**：自北向南贯穿中央平原，注入蛮荒沼泽
- **特征**：水流平缓，河面宽阔（80-120米），中下游可通航
- **沿岸**：河湾镇（中游港口）、麦田村（下游渡口）
- **作用**：人类王国的运输动脉，商船往来频繁
- **地理意义**：白石河将中央平原分为"河西"（偏富裕，近海岸）和"河东"（偏荒凉，近草原）

#### 银溪（支流）
- **源头**：银叶森林深处泉眼
- **流向**：自西北向东南，在中央平原中部汇入白石河
- **特征**：水清见底，河面窄（20-40米），不宜通航但可涉渡
- **沿岸**：绿溪村（森林出口）、石桥村（渡口）
- **地理意义**：人类与精灵领地的天然分界线

#### 熔岩河（季节性）
- **源头**：焦土荒原的火山裂隙
- **流向**：荒原内部蜿蜒
- **特征**：枯水期是干涸的岩石沟渠，雨季涌出滚烫的矿化热水
- **沿岸**：无人定居，仅有牛头人在附近设伏击点

### 3.2 海岸线

#### 西海海岸
- **位置**：地图最西侧
- **特征**：岩质海岸为主，北段是悬崖峭壁（霜冠山脉直插海中），南段是沙滩和礁石
- **港口**：晨曦城（最大港口）、河湾镇（河口港）
- **海域描述**：近海有散布的小岛和暗礁，远海风浪大，偶有海兽出没
- **对玩法的影响**：玩家无法渡海，海岸线是地图西边界

#### 内陆水域
- **暗月湖**：银叶森林北部的深水湖，精灵圣地，水面常年笼罩薄雾
- **死水潭**：蛮荒沼泽中散布的毒水潭，看似平静实则栖息水下魔物
- **霜须瀑布**：霜冠山脉南麓，白石河源头，落差巨大，矮人在此修建水车和锻造炉

### 3.3 山脉与关隘

#### 霜冠山脉
- **走向**：横亘地图北部，从西北海岸延伸到东北内陆
- **宽度**：约地图高度的15%
- **主要山峰**：
  - **霜冠峰**：最高峰，常年暴风雪，龙族筑巢于此
  - **铁砧峰**：矮人铁炉堡入口所在
  - **鸦喙崖**：山脉中段的险峻崖壁，通往深根堡的隐秘小径
- **关隘**：
  - **北关隘**：山脉唯一的宽阔通道，人类在此修建了磐石要塞
  - **矮人径**：霜塔堡通往地表的石阶路，矮人把守

#### 中央丘陵
- **位置**：中央平原东部，逐渐过渡为丘陵草原
- **特征**：平缓的红土丘陵，散布灌木和岩石
- **作用**：人类与半兽人的缓冲地带，常有摩擦

### 3.4 森林

#### 银叶森林
- **位置**：地图西部，从西北山脉麓延伸到西南
- **面积**：约占地图20%
- **特征**：古木参天，树冠遮蔽阳光，林中有精灵修筑的石径和吊桥
- **核心**：暗月湖北岸的"世界树"——一棵直径50米的远古巨树，精灵王庭所在地
- **边缘**：森林边缘有被砍伐的空地（人类伐木场）和哥布林营地

#### 松林带
- **位置**：霜冠山脉南麓与中央平原之间
- **特征**：针叶林，冬季积雪，是山地到平原的过渡带
- **作用**：猎人谷所在，也是通往山脉关隘的必经之路

#### 枯木林
- **位置**：焦土荒原与丘陵草原的交界处
- **特征**：枯死的巨树矗立，树皮焦黑，地面有火山灰
- **作用**：暗影教团在此设秘密祭坛，危险区域

## 五、道路与商路

### 5.1 主干道（铺石大路，×1.5移速，低遭遇）

```
晨曦城 ────── 艾尔德镇 ────── 铁锤堡 ────── 磐石要塞（北关隘）
   │              │                             │
   │              │                          霜塔堡（矮人径）
   │              │
   │          河湾镇 ──── 渡口村（→ 蛮荒沼泽方向）
   │
石桥村 ──── 绿溪村（→ 月影哨站，精灵贸易路）
```

- **王都大道**：晨曦城→艾尔德镇→铁锤堡，王国最重要的东西向干道，沿途有驿站
- **北境路**：铁锤堡→磐石要塞，通往山脉关隘的军用道路，巡逻频繁
- **河运道**：河湾镇沿岸河流的船运路线，非道路但功能类似

### 5.2 商路（泥土路，×1.2移速，中遭遇）

```
艾尔德镇 ──── 果园区 ──── 矮丘村 ──── 鹰巢堡（→ 丘陵草原边境）
    │
河岸村 ──── 麦田村 ──── 矮丘村
```

- **东境商路**：艾尔德镇→果园区→矮丘村→鹰巢堡，通往半兽人边境的贸易线
- **河西支路**：艾尔德镇→河岸村→麦田村，河东农产品的运输线

### 5.3 小径（未铺装，×0.8移速，高遭遇）

| 小径 | 连接 | 危险 |
|------|------|------|
| **猎人小径** | 松林村→猎人谷→山脚村 | 中（狼群） |
| **森林边道** | 绿溪村→湖畔村→月影哨站 | 低（偶尔有精灵巡逻） |
| **沼泽径** | 渡口村→蛮荒沼泽边缘 | 高（魔物、毒雾） |
| **荒原古道** | 鹰巢堡→枯木林→焦土荒原入口 | 极高（牛头人、暗影教团） |
| **矮人石阶** | 霜塔堡→铁炉堡（地下通道） | 低（矮人把守） |

## 六、地图探索系统（战争迷雾）

### 6.1 设计理念

参考文明系列的地图揭示机制：
- 玩家开局时只能看到**出身种族能了解的区域**
- 其他区域被**神话生物插画**覆盖，标注着传说和猜测
- 只有亲自探索或从NPC获取情报后，地图才会揭示真实样貌
- 已经揭示的区域即使离开也**保持可见**（文明式永久揭示）

### 6.2 迷雾层级

| 层级 | 视觉表现 | 游戏效果 |
|------|---------|---------|
| **未探索** | 羊皮纸上覆盖神话生物插画（海蛇/巨龙/奇美拉/利维坦等），地名标注为"传说中…"或问号 | 地形不可见，POI不可见，无法移动到该区域 |
| **已揭示** | 正常地形显示，但无实时信息 | 地形可见，POI可见（静态），敌方实体不可见 |
| **视野内** | 正常地形 + 实时动态 | 一切可见：移动中的AI实体、战斗、围攻等 |

### 6.3 出生时已知区域（按种族）

| 种族 | 初始已知区域 | 初始已知城镇 |
|------|------------|------------|
| **人类** | 中央平原大部（西岸+白石河两岸），西海岸 | 艾尔德镇、晨曦城、河湾镇、铁锤堡 + 附近村庄 |
| **精灵** | 银叶森林全域 + 中央平原西部边缘 | 世界树庭、月影哨站、绿溪村、湖畔村 |
| **矮人** | 霜冠山脉（地表+地下）+ 北关隘附近 | 铁炉堡、深根堡、霜塔堡、山脚村 |
| **半兽人** | 丘陵草原 + 中央平原东部边缘 | 矮丘村、鹰巢堡周边 |
| **半精灵** | 中央平原 + 银叶森林边缘（两个世界的交汇） | 艾尔德镇、绿溪村、月影哨站 |

### 6.4 揭示方式

| 方式 | 揭示范围 | 条件 |
|------|---------|------|
| **亲自探索** | 以玩家为中心半径 VISION_RANGE（600px）的圆形区域 | 玩家队伍移动到该区域 |
| **购买地图** | 揭示特定区域（如"沼泽地图"揭示蛮荒沼泽轮廓） | 在城镇商店花费金币购买区域地图 |
| **NPC情报** | 揭示单个POI或小范围区域 | 完成委托后NPC提供情报，或酒馆对话 |
| **侦察技能** | 以玩家为中心更大范围（×1.5）的揭示 | 队伍中有游侠/斥候时被动生效 |
| **间谍网络** | 揭示指定势力的所有领土 | 领主阶段（Phase 3），通过金钱和关系建立 |

### 6.5 未探索区域的神话插画设计

未探索区域不显示为黑幕或灰雾，而是用**手绘神话生物**覆盖，营造"这是未知之地"的氛围：

| 覆盖区域类型 | 插画内容 | 标注文字 |
|-------------|---------|---------|
| 海域 | 巨大海蛇缠绕船只，浪花中有触手 | "此处有海兽出没" |
| 山脉 | 巨龙盘踞山峰，暴风雪中隐约可见城堡轮廓 | "龙族领地，凡人勿入" |
| 森林 | 巨树间闪烁的幽光，树枝形成怪异的人脸 | "精灵的禁地" |
| 荒原 | 巨大的牛头人骨架半埋沙中，远处有火山喷烟 | "死亡之地" |
| 沼泽 | 沼泽中浮出的巨大眼球，枯树上挂着不明生物的骸骨 | "瘴气弥漫，有去无回" |
| 通用未知 | 手指从迷雾中伸出，罗盘指向不明方向 | "未知领域" |

插画风格：褐色墨水线条画，与羊皮纸底色融合，有中世纪版画/古地图怪诞插图的风格。

### 6.6 技术实现要点

```
地图网格划分为 cell_size（20px）的区块，每个区块有探索状态：
- explored: bool（是否已被揭示）
- last_seen: Dictionary（最后看到时的POI/地形快照）

数据结构：
  explored_grid: Array[Array[int]]  # grid_y][grid_x] → 0=未探索, 1=已揭示, 2=当前视野

每帧更新：
  1. 清除所有"当前视野"标记回退为"已揭示"
  2. 以玩家位置为中心，VISION_RANGE内的格子标记为"当前视野"
  3. 将这些格子永久标记为"已揭示"

视觉渲染：
  - 未探索格子：绘制对应的神话生物贴图（根据区域类型选择）
  - 已揭示格子：正常绘制地形，但不显示动态实体
  - 当前视野格子：正常绘制一切（含AI实体动画）

存档：
  - explored_grid 随存档保存，探索进度永久保留
```
