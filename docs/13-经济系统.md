# 经济系统策划案：金币、血汗与契约

## 1. 核心资源
游戏主要包含三种基础资源：
- **金币 (Gold)**：用于购买装备、招募佣兵、支付军饷。
- **食物 (Food)**：大地图行军消耗。归零时士气大幅下降，触发饥饿减益。
- **补给品 (Tools/Medicine)**：用于战后治疗和装备维护。

## 2. 收入来源 (Inflow)
- **战斗战利品**：击败敌人后获得的掉落金币和可变现物品。
- **任务奖励**：城镇发布的契约任务。
- **贸易差价**：不同城镇间商品价格波动带来的利润。
- **领地税收 (后期)**： Phase 5 开启。

## 3. 支出项 (Outflow)
- **每日军饷**：根据队伍规模和佣兵等级，每天 00:00 结算。
- **行军消耗**：在大地图移动时按时间/距离消耗食物。
- **补给采购**：购买消耗品。
- **装备更替**：提升队伍战斗力的核心支出。

## 4. 城镇贸易机制
- **库存刷新**：城镇商店每隔 3 个游戏日刷新一次货架。
- **供需影响**：城镇特产（如产粮区的食物）价格更低。
- **声望折扣**：完成该地区任务可提升声望，降低采购价格。

## 5. 经济循环 (Loop)
1. 队伍在大地图行军（消耗食物/金币）。
2. 触发战斗/任务（获取战利品/赏金）。
3. 进入城镇（补给物资、升级装备、招募新人）。
4. 提升实力以挑战更高产出的区域。
