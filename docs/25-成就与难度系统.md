# 成就与难度系统

## 概述

成就系统为玩家提供长期目标感，难度系统则让不同水平的玩家都能享受游戏。两者共同服务于一个目的：**让重玩有价值**。游戏不设"简单模式"——最低难度依然是需要认真对待的战术游戏，最高难度则是无情的铁人挑战。

---

## 一、难度系统

### 1.1 难度等级

| 难度 | 名称 | 核心差异 | 适合玩家 |
|------|------|----------|----------|
| 1 | 老兵 | 战术有容错，允许试错 | 战棋新手，想体验剧情 |
| 2 | 统帅 | 标准难度，设计基准 | 有战棋经验的玩家 |
| 3 | 传奇 | 高惩罚，资源紧张 | 硬核战棋爱好者 |
| 4 | 铁人 | 单存档+永久死亡 | 追求极限挑战的玩家 |

### 1.2 难度参数对照

| 参数 | 老兵 | 统帅 | 传奇 | 铁人 |
|------|------|------|------|------|
| 敌人属性修正 | ×0.85 | ×1.0 | ×1.15 | ×1.25 |
| 敌人AI策略 | 偏鲁莽 | 标准 | 偏战术 | 战术+本能混合 |
| AI失误注入率 | 30% | 15% | 5% | 0% |
| 战利品掉落率 | ×1.2 | ×1.0 | ×0.8 | ×0.7 |
| 经验获取率 | ×1.1 | ×1.0 | ×0.9 | ×0.8 |
| 金币获取率 | ×1.1 | ×1.0 | ×0.85 | ×0.7 |
| 士气惩罚 | ×0.7 | ×1.0 | ×1.3 | ×1.5 |
| 伤势恢复速度 | 快 | 标准 | 慢 | 慢 |
| 食物消耗 | ×0.8 | ×1.0 | ×1.2 | ×1.3 |
| 存档限制 | 多存档 | 多存档 | 多存档 | **单存档** |
| 角色永久死亡 | 可复活 | 可复活 | 可复活 | **永久死亡** |

### 1.3 AI难度调节

当前 `AIDifficultyConfig` 已实现失误注入机制。难度越高，AI越少犯错：

| AI行为 | 老兵难度 | 统帅难度 | 传奇难度 | 铁人难度 |
|--------|----------|----------|----------|----------|
| 选择最优策略概率 | 70% | 85% | 95% | 100% |
| 忽略低血量目标 | 经常 | 偶尔 | 极少 | 从不 |
| 利用地形掩护 | 偶尔 | 经常 | 总是 | 总是 |
| 包夹配合 | 罕见 | 偶尔 | 经常 | 总是 |
| 优先攻击治疗者 | 不主动 | 偶尔 | 经常 | 总是 |
| 集火脆弱目标 | 不主动 | 偶尔 | 经常 | 总是 |

### 1.4 难度与游戏阶段

难度在创建新游戏时选择，**不可在游戏中更改**（铁人模式全程锁定）。

| 阶段 | 难度影响 |
|------|----------|
| 雇佣兵时期 | 主要影响战斗AI和战利品 |
| 骑士时期 | 影响战斗AI+经济压力+食物消耗 |
| 领主时期 | 全面影响：战斗+经济+外交+领地防御 |

### 1.5 动态平衡（仅老兵难度）

老兵难度下，如果玩家连续3场战斗损失惨重（队伍死亡≥30%），自动触发以下缓冲：

| 缓冲 | 效果 | 持续 |
|------|------|------|
| 雇佣兵折扣 | 下次招募费用-25% | 1次 |
| 补给包 | 获得免费食物和药品 | 1次 |
| 敌人削弱 | 下场战斗敌人属性-10% | 1场 |

## 二、成就系统

### 2.1 设计原则

| 原则 | 说明 |
|------|------|
| 不可刷 | 成就必须在正常游戏流程中获得，不可通过重复操作刷取 |
| 可见进度 | 长期成就显示进度条（如"击杀100个哥布林：47/100"） |
| 分类清晰 | 按游戏系统分类，不杂乱 |
| 不剧透 | 隐藏成就用"???"显示，直到解锁 |
| 有意义 | 每个成就对应一种有意义的游戏体验，不是"点击1000次" |

### 2.2 成就分类

#### 战斗成就

| 成就 | 条件 | 奖励 |
|------|------|------|
| 初试锋芒 | 完成第一场战斗 | 称号：新兵 |
| 以一敌众 | 一场战斗中单人击杀≥5敌人 | 称号：屠夫 |
| 零伤通关 | 一场战斗中无任何人受伤 | 称号：完美指挥 |
| 暴击大师 | 一场战斗中触发≥10次暴击 | 称号：命运之手 |
| 逆转乾坤 | 在队伍≥50%倒下时翻盘获胜 | 称号：奇迹 |
| 灭团 | 一场战斗中全灭敌方（无一逃跑） | 称号：铁血 |
| 连锁反应 | 一个回合内触发≥3个连锁反应（技能联动） | 称号：战术大师 |
| 一回合清场 | 一回合内消灭所有剩余敌人 | 称号：闪电战 |

#### 角色成就

| 成就 | 条件 | 奖励 |
|------|------|------|
| 全能战士 | 单角色六维属性均≥16 | 称号：完美体魄 |
| 法术收藏家 | 收集≥20个不同法术 | 称号：博学者 |
| 装备大师 | 同时拥有每种武器类型各一把 | 称号：武器大师 |
| 骑兵冲锋 | 单次冲锋伤害≥50 | 称号：铁骑 |
| 技能盘满点 | 点亮技能盘上≥80%的节点 | 称号：全面发展 |

#### 战略成就

| 成就 | 条件 | 奖励 |
|------|------|------|
| 第一桶金 | 累计获得≥1000金币 | 称号：淘金者 |
| 富甲一方 | 累计获得≥50000金币 | 称号：大富翁 |
| 探索者 | 揭露大地图≥50%区域 | 称号：制图师 |
| 全图探索 | 揭露大地图100%区域 | 称号：世界行者 |
| 雇佣兵团 | 拥有≥10名英雄 | 称号：领袖 |
| 征服者 | 攻占第一座城堡 | 称号：征服者 |
| 统一大陆 | 占领所有主要据点 | 称号：大帝 |

#### 剧情成就

| 成就 | 条件 | 奖励 |
|------|------|------|
| 雇佣兵毕业 | 完成雇佣兵时期主线 | 称号：正式骑士 |
| 骑士加冕 | 完成骑士时期主线 | 称号：领主 |
| 最终胜利 | 完成领主时期主线 | 称号：传奇 |
| 同伴知己 | 任意同伴关系达到"生死之交" | 称号：挚友 |
| 化解偏见 | 化解一对种族冲突（精灵+半兽人） | 称号：和平使者 |
| 真相大白 | 完成一个同伴的个人任务 | 称号：知己 |

#### 隐藏成就

| 成就 | 条件 | 显示 |
|------|------|------|
| ??? | 背叛一个忠诚度满的同伴 | 解锁后显示："冷血" |
| ??? | 在暴风雪中战斗并获胜 | 解锁后显示："暴风之子" |
| ??? | 让一个角色从1级升到满级 | 解锁后显示："不朽传奇" |
| ??? | 在铁人模式通关 | 解锁后显示："真·传奇" |
| ??? | 全灭己方队伍 | 解锁后显示："自作自受" |

### 2.3 成就奖励

成就奖励**仅限称号**，不影响数值平衡：

| 奖励类型 | 说明 |
|----------|------|
| 称号 | 在角色名下方显示，纯装饰 |
| 百科解锁 | 部分成就解锁隐藏百科条目 |
| 统计记录 | 成就页面记录完成时间和条件 |

## 三、统计面板

### 3.1 记录的数据

| 类别 | 统计项 |
|------|--------|
| 战斗 | 总战斗数、胜/败、击杀总数、最高单体伤害、总伤害 |
| 角色 | 累计获得XP、最高等级、法术数量、装备更换次数 |
| 经济 | 累计收入/支出、最大单笔收入、最贵装备 |
| 战略 | 行走距离、探索百分比、攻城次数、招募次数 |
| 同伴 | 招募/失去同伴数、最高关系值、化解冲突次数 |

### 3.2 统计面板入口

主菜单 → "统计" 按钮或游戏内暂停菜单 → "统计"。

---

## 四、铁人模式特殊规则

### 4.1 核心限制

| 规则 | 说明 |
|------|------|
| 单存档 | 只有一个存档位，自动保存覆盖 |
| 不可重载 | 不允许手动读档（退出游戏后自动加载） |
| 永久死亡 | 角色死亡不可复活，永久失去 |
| 不可调难度 | 选定后全程锁定 |

### 4.2 铁人模式专属内容

| 内容 | 说明 |
|------|------|
| 死亡记录 | 在统计面板记录每个阵亡角色的名字和死因 |
| 纪念墙 | 主菜单的"阵亡将士"列表 |
| 专属称号 | 铁人通关获得"真·传奇"称号 |
| 专属成就 | "铁人之路"系列成就 |
