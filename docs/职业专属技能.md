# 职业专属技能

> **剑与六芒星（Sword & Hex）** 职业称号对应技能设计
>
> 每个职业称号拥有一个专属技能，当角色的职业称号匹配时自动获得。
> 技能设计原则：**机制独特 > 数值强度**，每个技能提供一种其他技能无法复制的战术选择。

---

## 设计原则

- **称号触发**：角色当前职业称号匹配时，该技能自动可用（无需额外技能点）
- **不锁定**：称号是动态的，称号变了技能也跟着变
- **AP体系**：MaxAP = 12 + Mod(DEX) + floor(Mod(CON)/2) - 护甲AP惩罚。技能消耗具体AP值
- **命中判定**：d20 + 专精加成 + 武器修正 ≥ 目标AC
- **伤害公式**：Total = Dice × (100% + floor(√STR)×10% + 精通加成%)
- **暴击**：d20自然骰 ≥ 暴击阈值(WIS影响) → 伤害×2（技能盘可提至×3）
- **穿透**：d20自然骰 ≥ DR阈值 → 按伤害类型分配HP/DR
- **平衡基准**：每个技能≈1个大节点的强度，AP消耗参考现有技能（2-8AP）
- **机制优先**：优先设计「能做到别人做不到的事」，而非「数值比别人高」

## 一、单属性职业（6）

### 战士 / Warrior（STR）

**碎甲打击 / Armor Break**
- 类型：主动 · 近战 · 5 AP
- 效果：对相邻目标发动一次近战攻击（正常命中判定）。命中后：目标DR阈值-5（更容易穿透），持续2回合。若目标无DR（已损毁或未穿甲），改为目标AC-1，持续2回合。此效果不叠加，刷新持续时间。
- 机制核心：让重甲敌人变脆，轻甲/无甲敌人也受AC惩罚

### 游侠 / Ranger（DEX）

**散射回避 / Evade Volley**
- 类型：被动
- 效果：被远程攻击命中时（命中判定通过后、伤害结算前），可立即向任意相邻空格位移1格，然后以新位置重新计算该次攻击的命中（新AC、新掩护）。若新位置使攻击未命中，该次攻击失效。每回合限触发1次。不消耗AP。
- 机制核心：移动即闪避，把位置变成防御资源

### 守卫 / Guardian（CON）

**铜墙铁壁 / Living Wall**
- 类型：主动 · 自身 · 4 AP
- 效果：本回合控制区扩展至周围6格（代替默认正面3格）。站在通道（两侧相邻格有不可通过地形或单位）中时，敌方不可用任何技能/效果穿越控制区（即使该效果声明忽略控制区）。持续到本回合结束。
- 机制核心：真正锁死通道，不可绕过的封路型坦克

### 法师 / Mage（INT）

**以太过载 / Arcane Overload**
- 类型：主动 · 自身 · 4 AP
- 效果：消耗当前剩余法力（至少1点），下一个施放的法术获得加成：DC + 消耗法力×1，法术伤害骰额外+消耗法力。若消耗≥5点法力，法术范围+1格。持续到本回合结束或施放1个法术后解除。
- 机制核心：把法力资源转化为爆发力，倾泻所有法力一波带走

### 刺客 / Assassin（WIS）

**弱点暴露 / Expose Weakness**
- 类型：主动 · 需视线 · 3 AP
- 效果：选择视野内一个目标进行标记。标记期间（持续2回合）：该目标被任何友军攻击时，暴击阈值降低3（更容易暴击）。若目标当前HP<100%，暴击阈值额外再降2。同一时间只能标记1个目标。
- 机制核心：把WIS的暴击属性转化为团队资源，集火标记

### 诗人 / Bard（CHA）

**战歌切换 / Battle Hymn**
- 类型：主动 · 自身 · 2 AP
- 效果：在三种战歌中切换（每次使用消耗2AP），当前战歌对周围3格内友军（含自身）生效，持续到下次切换或战斗结束：
  - **进军号 / March**：移动消耗-1AP/格（最低1AP/格）
  - **铁壁颂 / Bulwark**：AC+1
  - **嗜血曲 / Fury**：近战/远程伤害+10%
- 机制核心：灵活的战场光环，用少量AP换全局微调

## 二、双属性复合职业（15）

### 剑舞者 / Blade Dancer（STR + DEX）

**连旋斩 / Whirling Strike**
- 类型：主动 · 近战 · 6 AP
- 效果：对正面3格扇形范围内所有敌人各发动一次近战攻击。每次命中后自身旋转移动1格（顺时针或逆时针，玩家选择），旋转移动不触发借机攻击。每次攻击伤害为正常伤害的70%（乘数0.7x）。
- 机制核心：AOE + 位移一体，砍完就换位

### 重战士 / Juggernaut（STR + CON）

**不可阻挡 / Unstoppable**
- 类型：被动
- 效果：冲锋时路径上的第一个轻型掩体（栅栏、灌木等）被破坏；路径上的友方单位被推移1格（不造成伤害）。冲锋不被任何效果中断。冲锋后获得临时HP = Mod(CON)×3，持续1回合。
- 机制核心：真正的不可阻挡——破障推友，冲锋必达

### 魔剑士 / Spellsword（STR + INT）

**符文武器 / Rune Imbue**
- 类型：主动 · 自身 · 3 AP
- 效果：为当前主手武器注入符文（持续3回合）。下一次近战攻击命中时额外造成1d6火焰伤害 + 目标燃烧1回合（每回合1d6火伤）。符文在攻击命中后消耗。同一时间只能维持1个符文。冷却3回合。
- 机制核心：法术资源转化为武器附魔，低频高质

### 处刑者 / Executioner（STR + WIS）

**终结宣告 / Death Sentence**
- 类型：被动
- 效果：攻击HP≤30%的目标时，暴击阈值降低3，且暴击伤害倍率额外+0.5x（如基础×2变为×2.5，技能盘×3变为×3.5）。对已处于流血/中毒状态的目标暴击阈值额外再降1。
- 机制核心：收割型被动，越残血越危险

### 军阀 / Warlord（STR + CHA）

**身先士卒 / Lead from Front**
- 类型：被动
- 效果：冲锋时，冲锋终点相邻的每个友军可立即向同一方向免费移动1格（不消耗AP，不触发借机攻击）。此效果每回合对每个友军限触发1次。冲锋自身伤害加成正常（1.25x）。
- 机制核心：一个人冲锋带着整条线推进

### 决斗家 / Duelist（DEX + CON）

**以伤换伤 / Riposte**
- 类型：被动
- 效果：被近战攻击命中后，若攻击者在本单位正面3格内且本回合反击次数未耗尽，立即发动反击。此反击伤害为正常伤害的100%（而非默认50%）。若此次反击暴击，恢复2AP。每回合限触发1次。
- 机制核心：高风险高回报的换血——挨打必还手，暴击回AP

### 秘射手 / Arcane Archer（DEX + INT）

**魔矢追踪 / Homing Shot**
- 类型：主动 · 远程 · 6 AP
- 效果：射出一支追踪箭矢。此次射击：无视半掩体和全掩体的命中惩罚/阻挡，射程+3，命中后额外造成1d8奥术伤害（不参与武器伤害公式，固定骰）。箭矢可绕过1个障碍物（不绕墙壁）。冷却2回合。
- 机制核心：反掩体远程，躲在树后也能命中

### 猎鹰手 / Falconer（DEX + WIS）

**鹰眼锁定 / Hawk's Mark**
- 类型：主动 · 需视线 · 3 AP
- 效果：标记视野内一个目标（持续3回合）。被标记的目标：失去所有地形AC加成和掩护效果，被远程攻击时攻击者射程视为+1。同一时间只能标记1个目标。
- 机制核心：把掩体变成纸，远程点名

### 浪客 / Rogue（DEX + CHA）

**声东击西 / Misdirection**
- 类型：主动 · 5 AP
- 效果：选择一个相邻敌方单位，强制其转向指定方向（消耗其本回合免费转向机会）。然后自身立即进入潜行状态。该目标下回合第一次攻击检定获得劣势。冷却2回合。
- 机制核心：操控朝向+自身潜行，为包夹创造条件

### 战法师 / Battlemage（CON + INT）

**法力护盾 / Mana Shield**
- 类型：被动
- 效果：受到伤害时（DR穿透判定之后、HP扣除之前），可自动消耗1点法力将伤害降低 Mod(INT)×2 点。法力为0时不触发。此效果不消耗AP，不消耗行动。每回合触发次数无上限（有法力就能挡），但每次触发最低降低1点伤害。
- 机制核心：法力值变成无限次微减伤，持续续航

### 老兵 / Veteran（CON + WIS）

**临危不乱 / Old Timer**
- 类型：被动
- 效果：HP首次降至50%以下时立即触发（整场战斗1次）：本回合AC+3，反击范围扩展至周围6格（代替默认正面3格），所有反击伤害×1.5。效果持续到本回合结束。
- 机制核心：血量跌破一半时的绝境反击

### 铁壁将军 / Iron Commander（CON + CHA）

**坚守阵线 / Hold the Line**
- 类型：主动 · 自身 · 4 AP
- 效果：周围2格内所有友军进入「阵型锁定」：本回合不可移动，但获得AC+2且免疫恐惧和溃逃。若友军处于防御模式，额外获得伤害减免3点。持续到本回合结束。
- 机制核心：用移动换取防御，阵地战核心

### 贤者 / Sage（INT + WIS）

**预知回避 / Forewarning**
- 类型：被动
- 效果：每回合第一次被攻击时（无论近战/远程/法术，在d20掷出后、命中判定前），可选择消耗1点法力使该次攻击获得劣势。若选择不消耗或法力为0，此效果不触发。无论是否触发，本回合不再重复。
- 机制核心：预判闪避，知识即防御

### 术士 / Sorcerer（INT + CHA）

**血脉共鸣 / Blood Resonance**
- 类型：主动 · 自身 · 4 AP（HP<30%时免费，0AP）
- 效果：消耗自身当前HP的10%（最少5点，向下取整），恢复消耗HP×2点法力。若自身HP低于30%时使用，不消耗AP（但仍扣HP），恢复法力量×3（濒死时血液共鸣更强）。此技能造成的HP消耗不触发伤势惩罚。
- 机制核心：血换蓝的极限手段，伤势豁免让风险可控

### 先知 / Prophet（WIS + CHA）

**命运干涉 / Fate Intervention**
- 类型：被动 · 每场战斗1次
- 效果：当任意一个视野内友军被攻击命中且伤害会导致其HP归零时，可宣布「干涉」——强制该次攻击的伤害骰重掷一次（取两次中较低的结果）。若重掷后伤害仍致死，先知自身受到该次伤害的25%（向上取整）作为反射伤害。
- 机制核心：救命技能，有限次数，有代价

## 三、三属性复合职业（20）

### 斗士 / Bruiser（STR + DEX + CON）

**铁壁冲锋 / Iron Rush**
- 类型：主动 · 7 AP
- 效果：向目标方向直线冲锋最多3格（正常冲锋判定）。冲锋期间AC+3（不会被借机攻击打断）。命中第一个敌方单位时造成一次正常近战攻击伤害。冲锋结束后AC+3持续到本回合结束。路径上有友军时可将其推移1格（不造成伤害，友军不触发借机攻击）。
- 机制核心：推友+冲锋+临时防御，多功能笨重

### 魔武者 / Spellweaver（STR + DEX + INT）

**瞬发符印 / Instant Glyph**
- 类型：主动 · 3 AP
- 效果：在脚下或相邻空格放置一个符印（持续3回合）。任何单位经过或停留在符印格时触发：友军恢复1d4 HP，敌军受到1d4奥术伤害。每个符印只触发1次后消失。场上最多存在2个符印。
- 机制核心：地形控制，把位置变成资源

### 鹰眼猎手 / Hawkeye（STR + DEX + WIS）

**致命弹道 / Kill Shot**
- 类型：主动 · 远程 · 8 AP
- 效果：进行一次远程攻击。此次攻击暴击阈值降低4，且暴击时伤害倍率额外+1.0x（如基础×2变为×3）。若此攻击未命中（d20判定失败），返还4AP。射程无上限但有视线要求。冷却3回合。
- 机制核心：要么一箭要命，要么白打不亏

### 战神 / Champion（STR + DEX + CHA）

**战吼冲锋 / War Cry Charge**
- 类型：主动 · 7 AP
- 效果：冲锋后发动近战攻击（正常冲锋优势+1.25x伤害）。冲锋终点周围3格内所有友军士气+8，且这些友军下回合第一次攻击获得优势。冷却3回合。
- 机制核心：冲锋+士气爆发，团战发动机

### 铁焰魔战 / Ironweaver（STR + CON + INT）

**符文壁垒 / Rune Barricade**
- 类型：主动 · 5 AP
- 效果：在相邻空格召唤一个符文屏障（占据1格，阻挡移动和视线）。屏障HP = Mod(CON)×5。屏障被摧毁时对周围1格内所有敌方造成1d6奥术伤害。场上最多1个屏障，持续3回合或被摧毁。
- 机制核心：人造墙壁+地雷，攻防一体

### 碎颅者 / Skullcrusher（STR + CON + WIS）

**弱点粉碎 / Crush the Weak Point**
- 类型：被动
- 效果：近战攻击时，若目标当前HP<100%，伤害额外+（目标已损失HP%÷10，向下取整）点。对DR仍有剩余的目标，DR穿透检定获得+3加值。
- 机制核心：越打越疼+穿甲专精，磨死铁罐头

### 征服者 / Conqueror（STR + CON + CHA）

**镇压 / Subjugate**
- 类型：主动 · 近战 · 5 AP
- 效果：对相邻目标发动近战攻击。命中后目标士气-15，且下回合被「恐惧」效果影响（不可主动攻击征服者，可攻击其他人，WIS豁免DC=10+Mod(CHA)可抵抗）。冷却2回合。
- 机制核心：士气打击+针对性恐惧，瓦解敌方核心

### 灭世骑士 / Doom Knight（STR + INT + WIS）

**毁灭凝视 / Gaze of Ruin**
- 类型：主动 · 需视线 · 4 AP
- 效果：选择视线内一个目标进行凝视（持续1回合，若目标已有负面状态则持续2回合）。被凝视的目标被任何友军攻击命中时，暴击阈值视为13（更容易暴击，但自然20暴击规则不变）。
- 机制核心：单点名标记+暴击放大

### 霸主 / Overlord（STR + INT + CHA）

**恐惧光环 / Aura of Dread**
- 类型：被动
- 效果：周围2格内敌方士气每回合-3。处于此范围内且士气≤-40的敌方单位攻击霸主时获得劣势。夜间效果范围+1格。
- 机制核心：持续士气压力，不知不觉就崩溃

### 圣战者 / Crusader（STR + WIS + CHA）

**圣光冲锋 / Holy Charge**
- 类型：主动 · 7 AP
- 效果：冲锋后发动近战攻击。此次冲锋无视地形移动惩罚（森林/丘陵/沼泽按平地1AP/格消耗）。冲锋路径上经过的每个格子留下圣光痕迹（持续2回合），友军经过痕迹格时恢复1d4 HP。冷却3回合。
- 机制核心：净化地形+治疗路径，开路先锋

### 影法师 / Shadow Mage（DEX + CON + INT）

**暗影置换 / Shadow Swap**
- 类型：主动 · 4 AP
- 效果：选择视野内一个空格（该格必须是此前施放法术时法术落点，最多记忆3个法术落点）。与该格交换位置（不触发借机攻击）。交换后获得1回合潜行。
- 机制核心：利用法术历史落点做传送，需要规划

### 夜行者 / Nightstalker（DEX + CON + WIS）

**暗杀标记 / Death Mark**
- 类型：主动 · 近战 · 5 AP
- 效果：对相邻目标施加暗杀标记（持续2回合）。标记期间：夜行者对该目标的所有攻击无视DR直接打HP，且攻击获得优势。若目标在标记期间HP被降至0，夜行者恢复3AP。冷却3回合。
- 机制核心：锁定目标+穿甲+击杀回AP，猎杀型刺客

### 千面客 / Faceless（DEX + CON + CHA）

**身份窃取 / Identity Theft**
- 类型：主动 · 4 AP
- 效果：选择视野内一个敌方单位进行模仿（仅对AI控制的敌方单位生效）。获得该单位的外观（不改变任何属性数值），敌方AI在本回合内不主动攻击千面客（视为友军）。模仿期间千面客可自由行动，但发动攻击时模仿解除。冷却4回合。
- 机制核心：暂时隐身于敌方，战术欺骗

### 星辰行者 / Stargazer（DEX + INT + WIS）

**星图定位 / Star Map**
- 类型：主动 · 3 AP
- 效果：选择一个视野内或此前记忆过的位置（最多记忆Mod(INT)个位置），在下回合开始时免费传送至该位置（不触发借机攻击，不可传送至有单位占据的格子）。每场战斗可使用次数 = Mod(INT)（最少1次）。
- 机制核心：预存传送点，高机动战术

### 幻术师 / Illusionist（DEX + INT + CHA）

**镜像分身 / Mirror Image**
- 类型：主动 · 4 AP
- 效果：在相邻空格生成1个幻影分身（HP=1，无攻击能力，AC=12）。敌方攻击幻术师时，若分身存在，50%概率攻击打到分身上（分身被击破，伤害由分身承受）。同时最多存在2个分身。每场战斗最多生成 Mod(CHA) 个分身（最少2个）。
- 机制核心：概率闪避，CHA越高分身越多

### 风语者 / Windwalker（DEX + WIS + CHA）

**顺风传递 / Tailwind**
- 类型：主动 · 4 AP
- 效果：选择视野内一个友军，将其传送到自身周围2格内的任意空位（不触发借机攻击）。传送后双方各获得本回合移动消耗-1AP/格（最低1AP）。冷却2回合。

## 四、四属性复合职业（15）

> 多属性技能的特点：适用场景广但单一效果不强，体现"广而不精"。

### 大贤者 / Archsage（CON + INT + WIS + CHA，缺STR DEX）

**万卷通鉴 / Omnibus**
- 类型：被动
- 效果：战斗开始时揭示敌方所有单位的最低豁免值（强韧/反射/意志中最低的一个）。每回合可消耗2AP对周围3格内1个友军发出战术建议——该友军下次攻击命中+1。同一友军不可叠加。
- 机制核心：信息优势+微量全局辅助

### 灵风大师 / Zephyr Master（DEX + INT + WIS + CHA，缺STR CON）

**风之眷顾 / Wind's Favor**
- 类型：被动
- 效果：每回合结束时若本回合移动了≥3格，获得1层「风痕」（最多3层）。每层风痕使下一次远程攻击射程+1。被近战攻击命中时失去全部风痕。
- 机制核心：移动积累射程奖励，被贴身即清零

### 荒野酋长 / Warchief（DEX + CON + WIS + CHA，缺STR INT）

**野性直觉 / Feral Instinct**
- 类型：被动
- 效果：在森林/密林/沼泽/荒原地形上时：移动力+1（每格移动消耗-1AP，最低1AP），不可被伏击（自动感知视野内的潜行单位），对野兽类敌人攻击获得优势。
- 机制核心：荒野主场优势，野外作战全面增强

### 暗影领主 / Shadow Lord（DEX + CON + INT + CHA，缺STR WIS）

**幕后操纵 / Puppet Master**
- 类型：主动 · 6 AP
- 效果：选择周围2格内一个士气≤-40的敌方单位，强制其向指定方向移动2格并攻击该方向最近的单位（友敌皆可，伤害×0.5）。冷却3回合。

### 寂灭之手 / Silent Death（DEX + CON + INT + WIS，缺STR CHA）

### 毁灭之主 / Lord of Ruin（STR + INT + WIS + CHA，缺DEX CON）

## 五、五属性复合职业（6）

> 五属性技能 = 实用但平庸，不如任何专精技能强。

**万法通识 / Jack of All Trades**

- 机制核心：微弱全局增幅，没有任何独特机制

### 山岳之主 / Mountain Lord（缺DEX）

**磐石姿态 / Mountain Stance**
- 类型：主动 · 自身 · 4 AP
- 效果：本回合不可移动。获得DR阈值+3，免疫所有负面状态。下回合开始时AC-2持续1回合（僵直代价）。

### 暮光行者 / Twilight Walker（缺CON）

**暮光步 / Twilight Stride**
- 类型：被动
- 效果：每回合第一次移动时不触发借机攻击，且可穿越敌方占据的格子（不可停留在有单位的格子上）。移动结束后若HP<50%，恢复1d4 HP。

### 铁血暴君 / Iron Tyrant（缺WIS）

**暴君之怒 / Tyrant's Wrath**

- 效果：自身造成的伤害永远不因任何debuff降低（免疫虚弱、士气低落等减伤效果）。代价：暴击阈值固定为20（仅自然20暴击），不可被任何效果降低。
- 机制核心：稳定输出永不衰减，代价是无法暴击

### 孤影行者 / Lone Shadow（缺CHA）

**独行术 / Lone Operative**

- 效果：周围2格内每有1个友军，自身AC-1（最低AC=8）。周围2格内无友军时：AC+2，移动消耗-1AP/格（最低1AP/格）。进入战斗时若周围2格内无友军，获得1回合潜行。
- 机制核心：越孤立越强，越扎堆越弱

### 怒涛战神 / Wrath Avatar（缺INT）

**野蛮直觉 / Savage Instinct**

- 机制核心：本能猎杀，无需思考的杀戮直觉

### AP消耗分布参考

技能AP消耗严格对齐现有技能盘大节点：
- 轻量辅助/标记：2-3 AP（对标 shield_bash 5AP的轻量版）
- 中等主动：4-6 AP（对标 double_attack 4AP、aimed_shot 8AP中间值）
- 重量主动：7-8 AP（对标旋风斩 5AP + 冲锋成本、流星箭雨级别）
- 被动：0 AP（常态生效）
