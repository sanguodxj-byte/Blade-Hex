# 职业专属技能（v1.0 分阶版本）

> **剑与六芒星（Sword & Hex）** 职业称号对应的专属技能系统
>
> 一至四属性职业称号提供**常驻被动效果**；五属性职业提供**每场战斗 1 次主动技能**；六属性职业提供**代价型主动技能**。
> 数据权威源：`BladeHexCore/src/SkillTree/CareerSkillRegistry.cs`

---

## 设计核心

1. **分阶生效** — 一至四属性为常驻被动；五属性为每场战斗 1 次主动；六属性为每回合可用但带永久代价的主动
2. **高阶称号是完全不同的效果** — 不是低阶称号的简单数值升级，而是全新的战斗机制
3. **不引入额外数据层** — 所有效果基于已有战斗系统（伤害公式、移动规则、护甲/DR体系、法力系统）
4. **不依赖属性修正值** — 修正值范围仅 1-5，不适合作为技能触发/数值核心；采用百分比、固定机制、特殊规则
5. **现实逻辑可解释** — 每个被动都有直观的实战解释
6. **仅当前称号生效** — 角色只获得当前职业称号对应的专属技能效果

### 世界观基调（低魔西幻 / 无神圣魔法）

- "圣光、神圣、神迹" → 改为"信念、誓言、热情、纯粹意志"等世俗描述
- "奥术 / 魔法 / 符文" → 保留，作为稀有但合理的技艺
- WIS = 感知 → 暴击 / 抗暴击 / 洞察弱点

### 设计说明

本版本从 v0.8 的"消耗全部 AP + 每场 1 次"主动大招系统重构为分阶职业技能系统。核心动因：
- 加深技能盘的养成深度，鼓励玩家探索不同点法
- 一至四属性通过常驻被动定义战斗风格
- 五属性保留高影响力主动，作为稀有复合职业的战局按钮
- 六属性用强恢复换取逐步累积的行动禁令，体现"广度"的代价

## 通用规则

### 生效方式

- 职业技能由角色**当前职业称号**决定；称号变化后，旧称号技能失效，新称号技能生效。
- 队伍中可出现重复职业称号；同种 Buff 不叠加，只保留一个生效。
- 一至四属性职业技能为常驻被动，不占行动、不消耗 AP、无触发次数限制（除非效果中特别注明）。
- 五属性职业技能为主动技能，默认消耗最大 AP；AP 不满无法使用，且每场战斗 1 次。
- 六属性职业技能为代价型主动技能，按技能文本记录已抽取代价，不会重复抽取同一代价。

### 数值采用

- 所有百分比效果基于面板数值计算。
- 所有特殊机制基于已有战斗系统（详见《03-战术战斗系统.md》）。
- 与主动技能、法术、物品效果**加法叠加**。
- **不依赖属性修正值**：所有效果使用百分比、固定乘数或特殊规则，避免因修正值低（1-5）导致效果微弱。

### 触发型被动

部分被动有严格触发条件（如"背面攻击"、"移动≥3 格"、"命中后"等），系统自动检测条件，条件满足时自动生效。

## 一、单属性职业（6） · 基础被动

### 战士 / Warrior（STR）

- 常驻被动 · 近战武器伤害 +20%
- **解释**：重装战士的每一次挥砍都势大力沉，在基础伤害上直接放大。

### 游侠 / Ranger（DEX）

- 常驻被动 · 远程攻击无视一半障碍物命中减益
- **解释**：久经训练的射手总能找到弹道的缝隙，障碍物对 ta 的影响减半。

### 守卫 / Guardian（CON）

- 常驻被动 · 每回合首次受到的伤害 -30%
- **解释**：经验丰富的前排懂得如何用盾牌和甲胄最厚的部分迎接第一击。

### 法师 / Mage（INT）

- 常驻被动 · 法术伤害 +15%
- **解释**：对魔法能量的操控更为激进，每一发法术都注入了更多魔力。

### 刺客 / Assassin（WIS）

- 常驻被动 · 背面攻击伤害 ×2
- **解释**：刀刃从不正面相迎——精准命中要害，伤害翻倍。

### 诗人 / Bard（CHA）

- 常驻被动 · 周围 2 格内友军伤害 +5%
- **解释**：无需吟唱，仅仅是 ta 在战场上的存在就能鼓舞周围人的斗志。

## 二、双属性复合职业（15） · 进阶被动

### 剑舞者 / Blade Dancer（STR + DEX）

- 常驻被动 · 近战命中后，强制对另一个相邻敌方发动一次额外攻击（每回合 1 次）
- 该额外攻击不会再次触发旋斩
- **解释**：剑锋所至，顺势回旋——命中一个目标后借力打力，削向旁边另一个。单回合最多触发一次，不会连斩。

### 重战士 / Juggernaut（STR + CON）

- 常驻被动 · 移动至少 3 格后的近战攻击，伤害 +50%
- **解释**：助跑距离越远，冲击力越强。冲锋起来的重战士是一辆人形战车。

### 魔剑士 / Spellsword（STR + INT）

- 常驻被动 · 近战攻击时，消耗当前法力 5%，命中造成等额额外伤害；法力为 0 时无效
- **解释**：在剑刃上附着魔力，每次挥砍都带出魔法能量的余波。

### 处刑人 / Executioner（STR + WIS）

- 常驻被动 · 对 HP ≤ 30% 的敌人伤害 +50%
- **解释**：处刑人专攻垂死之敌——越是虚弱的对手，越逃不过致命一击。

### 征讨者 / Warlord（STR + CHA）

- 常驻被动 · 周围 2 格内每有一个友军（含自身），所有受影响友军造成的伤害 +5%（最多 +20%）
- **解释**：征讨者的战旗所在，便是军队的意志所向——兵越多，势越盛。

### 决斗者 / Duelist（DEX + CON）

- 常驻被动 · 被近战攻击后，立即对攻击者发动一次近战攻击（需命中判定），造成正常伤害（每回合 1 次）
- **解释**：决斗者的剑从不虚发——每一次还击都必须精准命中。

### 秘射手 / Arcane Archer（DEX + INT）

- 常驻被动 · 对同一目标连续远程攻击时，每次命中后下一次伤害 +10%（最多 +30%，切换目标重置）
- **解释**：奥术标记会附着在目标身上，后续箭矢自动微调弹道，越射越准。

### 狩猎者 / Falconer（DEX + WIS）

- 常驻被动 · 攻击已受伤的敌人时，命中 +30%
- **解释**：狩猎者循着血迹追踪猎物——受伤的目标动作迟钝，更容易被命中。

### 游荡者 / Rogue（DEX + CHA）

- 常驻被动 · 近战攻击未命中时，返还 1 AP（不限次数）
- **解释**：游荡者的剑法虚实难辨，虚招试探不中也可以快速收势，不浪费体力。

### 战法师 / Battlemage（CON + INT）

- 常驻被动 · 受到伤害时，自动消耗 1 法力抵消 1 点伤害；法力为 0 时不触发
- **解释**：战法师将法力编织成护盾——每一分魔力都用来抵消等量的冲击。

### 苦修者 / Veteran（CON + WIS）

- 常驻被动 · HP 低于 80% 时，暴击率 +5%
- **解释**：久经沙场的老兵在受伤后反而能看穿敌人的破绽，精准反击。

### 守御者 / Iron Commander（CON + CHA）

- 常驻被动 · 自身和相邻友军 AC +2
- 若友军本回合未移动，则该友军受到的近战伤害 -20%

### 大贤者 / Sage（INT + WIS）

- 常驻被动 · 法力值为 100% 时，法术伤害 +50%；法力值 >50% 且 <100% 时，法术伤害 +30%
- **解释**：大贤者精于法力的精妙调控——魔力越充沛，法术越磅礴。

### 指引者 / Sorcerer（INT + CHA）

- 常驻被动 · 法术伤害使目标 AC -25%（向下取整），持续 3 回合
- **解释**：指引者的法术会扭曲目标周围的魔力场，让队友的攻击更容易命中。

### 预言者 / Prophet（WIS + CHA）

- 常驻被动 · 周围 2 格内友军受到致命伤害时，该次伤害 -20%
- **解释**：预言者能瞥见命运的轨迹——在致命一击到来前稍稍拨转它的方向。

## 三、三属性复合职业（20） · 高阶被动

### 大宗师 / Grandmaster（STR + DEX + CON）

- 常驻被动 · 近战命中后，自身近战伤害 +5%（最多 +50%）
- **解释**：大宗师在战斗中不断攀升气势——每一刀都比上一刀更重。


### 魔武者 / Spellweaver（STR + DEX + INT）

- 常驻被动 · 近战命中后，下一次法术伤害 +8%；施法命中后，下一次近战伤害 +8%（各最多叠 3 层）
- **解释**：魔武者身随意转，剑与法交替不息——物魔双修，愈战愈强。

### 审判官 / Hawkeye（STR + DEX + WIS）

- 常驻被动 · 自身 AP 为最大值时，该次攻击暴击率 +20%
- **解释**：审判官蓄势待发——满状态下的第一击永远是最精准的。


### 战誓者 / Champion（STR + DEX + CHA）

- 常驻被动 · 自身移动 3 格后，本回合下一次近战攻击命中 +2
- 若该次攻击命中，相邻友军本回合下一次攻击伤害 +15%

### 述法者 / Ironweaver（STR + CON + INT）

- 常驻被动 · 自身未移动并结束回合时，消耗 50% 最大法力值，周围 1 格内友军本回合免疫法术伤害；法力为 0 时无效
- **解释**：述法者以法力为契，为战友刻下抗法的符文结界。


### 惩罚者 / Skullcrusher（STR + CON + WIS）

- 常驻被动 · 近战攻击穿透护甲时，不对护甲造成伤害，转而造成 200% 伤害直接作用于血量
- **解释**：惩罚者的战锤专找盔甲的缝隙——一旦破甲，伤害直取要害。


### 征服者 / Conqueror（STR + CON + CHA）

- 常驻被动 · 移动至少 5 格后，下次近战攻击对所有相邻敌方单位各发动一次攻击
- **解释**：征服者冲入敌阵，长刃过处无一合之敌。


### 毁灭者 / Doom Knight（STR + INT + WIS）

- 常驻被动 · 击杀敌人后，可立即免费释放一次法术（不消耗 AP 和法力）
- **解释**：毁灭者以敌人的生命为燃料——每一条逝去的灵魂都化作一道法术。

### 支配者 / Overlord（STR + INT + CHA）

- 常驻被动 · 周围 2 格内敌方命中 -10%；已受伤的敌方命中 -20%
- **解释**：支配者的存在本身就是一种压迫——越是受伤的敌人，越难以集中精神瞄准。

### 十字军 / Crusader（STR + WIS + CHA）

- 常驻被动 · 自身移动后，周围 1 格内友军下一次攻击伤害 +10%
- **解释**：十字军的冲锋本身就是信号——战友们看到 ta 出击便紧随而上。


### 影缄者 / Shadow Shroud（DEX + CON + INT）

- 常驻被动 · 回合结束时，若自身位于半掩体或全掩体地形，则获得 1 回合"影幕"
- 影幕期间：敌方对自身的远程攻击命中 -2；自身下一次远程攻击命中 +2、伤害 +15%
- 若触发时位于全掩体中，则该次远程攻击再额外无视一半障碍物命中减益

### 鹰眼卫 / Hawkeye Guard（DEX + CON + WIS）

- 常驻被动 · 远程攻击无视掩体和障碍物的命中减益

### 游骑兵 / Outrider（DEX + CON + CHA）

- 常驻被动 · 自身移动 5 格后，回合结束时，范围 1 格内友军获得"奔袭"
- 获得"奔袭"的友军下次行动可免费移动 5 格；这 5 格移动消耗完之前不会触发借机攻击
- "奔袭"提供的免费移动视为正常移动，可触发"移动 X 格后"类效果

### 唤星者 / Starcaller（DEX + INT + WIS）

- 常驻被动 · 自身本回合第一次法术的射程改为：所有比自身所处高度更低的地图格
- 被法术造成伤害的目标每比自身低 1 高度，该次法术伤害 +20%

### 幻术师 / Illusionist（DEX + INT + CHA）

- 常驻被动 · 自身释放法术后获得 1 层"幻影"
- 每层"幻影"使自身 AC +1，最多 10 层
- 自身被攻击后失去 1 层"幻影"

### 风语者 / Windwalker（DEX + WIS + CHA）

- 常驻被动 · 自身移动消耗的行动力固定为 1，不受地形影响
- 每移动 1 格，本场战斗暴击率 +1%，最多 +50%

### 敌法师 / Antimage（CON + INT + WIS）

- 常驻被动 · 若自身以 AP 和法力均为最大值的状态结束回合，则消耗所有法力
- 本场战斗剩余时间内免疫法术伤害；不免疫法术附带的非伤害效果

### 铁幕领主 / Iron Sovereign（CON + INT + CHA）

- 常驻被动 · 相邻友军和自身 AC +20%，向上取整

### 誓盾卫 / Oathshield（CON + WIS + CHA）

- 常驻被动 · 相邻友军受到暴击时，免除该次伤害

### 天选者 / Chosen One（INT + WIS + CHA）

- 常驻被动 · 自身法术可以暴击

- 本场战斗剩余时间内，紧邻的敌人护甲 100% 被穿透

### 铁血之律令 / Iron Tyrant（缺 WIS）

- 主动 · 全体友军 · 消耗最大 AP · AP 不满无法使用 · 每场战斗 1 次
- 所有友军获得 1 次"铁血"：下一次未命中的攻击转为暴击

### 孤星之刃影 / Lone Shadow（缺 CHA）

- 主动 · 紧邻敌方目标 · 消耗最大 AP · AP 不满无法使用 · 每场战斗 1 次
- 仅当自身周围 6 格内只有 1 个敌方单位时可使用
- 本场战斗剩余时间内，自身对该目标造成的所有伤害必中、必暴；该目标无法离开自身控制区

## 六、全属性职业（1） · 倍率 ×2.5

### 万象 / Paragon（六属性全开）

- 主动 · 自身 · 每回合 1 次
- 随机从"无法移动、无法施法、无法攻击"中抽取 1 个未被抽中过的代价施加给自身
- 随后恢复全部生命、全部法力和全部 AP
- 不会重复抽到同一个代价
- 代价持续至本场战斗结束；三个代价均被抽取后，本场战斗不能再次使用

> **设计注**：万能之手不是"借用别人的技能"，而是把"广度"具象化为一段 5 回合的全属性巅峰。
> 对一个把 25 点平均分到六属性、每条路只点 1 个大节点的角色来说，这 5 回合是他证明价值的全部窗口。

### 称号切换说明

角色只享有当前职业称号对应的专属技能效果。已经解锁但当前未显示为职业称号的组合，不提供额外被动或主动技能。

### 与 AI 的对称

AI 单位同样享有称号被动——玩家面对的精英敌人也拥有其职业称号对应的被动效果，无需额外处理。
