# 音效与音乐设计

## 概述

音效和音乐是游戏氛围的核心载体。本游戏采用古朴、粗粝的中世纪暗黑风格，音效设计需与此匹配——拒绝华丽和明亮，追求压抑、紧张、偶尔的温暖。

当前代码基础：`AudioManager.gd` 已完整实现 BGM 交叉淡入淡出、SFX 池化播放、场景枚举（MAIN_MENU/OVERWORLD/COMBAT/TOWN/DUNGEON/EVENT/TAVERN/VICTORY/DEFEAT）。

---

## 一、BGM 设计

### 1.1 场景音乐

| 场景（AudioManager.Scenario） | 风格 | 参考情绪 |
|-------------------------------|------|----------|
| MAIN_MENU | 管弦乐+合唱，史诗但不欢快 | "远方的传说，你即将踏入" |
| OVERWORLD | 环境音乐+民谣乐器 | "漫长旅途，天色阴沉" |
| COMBAT | 紧张鼓点+铜管，节奏感强 | "刀剑相击，生死一线" |
| TOWN | 民谣吉他+笛子，温暖但不欢快 | "短暂的安宁" |
| DUNGEON | 低沉弦乐+环境音 | "黑暗中有什么在移动" |
| EVENT | 钢琴独奏或管弦乐 | "命运在此刻转折" |
| TAVERN | 嘈杂环境音+手风琴/提琴 | "啤酒、喧哗和雇佣兵" |
| VICTORY | 号角+铜管，克制庆祝 | "今天活下来了" |
| DEFEAT | 低沉弦乐+风声 | "一切都结束了" |

### 1.2 区域变体

大地图不同区域应有不同的音乐变体：

| 区域 | 音乐变体 | 特色乐器 |
|------|----------|----------|
| 中央平原 | 主旋律变体 | 笛子、手鼓 |
| 霜冠山脉 | 寒冷/空旷 | 低音弦乐、风声 |
| 银叶森林 | 神秘/空灵 | 竖琴、鸟鸣 |
| 丘陵草原 | 粗犷/荒凉 | 角笛、鼓 |
| 焦土荒原 | 压抑/危险 | 低沉铜管、不和谐音 |
| 蛮荒沼泽 | 潮湿/不安 | 水声、蛙鸣、低频 |

### 1.3 BGM 切换规则

| 切换场景 | 过渡方式 | 时间 |
|----------|----------|------|
| 大地图→战斗 | 快速淡出+战斗BGM淡入 | 1秒 |
| 战斗→大地图 | 缓慢淡出+大地图淡入 | 2秒 |
| 大地图→城镇 | 交叉淡入淡出 | 1.5秒 |
| 任何→主菜单 | 快速淡出 | 0.5秒 |

> 当前 `AudioManager` 已实现双播放器交叉淡入淡出。

## 二、战斗音效（SFX）

### 2.1 武器攻击音效

| 武器类型 | 音效描述 |
|----------|----------|
| 长剑 | 金属撞击声，锐利短促 |
| 巨斧 | 重物劈砍声，低沉有力 |
| 长枪 | 穿刺声，尖锐但轻 |
| 弓箭 | 弓弦嗡鸣+箭矢破空 |
| 十字弩 | 机械击发声+弩箭入肉 |
| 法杖 | 沉闷击打声+微弱魔法余音 |
| 徒手/钝器 | 钝击声，闷响 |

### 2.2 法术音效

| 法术学派 | 音效风格 |
|----------|----------|
| 塑能 | 爆裂、炽热、冲击波感 |
| 防护 | 结晶感、屏障嗡鸣 |
| 幻术 | 空灵、回声、不确定感 |
| 死灵 | 低语、骨裂、阴冷 |
| 变化 | 变形、扭曲、过渡感 |
| 附魔 | 迷惑、催眠、精神冲击 |
| 预言 | 清脆、透视感 |
| 咒唤 | 召唤、空间裂开 |

### 2.3 战斗事件音效

| 事件 | 音效 |
|------|------|
| 命中 | 短促打击声 |
| 未命中 | 挥空/弹开声 |
| 暴击 | 金属碎裂+重低音 |
| 失误 | 尴尬的"嗡"声 |
| 角色死亡 | 沉重倒地声 |
| 士气崩溃 | 恐慌喊叫+金属丢弃声 |
| 回合开始 | 号角（我方）/鼓声（敌方） |
| 战斗胜利 | 凯旋号角 |
| 战斗失败 | 哀号+风声 |

## 五、音频优先级与限制

### 5.1 同时播放上限

| 类型 | 上限 | 说明 |
|------|------|------|
| BGM | 1 | 已实现（双播放器交叉） |
| SFX | 10 | 已实现（池化播放） |
| 环境音 | 3 | 需实现 |
| 语音/对话 | 1 | 需实现 |

### 5.2 优先级

| 优先级 | 类型 | 说明 |
|--------|------|------|
| 最高 | 暴击/死亡/升级 | 关键战斗事件 |
| 高 | 武器攻击/法术 | 核心战斗音效 |
| 中 | UI交互 | 界面反馈 |
| 低 | 环境音 | 氛围填充 |
