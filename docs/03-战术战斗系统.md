# 战术战斗系统

## 概述
六边形网格上的回合制战斗。融合奇迹时代4、文明6、Battle Brothers等经典战棋的战术要素，构建深度与可读性兼备的战斗体系。

**视觉表现**：采用 **HD-2D 风格**（3D六棱柱网格 + 2D Billboard 角色），以直观地展现高度差、高低地优势以及 3D 掩体对视线的真实物理阻挡。

本系统覆盖两种战斗规模（详见04-战略层系统），共享同一套基础规则。

**单位身份由装备决定：** 本文档中出现的"骑乘单位""远程单位""近战单位"等称谓，均指**当前装备状态**下的角色身份，不是固定标签。角色换装即换身份（详见06-装备与物品.md）。

---

## 一、六边形网格基础

### 坐标系
轴向坐标(axial coordinates)：(q, r)，尖顶(pointy-top)，6方向邻居。

### 朝向系统（Facing）
每个单位始终面向6个方向之一。朝向影响：
- 正面（3格）：单位面对的3个相邻六边形
- 侧翼（2格）：左右各1格
- 背后（1格）：正后方1格

```
          正面
        ↗ ↑ ↖
  侧翼 ← [单位] → 侧翼
        ↖ ↓ ↗
          背后
```

- 移动时自动面向移动方向（免费）
- 原地转向消耗0行动点（每回合限1次免费转向）
- 防御模式下可指定朝向

| 地形 | 移动消耗 | 防御加成 | 掩护等级 | 视野效果 | 特殊 |
|------|----------|----------|----------|----------|------|
| 平地 | 1 | — | 无 | 正常 | 默认地形 |
| 草地 | 1 | — | 无 | 正常 | 无特殊 |
| 稀树草原 | 1 | +1 AC | 无 | 正常 | — |
| 森林 | 2 | +2 AC | 半掩体 | 阻挡穿越视线 | 潜行加成 |
| 密林 | 3 | +3 AC | 全掩体 | 阻挡全部视线 | 潜行大幅加成 |
| 丘陵 | 2 | +2 AC | 半掩体 | 站在高处可越过低矮障碍 | 高地优势 |
| 山地 | 3 | +3 AC | 全掩体 | 高处视野+2 | 不可骑乘（坐骑无法进入） |
| 浅水 | 2 | -1 AC | 无 | 正常 | 火焰抗性+2，冰霜/雷电弱点 |

| 幸运草丛 | 1 | — | 无 | 正常 | 站上去暴击率+10%（1次攻击） |

### 高程系统
3级高程：**低地 / 平地 / 高地**

| 攻击方向 | 效果 |
|----------|------|
| 高地 → 平地 | 攻击获得优势，射程+1 |
| 高地 → 低地 | 攻击获得优势，射程+2 |
| 平地 → 高地 | 攻击获得劣势 |
| 低地 → 高地 | 攻击获得劣势，射程-1 |
| 同一高程 | 无加成 |

高处单位可越过低处的森林/建筑等障碍物进行远程攻击。

### 河流与渡河
- 沿河流相邻格移动视为困难地形（消耗+1）
- **渡河攻击**：攻击方跨越河流攻击时，攻击方获得劣势（模拟渡河时的混乱）
- 桥梁：视为道路，无惩罚
- 某些技能（如两栖攻击）可忽略渡河惩罚

## 三、视线与掩护

### 视线规则（Line of Sight）
- 远程攻击和部分法术需要视线
- **视线被阻挡**：森林、密林、山地、墙壁、建筑废墟阻挡穿越视线
- **单位不阻挡视线**：友方/敌方单位不阻挡视线（箭从头上飞过）
- **高度优势**：站在丘陵/山地的单位可越过低处的障碍物看和射击

### 掩护等级
| 掩护 | 来源 | 效果 |
|------|------|------|
| 无掩体 | 平地/草地/道路 | 远程攻击正常 |
| 半掩体 | 森林/丘陵/废墟 | 远程攻击命中率-40%（或获得AC+2 vs远程） |
| 全掩体 | 密林/山地/墙壁 | 完全不可被远程攻击，但AOE仍有效 |
| 自我掩护 | 防御模式 | 获得半掩体效果 |

**掩护只对远程攻击生效**，近战攻击无视掩护。

## 四、控制区与借机攻击

### 控制区（Zone of Control）
大部分近战单位向其**正面3格**投射控制区。

控制区效果：
1. **移动限制**：敌方单位进入控制区后，该回合**不能再移动到其他格**（被钉住）
2. **远程封锁**：处于敌方控制区内的远程单位**不能使用远程攻击**（被贴身）
3. **逃逸触发**：在敌方控制区内移动离开 → 触发借机攻击

### 无控制区的单位
- 远程单位（弓箭手/法师）不投射控制区
- 某些技能可忽略控制区（如骑兵冲锋、潜行单位）
- 飞行单位忽略地面控制区

### 借机攻击（Attack of Opportunity）
触发条件（任一）：
- 在敌方近战单位控制区内移动离开
- 在敌方近战单位控制区内使用远程攻击

结算：
- 借机攻击为**免费单次近战攻击**
- 伤害为基础武器伤害的50%
- 每个单位每回合只能发动1次借机攻击
- 被借机攻击的单位可以继续行动（除非死亡）

### 防御模式
单位消耗主行动进入防御模式：
- AC+2
- **免疫包夹伤害加成**
- **控制区扩展至全部6格**（而非仅正面3格）
- 指定面向方向
- 本回合不能再移动或攻击

## 五、包夹与伏击

### 包夹（Flanking）

#### 包夹判定
当攻击者从目标的**侧翼或背后**发起攻击时，视为包夹。

| 攻击方向 | 判定 | 效果 |
|----------|------|------|
| 正面攻击 | 非包夹 | 正常 |
| 侧翼攻击 | 包夹 | 伤害+25%，士气-3 |
| 背后攻击 | 背刺 | 伤害+50%，士气-5，不可反击 |
| 防御模式下的单位 | 免疫包夹 | 无加成 |

#### 多人夹击（Surrounding）
参考文明6的夹击加成——多名友方单位包围目标时，攻击者获得额外加成：

| 目标相邻的友方数量 | 攻击加成（叠加包夹） |
|-------------------|---------------------|
| 1个（仅攻击者） | 无加成 |
| 2个 | +1 命中 |
| 3个 | +2 命中，目标AC-1 |
| 4个+ | +3 命中，目标AC-2，伤害+10% |

注意：夹击加成需要友方单位在目标**不同方向**的相邻格上（不能都在同一侧）。

### 伏击（Ambush）

#### 伏击触发条件
攻击者处于**潜行/隐身**状态时发起攻击 → 伏击。

伏击效果：
- 此次攻击自动获得**优势**
- 被伏击单位本回合AC-2（措手不及）
- 伤害+25%
- 被伏击方士气-5
- 伏击后攻击者解除潜行

#### 伏击 vs 包夹
两者可叠加：从隐身状态背后攻击 = 伏击+背刺，伤害+75%。

### 反击（Retaliation）
- 近战单位被近战攻击命中后，如果攻击者在反击范围内，可以**立即进行一次反击**
- 反击伤害为基础伤害的50%
- 每个单位每回合只能反击1次
- **被包夹（侧翼/背后）时不可反击**
- 防御模式下的单位反击伤害=100%

## 六、士气系统

### 士气范围
每个单位的士气值范围：-60 到 +40

| 士气等级 | 范围 | 效果 |
|----------|------|------|
| 高昂 | +20 ~ +40 | 暴击率+20% |
| 正常 | -19 ~ +19 | 无效果 |
| 低落 | -39 ~ -20 | 失误率+20% |
| 崩溃 | -59 ~ -40 | 失误率+40%，AC-2 |
| 溃逃 | -60 | 强制向撤退点移动，忽略控制区和借机攻击 |

### 士气变动事件
| 事件 | 士气变化 |
|------|----------|
| 击杀敌方单位（相邻） | +10 |
| 击杀敌方单位（2格外） | +8 |
| 击杀敌方单位（3格外） | +6 |
| 击杀敌方英雄 | +6（相邻）/+4（远处） |
| 被包夹攻击 | -3 |
| 被暴击命中 | -5 |
| 友方单位死亡（相邻） | -10 |
| 友方单位死亡（2格外） | -8 |
| 友方单位死亡（3格外） | -6 |
| 友方英雄死亡 | -6（相邻）/-4（远处） |

### 英雄士气光环
英雄单位向周围友军投射士气光环：
- 相邻友军每回合+3士气
- 英雄死亡对周围友军士气打击极大
- CHA修正值影响光环范围（高CHA的英雄光环更大）

### 正面状态
| 状态 | 触发 | 效果 | 持续 |
|------|------|------|------|
| 祝福 | 祈福术 | 攻击/豁免+1d4 | 3回合 |
| 护盾 | 护盾术 | AC+5 | 1回合 |
| 加速 | 加速术 | 移动+2，额外一次次要行动 | 3回合 |
| 再生 | 再生术 | 每回合开始恢复1d6 HP | 3回合 |
| 隐身 | 隐身术 | 不可被直接瞄准，AOE有效，攻击后解除 | 到攻击为止 |
| 幻影 | 幻影分身 | 攻击者需先命中幻影（AC12），命中则消耗幻影 | 到被击破 |
| 临时HP | 神圣护盾等 | 额外HP层，先于本体HP消耗 | 到消耗为止 |

### 状态交互
- **燃烧 + 冰冻** → 互相解除
- **燃烧 + 油类** → 燃烧伤害翻倍
- **潮湿 + 冰系** → 冰冻持续时间+1
- **潮湿 + 雷系** → 雷电伤害+50%
- **中毒 + 燃烧** → 毒雾扩散（相邻格可能中毒）
- **隐身 + 攻击** → 伏击效果，解除隐身
- **沉默 + 法术** → 法术完全不可用
- **恐惧 + 士气崩溃** → 优先溃逃

## 八、视野与战争迷雾

### 视野范围
- 默认视野 = 周围6格
- 丘陵/山地单位视野+2
- 森林中单位对外视野-2，但对外视野中被发现难度+40%
- 夜间所有单位视野-2（除黑暗视觉）

### 战争迷雾
- 战斗地图初始有**迷雾区域**——未探索的格子不可见
- 单位移动时揭示视野范围内的格子
- 已揭示但不在当前视野内的格子显示**最后已知状态**
- 敌方单位在迷雾中不可见，潜行单位即使揭示区域也不可见（需感知检定）

### 伏击触发（大地图→战术地图）
- 大地图上被伏击时，战术地图的初始部署受影响：
  - 被伏击方集中在一个区域，阵型混乱
  - 伏击方分散部署，已占据有利位置
  - 被伏击方第一回合全员"措手不及"（AC-2）

## 九、特殊战术动作

### 冲锋
- 移动3格以上后发起近战攻击 = 冲锋
- 冲锋攻击获得**优势**
- 冲锋伤害+25%
- **骑乘冲锋**：有坐骑的单位冲锋伤害+50%（代替25%），且有高度优势
- 被长枪类武器迎击时冲锋无效且获得劣势（不论是否骑乘，长枪反一切冲锋）
- 沙地/沼泽地形不可冲锋
- 骑乘单位不可在密林/洞穴中冲锋

### 撤退
- 消耗全部行动点向撤退点移动
- 移动时忽略控制区
- 仍会触发借机攻击（但只有50%伤害）
- 到达撤退点后离开战场

### 过肩射击
- 弓箭手可越过1排友军射击
- 友方单位和攻击目标之间需有间隙
- 命中率-10%

### 推撞
- 特定技能/牛头人冲锋
- 目标被推后1-2格
- 被推向墙壁/悬崖时额外受到1d6伤害
- 被推向其他单位时双方各受到1d4伤害

### 抢占高地
- 向丘陵/山地移动的最后一格如果比攻击目标高 → 自动获得高地优势
- 从高地向下移动冲锋 → 额外+1格移动距离

## 十一、回合流程总览

```
┌──────────────────────────────────────────────────┐
│  战斗开始                                         │
│  - 双方按部署位置放置单位                           │
│  - 若为伏击：被伏击方"措手不及"（AC-2，首回合）      │
│  - 揭示初始视野                                     │
├──────────────────────────────────────────────────┤
│  我方回合                                          │
│  - 所有我方单位依次行动（玩家控制顺序）               │
│  - 每单位：移动 + 主行动 + 次要行动                  │
│  - 结束时未行动单位自动进入防御模式                   │
├──────────────────────────────────────────────────┤
│  敌方回合                                          │
│  - AI控制所有敌方单位                               │
│  - 同样的行动结构                                    │
├──────────────────────────────────────────────────┤
│  回合结束                                          │
│  - 状态效果结算（燃烧/中毒/再生等）                   │
│  - 环境事件结算（如有）                              │
│  - 士气自然恢复/衰减                                 │
│  - 回合计数+1                                       │
│  - 检查胜利/失败条件                                 │
└──────────────────────────────────────────────────┘
```

## 战斗结束条件
- 全灭敌方 → 胜利
- 我方全灭 → 失败
- 特殊条件（护送/生存N回合/占领目标区域等）

## 战斗奖励
- 经验值（按敌人威胁等级分配）
- 战利品（装备/金币/消耗品）
- 士气加成（胜利后全体士气+5）

## 十二、代码实现参考

### 战斗管理器状态机

`CombatManager` 管理战斗生命周期，状态转换如下：

```
INIT → PLAYER_TURN ⇄ ENEMY_TURN → COMBAT_END
```

| 状态 | 职责 |
|------|------|
| INIT | 单位注册、部署区初始化、先攻排序、初始化DR装甲耐久 |
| PLAYER_TURN | 等待玩家逐单位操作，死亡/胜利检查 |
| ENEMY_TURN | AI顺序执行各敌人行动，带延迟动画 |
| COMBAT_END | 战利品结算、经验分配、返回大地图 |

### 核心系统类

| 类 | 职责 |
|----|------|

### 派生属性公式（代码实际实现）

| 属性 | 公式 | 说明 |
|------|------|------|
| AP（行动点） | `4 + floor(sqrt((DEX + CON) / 8))` | 1级≈5AP，后期≈8AP |
| HP（生命值） | `baseMaxHp + skillTreeHpBonus` | baseMaxHp由生成器预含CON修正 |
| AC（闪避） | `10 + floor(sqrt(DEX/2)) + 盾牌 + 技能盘` | **不含护甲AC**，护甲只通过max_dex_bonus限制DEX |
| 暴击阈值 | `max(15, 20 - floor(sqrt(WIS / 2)))` | WIS越高暴击范围越宽 |
| 被暴击减免 | `max(0.2, 1.0 - floor(sqrt(WIS/2)) × 0.1)` | WIS越高被暴击伤害越低 |
| 伤害 | `武器骰(XdY) + 等级Nd20 + STR修正 + 技能盘` | Nd20 = min(6, 1+(lv-1)/20) 额外骰 |

### 命中判定流程（代码实际）

```
1. 计算攻击加值 = 属性修正(STR/DEX) + 专精加值 + 技能盘加成
2. 判定优劣势（高地/冲锋/士气/渡河/掩体）
3. 掷d20（优势取高，劣势取低）
4. 暴击判定：d20 >= 暴击阈值（WIS影响）
5. 失误判定：d20 == 1
6. 命中判定：total_attack >= target_ac
7. 擦伤判定：未命中但差值≤2 → 命中，伤害减半
```

### 装甲穿透系统（DR）

护甲提供额外的**装甲耐久(DR)**作为第二血条。攻击先过命中判定，再过穿透判定：

```
穿透判定：d20自然骰 >= 目标DR阈值
  ├── 穿透成功：
  │   ├── 砍伤(SLASH)  → 70%HP + 30%DR
  │   ├── 刺伤(PIERCE) → 100%HP（完全无视装甲）
  │   └── 钝伤(CRUSH)  → 30%HP + 70%DR（装甲损毁后×1.5）
  └── 未穿透：
      ├── 砍伤 → 0%HP + 40%DR（削甲）
      ├── 刺伤 → 0%HP + 10%DR（几乎不削甲）
      └── 钝伤 → 10%HP + 90%DR（始终穿透少量）
```

装甲损毁后（DR≤0），DR阈值降为0，所有攻击直接打HP。

> 代码实现见 `src/core/combat/CombatResolver.gd → _resolve_armor_penetration()`。
