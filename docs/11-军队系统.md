# 军队系统

## 概述

当玩家晋升为骑士/领主后，可以招募士兵组建军队。士兵以**兵种堆叠**形式参与战斗，与英雄单位并肩作战。战斗结算参考英雄无敌(HOMM)式设计。

**核心原则：兵种 = 装备方案。** 没有固定的"长枪兵""骑兵"分类——给一群士兵发长枪他们就是长枪兵，给他们发马就是骑兵。同一批人换个装备就是另一个兵种。

## 核心概念

### 英雄 vs 兵种

| 维度 | 英雄单位 | 兵种堆叠 |
|------|----------|----------|
| 来源 | 酒馆招募/剧情获得 | 兵营/村庄招募 |
| 属性 | 完整6维+特质+技能盘 | 简化属性（攻击/防御/HP/速度） |
| 装备 | 可自由更换 | 按**装备方案**整体配发，战斗外可换 |
| 成长 | 经验升级，选技能 | 不可升级（但训练场可提升基础属性） |
| 死亡 | 可复活（高阶法术） | 永久损失 |
| 战场占用 | 1格 | 1格（代表整个堆叠） |
| 操作 | 完整操控 | 简化操控（移动/攻击/特殊） |
| 身份 | 由当前装备决定 | 由**装备方案**决定 |

## 装备方案（兵种定义）

### 什么是装备方案
一个装备方案 = 给一批士兵的**完整配装清单**：
- 主武器 + 副武器
- 防具
- 是否配坐骑
- 是否配盾牌

**方案的属性 = 装备属性的总和**，不需要单独定义"兵种数据"。

### 方案示例

| 方案名 | 武器 | 防具 | 坐骑 | 盾牌 | 总计费用/人 | 实际身份 |

| 民兵 | 短剑 | 皮甲 | — | — | 5金 | 轻步兵 |
| 枪兵 | 长枪 | 锁甲 | — | — | 15金 | 反骑步兵 |
| 剑盾兵 | 长剑 | 锁甲 | — | 铁盾 | 18金 | 防御步兵 |
| 弓手 | 长弓+匕首 | 皮甲 | — | — | 20金 | 远程步兵 |
| 弩手 | 十字弩+短剑 | 锁甲 | — | — | 22金 | 重装远程 |
| 轻骑 | 长剑 | 皮甲 | 军马 | — | 25金 | 骑兵 |
| 枪骑 | 长枪 | 锁甲 | 军马 | — | 30金 | 反骑骑兵 |

### 方案的动态属性

一个方案的战斗属性由装备直接计算：

```
攻击力 = 武器伤害骰平均值 + STR修正(士兵基础STR=10, 修正=0)
防御力 = 防具AC + 盾牌加成
HP     = 基础HP(8) + 防具HP加成
速度   = 基础速度(4) + 坐骑加成
射程   = 武器射程（近战=1，长枪=2，弓=5-7）
特殊   = 武器属性 + 坐骑属性 + 防具属性
```

### 方案的核心特殊能力来自装备

| 特殊能力 | 来源 | 说明 |
|----------|------|------|
| 反骑兵（伤害×2） | 长枪 | 武器属性自带 |
| 冲锋（伤害+50%） | 坐骑 | 有马就能冲锋 |
| 盾墙（相邻友军+1AC） | 铁盾 | 持盾自带 |
| 双射（不移动时攻击2次） | 长弓/短弓 | 武器属性自带 |
| 破甲（目标AC-2） | 巨斧/十字弩 | 武器属性自带 |
| 骑射（马上射击） | 短弓/手弩 + 坐骑 | 组合效果 |
| 横扫（攻击相邻2目标） | 巨剑 | 武器属性自带 |
| 法术攻击 | 法杖 + 法术方案 | 组合效果 |

## 堆叠规则

### 堆叠上限
| 时期 | 每个堆叠上限 | 堆叠数量上限 |
|------|-------------|-------------|
| 骑士 | 20人 | 4个堆叠 |
| 领主 | 50人 | 7个堆叠 |

### 堆叠属性计算

```
堆叠总HP = 单兵HP × 当前人数
堆叠攻击力 = 单兵攻击 × sqrt(当前人数)  （取整）
堆叠防御力 = 单兵防御（不随数量增长）
```

### 伤害结算

```
攻击方输出 = 堆叠攻击力 + 命中检定加值
防御方减免 = 堆叠防御力 + AC修正
净伤害 = max(1, 攻击方输出 - 防御方减免)

// 攻击兵种堆叠时：
损失人数 = ceil(净伤害 / 单兵HP)

// 攻击英雄时：
英雄直接承受净伤害
```

### 士气与溃逃
- 堆叠损失达50%以上 → 触发士气检定
- 士气检定 = 意志豁免 vs DC 12
- 失败 → 溃逃（向地图边缘移动，不执行玩家指令）
- 溃逃的堆叠本回合不攻击
- 英雄在堆叠相邻时可"重整士气"（消耗主行动，DC 10指挥检定）

## 克制关系（装备决定，非兵种标签）

克制关系由**装备属性**自然产生，不需要查兵种对照表：

```
长枪 → 克制冲锋（任何有长枪的单位都能反骑，不只是"长枪兵"）
坐骑 → 冲锋加成（任何骑马的单位都有，不只是"骑兵"）
远程 → 克制无盾（射击无盾目标伤害×1.5）
重甲 → 克制远程（远程减伤50%）
范围法术 → 克制密集堆叠
高速度 → 克制法师团（快速贴身）
巨斧/弩 → 破甲（克制重甲）
```

**一个拿长枪的弓手也能反骑。** 装备组合创造战术，不是兵种标签。

## 换装 = 换兵种

### 战斗外换装
- 在城镇/营地可以给士兵堆叠更换装备方案
- 旧装备回库存，新装备从库存取出
- 换装消耗金币（新方案费用 - 旧方案回收价值）

### 换装示例
```
我有一堆"枪兵"(长枪+锁甲) × 20人
↓ 花费：(20金+80金) × 20 - 回收 = 差价
→ 变成"轻骑"(长剑+皮甲+军马) × 20人

同样是这20个人，换个装备就是骑兵。
```

### 战斗中不可换装
- 兵种堆叠在战斗中装备固定，不能切换
- 英雄可以切换（因为有主副武器），但堆叠不行
- 这就是英雄和兵种的核心区别之一

## 招募与维护

### 招募 = 招人 + 配装
```
1. 招募士兵（人）：村庄/兵营提供"人力"
   - 村庄：便宜但基础属性低
   - 兵营：标准价格，基础属性中等
   - 训练场：基础属性高

2. 配发装备（方案）：从库存中取出装备配给
   - 库存 = 你已有的武器/防具/马匹
   - 可以自己打造，也可以购买
   - 不同方案消耗不同库存
```

### 每日维护费
- 人力费：每人2金/天
- 坐骑费：每匹1金/天
- 装备损耗：重甲0.5金/天，轻甲无

### 补给消耗
- 每个士兵每天消耗1份补给
- 坐骑每天额外消耗1份
- 补给耗尽 → 士兵每天损失10%HP → 士气下降 → 溃逃率上升
