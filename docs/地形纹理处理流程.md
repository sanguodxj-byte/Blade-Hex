# 草地纹理处理流程

## 概述

战斗场景草地覆盖层使用 2D 精灵平铺在 hex 顶面。纹理来源为 AI 生成的铺满草图片，经过六边形 mask + 羽化处理后切割为 16 张 patch。

## 输入素材

- 1024×1024 铺满草纹理（RGB，无透明通道）
- 放置位置：`assets/` 根目录
- 命名示例：`铺满草_2K_YYYYMMDD.png`

## 处理步骤

### 1. 切割为 4×4 网格

将 1024×1024 图片切为 16 块 256×256 的 patch。

### 2. 六边形 Mask

每块 patch 应用平顶六边形 mask：
- 六边形中心 = 图片中心 (128, 128)
- 六边形半径 = `min(宽, 高) / 2 - 6`（留 6px 给羽化）
- 六边形方向：平顶（flat-top），与战斗场景 hex 一致

### 3. 高斯羽化

对六边形 mask 做 `GaussianBlur(radius=8)` 羽化，使边缘柔和过渡到透明。

### 4. 输出

- 格式：PNG，RGBA
- 尺寸：256×256
- 命名：`grass_00.png` ~ `grass_15.png`
- 输出目录：`assets/sprites/grass_patches/`

## 处理脚本

```python
import os, math
import numpy as np
from PIL import Image, ImageFilter, ImageDraw

input_path = "assets/铺满草纹理.png"
output_dir = "assets/sprites/grass_patches"

img = Image.open(input_path).convert('RGB')
w, h = img.size
os.makedirs(output_dir, exist_ok=True)

cols, rows = 4, 4
cell_w, cell_h = w // cols, h // rows

def draw_hex(draw, cx, cy, radius, fill=255):
    points = []
    for i in range(6):
        angle = math.radians(60 * i)
        px = cx + radius * math.cos(angle)
        py = cy + radius * math.sin(angle)
        points.append((px, py))
    draw.polygon(points, fill=fill)

for idx in range(16):
    row = idx // cols
    col = idx % cols
    x1, y1 = col * cell_w, row * cell_h
    patch_rgb = img.crop((x1, y1, x1 + cell_w, y1 + cell_h))

    mask = Image.new('L', (cell_w, cell_h), 0)
    draw = ImageDraw.Draw(mask)
    cx, cy = cell_w // 2, cell_h // 2
    hex_radius = min(cell_w, cell_h) // 2 - 6
    draw_hex(draw, cx, cy, hex_radius, fill=255)
    mask = mask.filter(ImageFilter.GaussianBlur(radius=8))

    patch_rgba = patch_rgb.convert('RGBA')
    patch_rgba.putalpha(mask)
    patch_rgba.save(os.path.join(output_dir, f"grass_{idx:02d}.png"))
```

## 运行时行为（GrassOverlayBatcher）

| 参数 | 值 | 说明 |
|------|-----|------|
| OverflowScale | 1.2 | 精灵比 hex 大 20%，相邻草地重叠覆盖缝隙 |
| PlacementChance | 1.0 | 所有草地 hex 都放置，不留空隙 |
| YOffset | 0.5 | 略高于顶面避免 Z-fighting |
| Billboard | Disabled | 平铺在地面，不朝向相机 |
| AlphaCut | Disabled | 完全混合，保留羽化过渡 |
| 旋转 | 无 | 2D 精灵不旋转，只随机选纹理 |
| 偏移 | 无 | 精确居中在 hex 上 |

## 适用地形

Grassland、Plains、Savanna、Forest、DenseForest

## 与非草地地形衔接

- 六边形 mask 形状匹配 hex 轮廓，不会在交界处露出方角
- 8px 羽化让边缘柔和过渡
- 非草地 hex 不放置精灵，自然露出底层六棱柱纹理
