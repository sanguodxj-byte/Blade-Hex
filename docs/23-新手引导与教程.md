# 新手引导与教程

## 概述

剑与六芒星的系统极为复杂——六维属性、技能盘、装备驱动身份、法术盲盒、堆叠战斗、势力关系……新玩家很容易迷失。教程系统不是一次性关卡，而是**贯穿前三幕的渐进式引导**，在玩家需要时才出现。

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| 做中学 | 不用文字墙教规则，用任务引导玩家体验 |
| 渐进揭示 | 雇佣兵时期只教核心，骑士时期教军队，领主时期教经营 |
| 可跳过 | 所有教程可跳过，老手不受干扰 |
| 失败也是教程 | 角色死亡后弹出"经验总结"（如"下次遇到牛头人时注意反冲锋"） |
| 上下文提示 | 不需要"教程关"，而是在真实场景中弹出提示 |

## 二、雇佣兵时期引导（Phase 1-2）

### 2.1 第一步：战斗基础

| 步骤 | 触发 | 教学内容 |
|------|------|----------|
| 1 | 第一场战斗开始 | 高亮移动按钮 → "点击移动范围中的蓝色格子" |
| 2 | 移动后 | 高亮攻击按钮 → "点击红色范围内的敌人攻击" |
| 3 | 攻击后 | 显示命中率 → "78%命中概率。攻击前悬停目标可查看预览" |
| 4 | 被攻击时 | 显示伤害数字 → "受到12伤害。敌人攻击也用同样的规则" |
| 5 | 首次暴击 | 弹出金色星星 → "★ 暴击！伤害翻倍" |
| 6 | 回合结束 | "所有单位行动完毕后点击结束回合" |

## 五、提示系统

### 5.1 上下文提示

不是弹窗，而是在UI边缘显示的**小提示条**：

```
┌────────────────────────────────────────────────────────────┐
│ 💡 悬停敌人可查看抗性和弱点                                │
└────────────────────────────────────────────────────────────┘
```

### 5.2 提示触发规则

| 提示类型 | 触发条件 | 重复 |
|----------|----------|------|
| 首次提示 | 第一次遇到某情况 | 不重复 |
| 提醒提示 | 10次未使用某功能 | 最多3次 |
| 失败提示 | 某操作失败 | 不重复 |

### 5.3 已实现的基础

`TipsData.gd` 和 `TipsDisplay.gd` 已存在于加载界面，可扩展为游戏内提示系统。

## 七、游戏内百科（Codex）

### 7.1 百科入口

主菜单和游戏内暂停菜单中可打开"百科"面板，按类别浏览所有游戏机制说明。

### 7.2 百科结构

```
百科
├── 基础规则
│   ├── 属性与检定
│   ├── 命中与伤害
│   ├── 豁免检定
│   └── 优势与劣势
├── 战斗
│   ├── 地形效果
│   ├── 包夹与伏击
│   ├── 士气系统
│   ├── 状态效果
│   └── 冲锋与撤退
├── 装备
│   ├── 武器类型
│   ├── 防具与AC
│   ├── 坐骑系统
│   └── 稀有度与词缀
├── 技能盘
│   ├── 点亮规则
│   ├── 跳跃机制
│   └── 职业称号
├── 法术
│   ├── 法术获取
│   ├── 魔力与冷却
│   └── 法术书
└── 战略
    ├── 大地图
    ├── 委托系统
    ├── 招募
    └── 势力关系
```

### 7.3 解锁规则

百科条目不是全部可见的——部分条目需要**触发后才解锁**：

| 条目 | 解锁条件 |
|------|----------|
| 暴击规则 | 首次暴击 |
| 包夹 | 首次从侧翼攻击 |
| 法术系统 | 首次获得法术 |
| 坐骑系统 | 首次骑乘 |
| 兵种堆叠 | 晋升为骑士 |
| 攻城战 | 首次攻城 |
