# 种族与招募系统

## 概述

游戏中有多个可玩种族和NPC种族。种族影响属性修正、初始好感度、招募成本、以及某些特殊能力。招募机制区分同族和外族：同族花钱就行，外族需要完成任务或支付溢价。

## 可玩种族

### 人类（Human）
- **聚居地**：中央平原，遍布全大陆
- **特色**：全能型，无短板无特长，适应力强
- **属性修正**：无
- **种族特性**：适应：每10级额外获取一个技能点。
- **初始好感**：人类+20，精灵+5，矮人+10，半兽人-5，半精灵+5
- **招募难度**：极低（到处都是）
- **适合职业**：全职业通用

### 精灵（Elf）
- **聚居地**：银叶森林，少量在城镇
- **特色**：敏捷与智慧兼备，长寿种族
- **属性修正**：无
- **种族特性**：技艺：剑类和弓类命中+1，伤害修正x1.1。
- **初始好感**：精灵+25，人类+5，矮人+0，半兽人-15，半精灵+15
- **招募难度**：中等（需在精灵聚居地，或完成精灵委托后解锁）
- **适合职业**：法师、游侠、游荡者

### 矮人（Dwarf）
- **聚居地**：霜冠山脉地下城，少量在城镇铁匠铺
- **特色**：体魄与力量的象征，擅长锻造和矿务
- **属性修正**：无
- **种族特性**：韧性：AC+1，生命值+20%。
- **初始好感**：矮人+25，人类+10，精灵+0，半兽人-20，半精灵+5
- **招募难度**：中等（需在矮人据点，或在矿场帮过忙）
- **适合职业**：战士、圣骑士、牧师

### 半兽人（Half-Orc）
- **聚居地**：焦土荒原边缘，少量在人类佣兵团
- **特色**：狂野而强壮的身体，天生的战士
- **属性修正**：无
- **种族特性**：狂暴：血量低于50%时，伤害加20%。威胁直觉：先攻+2。
- **初始好感**：半兽人+20，人类-10，精灵-20，矮人-15，半精灵-5
- **招募难度**：高（需完成证明实力的任务，或击败其首领后收编）
- **适合职业**：战士、野蛮人、游侠

### 半精灵（Half-Elf）
- **聚居地**：人类城镇和精灵聚落边缘
- **特色**：人类和精灵的混血，社交能力强
- **属性修正**：无
- **种族特性**：双重血统（人类和精灵聚居地都视为友好），社交天赋（交涉/招募价格-10%）
- **初始好感**：半精灵+25，人类+10，精灵+15，矮人+5，半兽人-5
- **招募难度**：低（两个种族的聚居地都可招募）
- **适合职业**：圣骑士、牧师、游荡者、法师

## 招募系统

### 英雄招募（酒馆/聚居地）

酒馆中出现的英雄是随机生成的：
- 属性完全随机（3d6×6 + 种族修正 + 特质修正）
- 随机2-4个特质（属性类+少量功能性，见05）
- 技能点已根据属性和特质量身分配（AI加点准确度随等级递进）
- 低级角色技能盘可能"歪"，高级角色更精准
- 玩家可以查看其完整属性、特质、技能盘再决定是否招募

#### 同族招募
- 在同族聚居地的酒馆中直接花钱招募
- 价格由英雄等级决定
- 无额外条件

| 英雄等级 | 基础价格 |
|----------|----------|
| 1级 | 50金 |
| 2-3级 | 100-200金 |
| 4-5级 | 300-500金 |
| 6级+ | 800金+ |

#### 外族招募
外族英雄在酒馆中出现概率较低，且需要满足额外条件：

| 条件类型 | 说明 |
|----------|------|
| 好感度门槛 | 与该种族好感度 ≥ 30 |
| 完成委托 | 完成该种族聚居地的1-2个委托 |
| 交涉检定 | CHA检定 vs DC 15（好感度高时DC降低） |
| 价格溢价 | 外族招募价格为同族的 ×1.5~2.0 |
| 特殊条件 | 某些种族需满足剧情条件 |

#### 外族招募流程
```
1. 酒馆遇到外族英雄 → 显示"此人对你的态度：冷淡"
2. 选择 [交谈] → 进入对话
   - 好感度不足 → "我不信任你这种人。" → 招募失败
   - 好感度足够 → "也许...我需要看看你的实力。" → 触发条件
3. 条件类型：
   a. 完成任务型 → 给你一个小任务（讨伐/护送）→ 完成后可招募
   b. 交涉型 → 进行CHA检定 → 成功则可招募（价格略高）
   c. 暴力型（半兽人常见）→ 单挑 → 赢了则获得尊重，可招募
4. 满足条件 → 以溢价价格招募
```

### 士兵招募（兵营/村庄）

#### 同族士兵
- 在己方种族领地的兵营直接招募
- 标准价格，无门槛

#### 外族士兵
| 种族 | 招募条件 | 溢价 | 特殊兵种 |
|------|----------|------|----------|
| 精灵 | 好感≥30 + 完成1次精灵委托 | ×1.5 | 精灵射手（射程+1，伤害+1） |
| 矮人 | 好感≥30 + 完成1次矮人委托 | ×1.5 | 矮人重甲兵（AC+2，速度-1） |
| 半兽人 | 好感≥50 + 击败半兽人首领 | ×2.0 | 兽人狂战士（伤害+3，无护甲） |
| 半精灵 | 好感≥20（两个种族聚居地都行） | ×1.3 | 无特殊兵种，但社交加成 |

### 招募价格公式
```
最终价格 = 基础价格 × 种族系数 × 关系系数

种族系数：
  同族 = 1.0
  外族 = 1.5 ~ 2.0

关系系数：
  同盟 = 0.8
  友好 = 0.9
  中立 = 1.0
  冷淡 = 1.3
  敌对 = 不可招募
```

### 忠诚度系统

| 忠诚度 | 来源 | 影响 |
|--------|------|------|
| 高(80+) | 同族+长期并肩+按时发饷 | 战斗中不溃逃，拒绝敌方招揽 |
| 中(40-79) | 正常状态 | 正常战斗 |
| 低(0-39) | 欠饷/种族冲突/损失惨重 | 战斗中可能逃跑或叛变 |
| 负值 | 长期虐待/强迫招募的战俘 | 随时叛变 |

#### 忠诚度影响因素
| 事件 | 忠诚度变化 |
|------|-----------|
| 每日正常发饷 | +1 |
| 赢得战斗 | +3~+5 |
| 升级 | +5 |
| 欠饷1天 | -3 |
| 同族队友死亡 | -5 |
| 外族队友加入（种族敌对时） | -3~-10 |
| 完成该种族相关任务 | +5~+10 |
| 长期不给休息 | -2/天 |
