# 开发路线图

## 总体结构：三幕式开发

```
Phase 1 ─── 战术核心    → 可玩战斗原型（小队战棋）
Phase 2 ─── 战术深化    → 有深度的战斗（多职业/法术/AI）
Phase 3 ─── 战略层      → 骑砍式大地图 + 遭遇系统
Phase 4 ─── 军队系统    → HOMM式堆叠战斗 + 攻城
Phase 5 ─── 领地经营    → 城堡/税收/势力博弈
Phase 6 ─── 打磨发布    → 美术/音效/教程/平衡
```

---

## Phase 1: 战术核心（最小可玩原型）

### 目标
一场完整的六边形战斗：我方3-4人 vs 敌方3-4人

### 任务清单

#### 1.1 六边形网格系统
- [ ] HexGrid 场景 — 六边形网格生成(axial坐标)
- [ ] HexCell — 单个六边形（地形类型、坐标、邻居查询）
- [ ] 网格渲染 — 绘制可交互的六边形网格
- [ ] 坐标转换 — 像素↔六边形坐标互转

#### 1.2 寻路与移动
- [ ] A*寻路算法 — 六边形版本
- [ ] 移动范围计算 — BFS(速度值)
- [ ] 移动动画 — 角色沿路径移动
- [ ] 障碍物处理 — 不同地形移动消耗

#### 1.3 角色系统（基础）
- [ ] Unit 基类 — HP/AC/属性/速度
- [ ] 角色选中与高亮
- [ ] 角色状态（待机/已移动/已行动）

#### 1.4 命运检定战斗核心
- [ ] 命中检定 — 随机数(1-20) + 加值 vs AC
- [ ] 伤害计算 — 武器伤害骰 + 属性修正
- [ ] 暴击系统 — 最高结果伤害翻倍
- [ ] 伤害数字弹出

#### 1.5 回合管理
- [ ] 我方回合/敌方回合切换
- [ ] 单位行动顺序管理
- [ ] 回合结束条件判定

#### 1.6 基础UI
- [ ] 操作面板（移动/攻击/结束回合）
- [ ] 角色HP条
- [ ] 战斗日志
- [ ] 简易角色信息面板

#### 1.7 原型数据
- [ ] 1张战斗地图(12×10格)
- [ ] 战士职业(近战)
- [ ] 长剑 + 长弓
- [ ] 哥布林战士(CR 1/4) × 3

### 交付物
- 可运行的Godot场景
- 点击移动 → 攻击 → 敌方回合 → 战斗结束
- 基础命中检定结果显示（显示概率和结果，不暴露骰子）

## Phase 2: 战术深化

### 目标
有深度的战斗体验，多职业，法术，AI。第一幕"雇佣兵时期"完整可玩。

### 任务清单
- [ ] 地形系统（森林/山地/水域/高程）
- [ ] 6个基础职业全部实现
- [ ] 法术系统（3戏法+5个1环）
- [ ] 敌方AI（行为树/策略风格）
- [ ] 多种敌人（哥布林/狗头人/亡灵/野兽）
- [ ] 装备系统（武器/防具/盾牌）
- [ ] 物品使用（治疗药水/解毒剂/火油瓶）
- [ ] 状态效果（中毒/燃烧/流血/恐惧）
- [ ] 战斗UI完善（命中率预览/法术选择/战斗日志）
- [ ] 多张战斗地图
- [ ] 角色招募（酒馆系统）
- [ ] 角色永久死亡机制
- [ ] 等级系统（1-5级）
- [ ] 进阶专精（5级觉醒）

### 交付物
- 完整可玩的第一幕"雇佣兵时期"
- 可反复游玩的委托-战斗循环

## Phase 3: 战略层（骑砍式大地图）

### 目标
大地图行军 + 遭遇系统，第二幕"骑士时期"的基础

### 任务清单
- [ ] 六边形大地图生成（80×60格）
- [ ] 实时行军系统（含暂停）
- [ ] 地形与视野系统
- [ ] 城镇/村庄/酒馆/商店场景
- [ ] 委托系统（布告栏 → 战斗 → 领赏）
- [ ] 野怪遭遇（地图上的哥布林营地等）
- [ ] 冒险者队伍AI（其他佣兵团在大地图行动）
- [ ] 领主军队AI（诸侯军队在大地图行动）
- [ ] 补给与疲劳系统
- [ ] 营地休息系统
- [ ] 大地图UI（缩放/拖拽/信息面板）
- [ ] 声望系统
- [ ] 存档/读档

### 交付物
- 大地图与战术战斗的无缝切换
- 委托驱动的完整循环

---

## Phase 4: 军队系统（HOMM式）

### 目标
兵种堆叠战斗，实现第二幕"骑士"到第三幕"领主"的跨越

### 任务清单
- [ ] 兵种定义系统（长枪兵/剑盾/弓手/骑兵/法师团/重甲）
- [ ] 堆叠单位（StackUnit）— 人数/HP聚合/伤害计算
- [ ] 堆叠伤害公式（基础伤害 × sqrt(数量)）
- [ ] 堆叠 vs 英雄 的交互规则
- [ ] 士气系统（损失过半需豁免，失败溃逃）
- [ ] 扩展战场（20×16格）
- [ ] 招募士兵系统（兵营/训练场）
- [ ] 军队管理UI
- [ ] 军队薪饷与补给消耗
- [ ] 阶段晋升事件（雇佣兵 → 骑士 → 领主）

### 交付物
- 英雄+军队混合战斗可玩
- 堆叠伤害结算正确
- 小队战斗和军团战斗平滑切换

## Phase 5: 领地经营

### 目标
攻城掠地，势力博弈，完整的第三幕"领主时期"

### 任务清单
- [ ] 城堡系统（木栅/石堡/要塞）
- [ ] 攻城战（城墙/城门/箭塔/攻城器械）
- [ ] 领地管理（税收/商路/矿场）
- [ ] 领地建设（升级城堡/训练场/法塔）
- [ ] 势力关系系统（同盟/友好/中立/敌对/交战）
- [ ] AI领主战略行为（扩张/结盟/宣战/防守）
- [ ] 史诗怪物系统（龙/远古魔像在大地图游荡）
- [ ] 暗影教团事件链
- [ ] 主线剧情推进
- [ ] 多结局系统

### 交付物
- 完整的三幕游戏循环
- 从雇佣兵到领主的全流程可玩
