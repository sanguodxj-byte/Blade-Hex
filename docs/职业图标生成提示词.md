# 职业图标生成提示词

## 基础风格（BASE_STYLE）

```
Dual-tone hand-painted game icon, ONLY two colors: warm parchment cream (#F5E6C8) as background fill 
and dark ink brown (#3D2B1F) as line art and shading, 
NO other colors, NO color gradients, strictly monochrome two-tone illustration, 
fully transparent background (alpha = 0 outside the icon), no background at all, 
clean minimal silhouette with crisp 1px ink-brown outline, 
ink-brown hatching and cross-hatching for shading instead of color fills, 
parchment cream used only for interior fills and highlights, 
flat 2-tone cel shading — dark ink for shadows/lines, parchment cream for lit areas, 
style reference: medieval manuscript marginalia meets FFTactics job class icons, 
simple symbolic emblem centered in a square frame, highly readable at 32x32 pixels, 
icon design is a single clear symbol representing the class, NOT a full character portrait, 
the symbol should be instantly recognizable by its silhouette alone
```

## 核心思路

- **双色**：仅羊皮纸暖色 (#F5E6C8) + 墨棕 (#3D2B1F)，禁止任何其他颜色
- **透明背景**：图标区域外完全透明 (alpha=0)
- **简洁符号**：每个职业对应一个标志性符号（剑、弓、盾、眼、皇冠等），不是人物立绘
- **小尺寸可读**：在32×32像素下仍能辨认轮廓
- **风格参考**：中世纪手稿页边插图 × FFTactics职业图标
- **多属性≠更强**：图标符号只是表达「涉足了哪些方向」，不代表实力强弱。单属性图标是一个清晰的大符号，多属性图标是多个小符号的拼合——数量多了，但每个都小了

## 与武器图标的区别

| 维度 | 武器图标 | 职业图标 |
|------|----------|----------|
| 色彩 | 多色（钢灰/铜金/木棕等） | 严格双色（羊皮纸+墨棕） |
| 背景 | 透明 | 透明 |
| 构图 | 45度倾斜武器 | 居中对称符号 |
| 比例 | 16:9竖版 (1024×1792) | 1:1方版 (1024×1024) |
| 内容 | 具体武器细节 | 抽象职业象征 |
| 风格参考 | FFT物品图 × 恶魔城装备 | 中世纪手稿 × FFT职业 |

## 完整提示词结构

```
{BASE_STYLE}.
Icon for the class '{English}' ({Chinese}): {icon_description}.
Represents: {class_flavor}.
The icon is a symbolic emblem, NOT a character or creature.
Keep it simple and instantly readable.
```
